## 적용 범위

- 대상: 전신 작용 즉시방출(IR) 경구 고형제형의 제네릭/하이브리드 의약품
- 목적: 시험약(Test)과 대조약(Reference)의 생물학적 동등성 입증을 위한 연구 설계, 수행, 평가 요건 규정
- 적용 제외: 국소 작용 국소 적용 제품, 흡입제, 특수 제형 등 (별도 가이드라인 적용)

## 1. 시험 설계 (Study Design)

### 1.1 표준 설계

- **표준: 단회 투여 2×2 교차 설계 (single-dose, 2×2 crossover)**
- 최소 12명의 건강 성인 자원자
- 비구획 분석(non-compartmental methods)으로 PK 파라미터 추정
- 비교 대상 파라미터:
  - 1차: AUC(0-t), Cmax
  - 2차: AUC(0-∞), tmax (분포 비교), t½
- 최소 공복 시간: 8시간

### 1.2 교차 설계 vs. 병렬 설계

| 설계 | 적용 조건 |
|------|-----------|
| 2×2 교차 (표준) | 대부분의 IR 제제 |
| 병렬 (parallel) | 반감기가 매우 길어 교차 설계 시 세척 기간이 비현실적인 경우; 인구학적 특성 동등성 보증 필요 |
| 반복 교차 (replicate) | 고변동 약물(HVD), CVw >30% 예상 시 |
| 다회 투여 (multiple-dose) | 단회 투여 시 내약성·안전성 문제 또는 분석 감도 한계가 있는 경우에 한해 허용 |

**병렬 설계 시 추가 요건:**
- 비교 가능한 인구학적 특성(연령, 성별, 체중 등) 문서화
- CYP 다형성 등 약동학 변동 요인에 대한 표현형/유전형 검사 권고

### 1.3 2단계 설계 (Two-Stage Adaptive Design)

- 가이드라인에서 처음 명시적으로 허용
- 1단계: 중간분석(interim analysis)
- 2단계: 전체 데이터셋을 합쳐 최종 분석
- **필수 조건:**
  - 1종 오류(Type I error) 조정 필요 — 90% 신뢰구간의 포함 확률 >90% 유지
  - Alpha spending plan을 프로토콜에 사전 정의
  - ANOVA 모델에 stage 항 포함

### 1.4 세척 기간 (Washout Period)

- 최소: 소거가 충분히 이루어지는 기간 (일반적으로 t½의 5배 이상)
- 교차 설계에서 투여 전 농도 >Cmax 5%이면 해당 데이터 제외 (carry-over 평가)

## 2. 생물학적 동등성(BE) 판정 기준

### 2.1 표준 허용 범위

| 파라미터 | 허용 범위 (90% CI) |
|----------|--------------------|
| AUC(0-t) | **80.00–125.00%** |
| Cmax | **80.00–125.00%** |
| AUC(0-∞) | 통상적으로 AUC(0-t)와 동일 기준 적용 |

- 로그 변환 후 기하평균비(GMR)에 대한 90% CI가 80.00–125.00% 이내에 완전히 포함되어야 함
- 하한: 소수점 2자리 반올림 시 ≥80.00%
- 상한: 소수점 2자리 반올림 시 ≤125.00%

### 2.2 장반감기 약물: 절단 AUC (Truncated AUC)

- 반감기가 긴 약물에서 AUC(0-∞) 추정의 불확실성 문제 해결을 위해 도입
- **AUC(0-72h)**: 72시간까지 채혈하여 절단 AUC 사용 가능 (AUC(0-t) 대체)
- 교차 설계뿐 아니라 병렬 설계에서도 사용 가능
- 단, AUC(0-72h)가 AUC(0-∞)의 충분한 비율을 포착해야 함

---

## 3. 고변동 약물 (Highly Variable Drugs, HVD)

### 3.1 정의

- 개체 내(within-subject) CV >30%인 약물 또는 제형

### 3.2 Scaled Average BE (ABEL) 허용 범위 확장

- **대상 파라미터: Cmax만 허용** (AUC는 표준 80.00–125.00% 유지)
- 반복 교차 설계 필수 (reference product의 within-subject 변동 추정 목적)
- **확장 공식:**
  - 허용 범위: (L, U) = exp(±k × s_WR)
  - k = 0.760 (EU 규제 상수)
  - s_WR = reference product의 within-subject 표준편차 (로그 척도)

| CVw (reference) | Cmax 허용 범위 |
|-----------------|----------------|
| <30% | 80.00–125.00% (표준, 확장 없음) |
| 30% | 80.00–125.00% (임계점) |
| 40% | ~74.62–134.02% |
| 50% | 69.84–143.19% (최대 한도) |
| >50% | **상한: 69.84–143.19%로 고정 (cap)** |

- **점 추정치 조건**: GMR은 여전히 80.00–125.00% 이내여야 함
- **임상적 정당화 필요**: Cmax 확장 적용은 임상적으로 정당화되어야 함
- **반복 설계 형식**: 어떤 반복 교차 설계(3기, 4기 등)도 허용

### 3.3 FDA와의 비교

| 항목 | EMA (ABEL) | FDA (RSABE) |
|------|-----------|-------------|
| k 상수 | 0.760 (더 보수적) | 0.893 (더 관대) |
| 점 추정치 요건 | 80.00–125.00% | 80.00–125.00% |
| 최대 한도 | 69.84–143.19% | 제한 없음 (단, 점 추정치 조건) |

---

## 4. 협범위 약물 (Narrow Therapeutic Index, NTI)

### 4.1 허용 범위 강화

| 파라미터 | 허용 범위 (90% CI) |
|----------|--------------------|
| AUC | **90.00–111.11%** |
| Cmax | **90.00–111.11%** (안전성·효능·혈중농도 모니터링에 중요한 경우) |

- Cmax 기준 강화 여부는 CHMP가 사례별로 결정
- 공복 및 식후 시험 모두 강화 기준 적용

### 4.2 NTI 약물 판정 방식

- 가이드라인 내에 확정 목록 없음 → **사례별(case-by-case) CHMP 결정**
- 일반적으로 NTI로 간주되는 약물(참고용):
  - 시클로스포린(ciclosporin): AUC + Cmax 모두 90.00–111.11%
  - 타크로리무스(tacrolimus): AUC만 90.00–111.11%
  - 디곡신(digoxin), 리튬(lithium), 페니토인(phenytoin), 와파린(warfarin), 테오필린(theophylline) 등

## 5. 공복 및 식후 시험

### 5.1 기본 원칙

- **원칙: 공복 시험이 표준** (제형 차이 검출에 가장 민감)
- EMA는 FDA와 달리 기본적으로 공복 시험을 먼저 요구

### 5.2 식후 시험 필요 조건

| 조건 | 식후 시험 필요 여부 |
|------|---------------------|
| 대조약 SmPC에서 식사와 함께 복용 권고 (PK 이유) | 필요 |
| 대조약 SmPC에서 공복 복용 권고 | 불필요 |
| 공복/식후 무관 | 공복 시험만으로 충분 (원칙) |
| 특수 기술 제형 (microemulsion, solid dispersion 등) | 공복 + 식후 **모두** 필요 |

### 5.3 식사 조성

대조약 SmPC에 명시된 경우 → SmPC 지침 따름  
명시 없는 경우 → 고지방·고열량(High-fat, high-calorie) 식사:
- 총 칼로리: 약 800–1,000 kcal
- 단백질: 약 150 kcal
- 탄수화물: 약 250 kcal
- 지방: 약 500–600 kcal (총 칼로리의 약 50%)

## 6. Biowaiver 조건

### 6.1 BCS Class I Biowaiver

| 요건 | 세부 기준 |
|------|-----------|
| 고용해성 | 최대 단회 투여량이 pH 1.0–6.8의 250 mL 이하 수성 용액에 완전 용해 |
| 고투과성 | 절대 생체이용률 ≥85% 또는 mass balance 연구 기반 흡수율 ≥85% |
| 신속 용출 | pH 1.2, 4.5, 6.8 각각에서 30분 내 ≥85% 용출 |
| 동일 제형 | 동일 IR 고형 경구 제형 (정제↔캡슐 전환 불가) |
| 동일 염 형태 | 다른 염 형태는 허용 |
| 부형제 요건 | BCS 기반 흡수에 영향을 미치지 않는 부형제 사용 |

### 6.2 BCS Class III Biowaiver (신설)

- 2010년 개정에서 처음 명시적으로 허용 (이전 가이드라인은 Class I만)

| 요건 | 세부 기준 |
|------|-----------|
| 고용해성 | Class I과 동일 기준 |
| 저투과성 | 인체 흡수 데이터 기반 확인 |
| **매우 신속한 용출** | 각 pH에서 **15분 내 ≥85%** (Class I의 30분보다 엄격) |
| 부형제 동일성 | 동일한 부형제를 동일한 양으로 사용 (bioavailability에 영향 없는 부형제도 일치 필요) |
| 계면활성제 제한 | 극미량 허용; 흡수 변화 가능 부형제 차이 엄격 제한 |
| 용출 시험 | pH 1.2, 4.5, 6.8, 최소 용해도 pH에서 추가 시험 |

**주의**: 미량 부형제 차이도 흡수에 영향 가능 (예: sodium lauryl sulfate 4 mg → alendronate 생체이용률 5–6배 증가; sorbitol 7 mg → risperidone Cmax 60% 감소)

### 6.3 추가 강도 Biowaiver (Additional Strength Biowaiver)

**적용 조건 (모두 충족):**
1. 동일 제조 공정 (다른 제조소 허용)
2. 동일 정성적 조성 (착색제 제외)
3. 정량적 비례 조성 (활성 성분 대비 부형제 비율 일정)
4. 유사 용출 프로파일 (in vitro 확인)
5. BE는 차이 검출에 가장 민감한 강도에서 수행

**용량 비례성 기준**: 검증 강도와 면제 강도 간 용량 조정 평균 AUC 차이 ≤25% (비율 0.75–1.33)  
**Bracketing approach**: 비례 조성 미충족 시 최저·최고 강도만 시험 가능

## 7. 통계 분석

### 7.1 분석 모델 (ANOVA)

- 로그 변환(logarithmic transformation) 후 ANOVA 수행
- **고정 효과(fixed effects)로 모든 항 처리** (random effects 사용 불가):
  - Sequence (투약 순서)
  - Subject within sequence (교차 설계에서 sequence 내 피험자)
  - Period (기간)
  - Formulation (제형: 시험약 vs. 대조약)
- **중요**: FDA와 달리 피험자를 random effect로 처리하지 않음

### 7.2 신뢰구간 계산

1. 로그 변환된 파라미터에서 ANOVA로 제형 간 차이의 신뢰구간 산출
2. 역변환(back-transformation)으로 원 척도의 기하평균비에 대한 90% CI 획득
3. CI가 허용 범위에 완전히 포함되어야 함

### 7.3 데이터 처리

| 상황 | 처리 방법 |
|------|-----------|
| 한 기간 데이터 누락 피험자 | 분석에서 제외 |
| 결측 대체 | 불허 (SAS Proc Mixed 등 대체 불가) |
| 통계적 이상치 제외 | 불허 |
| 투여 전 농도 >Cmax 5% | 해당 기간 데이터 제외 (carry-over 평가) |
| 비정상 거동 (reference 농도 <기하평균의 5%) | 제외 가능 (정당화 필요) |

### 7.4 분석 대상 물질 선택

| 우선순위 | 적용 상황 |
|----------|-----------|
| 모약물(parent drug) | 원칙 (방출 속도 차이 검출 민감도 우수) |
| 대사체 | 전구약물 또는 모약물 혈중 농도 낮고 변동 큰 경우 허용 |
| 거울상 이성질체(enantiomers) | 3가지 조건 모두 충족 시 키랄 분석 필수: (1) PK 차이, (2) PD 차이, (3) AUC 비율이 흡수 속도에 따라 달라짐 |

## 8. 기준 제품 선정 (Reference Product)

- EU 회원국에서 승인된 제품 우선
- 허용 기준 (Directive 2001/83/EC):
  - Article 8(3): 전체 신청 다시어로 승인된 제품
  - Article 10a, 10b, 10c: 문헌 기반, 혼합 신청 등으로 승인된 제품
- **배치 대표성**: 최소 EU 시판 배치 2개와 비교하여 대표성 정당화 필요
- **역가 보정(potency correction)**: 차이 >5%일 때만 허용; EU 시판 배치로 <5% 차이 불가 시 허용

---

## 9. Supra-Bioavailability (초과 생체이용률)

- Cmax 또는 AUC가 대조약 대비 25%를 초과하는 경우
- 허용 범위: 시험약의 AUC가 대조약의 80–125% 이내이어야 함
- **초과 생체이용률 우려 시**: 안전성 평가 필요; 필요한 경우 용량 조정 또는 추가 임상 데이터 요구

---

## 10. 특수 상황

### 10.1 내인성 물질 (Endogenous Substances)

- 식이 섭취 및 기저 조건 표준화
- 기저치 보정 방법: 상수 빼기 또는 투여 전 AUC 빼기
- 연구 민감도: 사전 파일럿 또는 BE 연구의 일부로 검증
- 수퍼치료 용량(supra-therapeutic dose) 사용으로 기저치 이상 측정 촉진

### 10.2 키랄 분석

- 비키랄 분석법 허용: 두 거울상이성질체의 PD 활성이 유사하거나 비선형 흡수 없음이 증명된 경우
- 키랄 분석법 필수: 상기 세 가지 조건 모두 충족 또는 불명확한 경우

### 10.3 병렬 설계 시 추가 요건

- 인구학적 특성 동등성 문서화
- 표현형/유전형 검사 권고
- 비례 인원 배정

## 11. 주요 변경 사항 (2001 가이드라인 대비 2010년 개정)

| 항목 | 변경 내용 |
|------|-----------|

| 고변동 약물 | Scaled ABEL 허용 (Cmax만), replicate 설계 명시화 |
| BCS biowaiver | Class I만 → Class III 추가 |
| 절단 AUC | AUC(0-72h) 명시적 허용 |
| 2단계 설계 | 처음으로 명시적 허용 |
| 추가 강도 면제 | 비례 조성 조건 완화 (다른 제조소 허용) |
| 역가 보정 | 조건 명확화 (>5% 차이) |

## 12. FDA와의 주요 차이점

| 항목 | EMA | FDA |
|------|-----|-----|
| 기본 시험 상태 | **공복** (우선) | 공복 + 식후 (모두 요구가 많음) |
| 고변동 약물 k 상수 | **0.760** (더 보수적) | 0.893 (더 관대) |
| BCS 3등급 biowaiver | 허용 (엄격 조건) | 미허용 (Class I만) |
| 통계 모델 | **모든 항 고정 효과** | 피험자를 random effect 허용 |
| NTI 허용 범위 | **90.00–111.11%** 명시 | 사례별 결정 |

| 부형제 (Class III) | 동일 양 요구 (비활성 부형제도) | 더 많은 양 허용 |
