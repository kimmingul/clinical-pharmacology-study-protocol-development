## 1. 개요

ICH E14는 신약 개발 시 심장 재분극에 대한 약물 효과를 평가하기 위한 **"Thorough QT/QTc (TQT) 시험"** 표준 방법론을 제시하였다. 이후 Q&A 개정을 통해 **Concentration-QTc (C-QTc) 분석**을 주요 대안으로 도입하였다.

### 주요 임상 맥락

- QTc 연장 → Torsades de Pointes (TdP) 발생 위험
- TdP → 심실세동 및 급사 위험
- QTc 10ms 이상 연장: 규제 우려의 임계값
- QTc 5ms: 양성 대조군의 민감도 입증 최소 기준

---

## 2. Thorough QT (TQT) 시험 설계

### 2.1 기본 설계: 4-기간 교차 설계 (4-Period Crossover)

TQT 시험의 표준 설계는 **무작위배정 4-기간 4-처치 교차 설계**이다:

| 처치 | 내용 |
|------|------|
| **A: 치료 용량** | 임상 예상 최대 치료 용량 |
| **B: 초과치료 용량(Supratherapeutic)** | 치료 용량보다 높은 용량 (안전 한도 내) |
| **C: 위약(Placebo)** | 맹검 위약 대조군 |
| **D: 양성 대조약** | Moxifloxacin 400mg (open-label 가능) |

**설계 특성:**
- 대상: 건강한 지원자 (원칙), 환자 집단이 필요한 경우 환자
- 교차 설계가 평행 설계보다 소규모로 가능 (개체 내 변동성 감소)
- 장기 반감기 약물: 병행 설계 고려
- 충분한 세척 기간(washout): 최소 5개 반감기 이상

**표본 수:**
- 일반적으로 32~48명 (교차 설계 기준)
- 90% 검정력, 2~3ms 기저 약물 효과 가정

### 2.2 변형 설계

| 설계 변형 | 적용 조건 |
|----------|----------|
| **평행 설계** | 장기 투약이나 적정(titration)이 필요한 경우 |
| **중첩 교차(Nested crossover)** | 양성 대조군을 전략적으로 다른 날 투약하여 표본 수 감소 |
| **중간 PK/PD 검토** | 고용량 달성 가능성 사전 평가 |

---

## 3. 양성 대조약 요건: Moxifloxacin 400mg

### 3.1 선택 근거

- Moxifloxacin: IKr 포타슘 채널의 가역적 차단제
- 400mg 단회 경구 투여 후 2~4시간에 QTc 평균 **10~14ms 연장** 유발
- 재현성이 우수하여 시험 민감도 입증에 최적

### 3.2 양성 대조 수용 기준 (Assay Sensitivity)

| 기준 | 값 |
|------|-----|
| 최대 기저 조정 위약 보정 QTcF 효과 | **8~15ms** |
| 최대 발생 시간 | 투여 후 **1~3시간** |
| 1측 95% 신뢰구간 하한 | **5ms 초과** |
| 최대 이후 QTcF 감소 | 확인 필요 |

### 3.3 주요 변경 사항

- 최초 요건: 이중맹검(double-blind) 필수
- 현재: **Open-label 투여 허용** (FDA 수용)

## 4. 일차 평가변수: ddQTcF (ΔΔQTcF)

### 4.1 정의

$$\Delta\Delta QTcF = (\Delta QTcF_{\text{drug}}) - (\Delta QTcF_{\text{placebo}})$$

여기서:
$$\Delta QTcF = QTcF_{\text{post-dose}} - QTcF_{\text{baseline}}$$

- **시간 매칭(Time-matched)**: 약물 투여군과 위약군의 ECG를 같은 시각에 측정하여 일주기(circadian) 변동 보정
- **기저(Baseline) 조정**: 투여 전 측정값 사용
- **위약 보정**: 약물 효과에서 위약 효과 제거

### 4.2 판정 기준 (음성 결과 = 우려 없음)

| 기준 | 값 |
|------|-----|
| **일차 기준** | 최고 임상 관련 혈장 농도에서 ddQTcF의 1측 97.5% 신뢰구간 상한 < **10ms** |
| **보조 기준** | 모든 측정 시점에서 ddQTcF 점 추정값 < 10ms |
| **양성 결과 판정** | 신뢰구간 상한 ≥ 10ms (규제 우려 발생) |

### 4.3 임계값 해석

| QTcF 변화 | 임상적 의미 |
|----------|-----------|
| < 5ms | 임상적으로 관련성 없음 |
| 5~10ms | 중간 우려 영역 |
| ≥ 10ms | 규제 우려 임계값 |
| ≥ 20ms | 심각한 위험 징후 |

## 5. ECG 측정 요건

### 5.1 기술 요건

| 항목 | 요건 |
|------|------|
| **ECG 유형** | 디지털 12-lead ECG |
| **중복 측정** | 각 시점에 **3회(triplicate)** 기록 |
| **측정 시간** | 위약/기저 측정은 약물군과 **동일 시각** (시간 매칭) |
| **읽기(Reading)** | 맹검(blind): 대상자, 시간, 처치에 대해 맹검 권장 |
| **판독자** | 가능하면 한 시험대상자의 ECG를 같은 날 1명이 판독 |

### 5.2 3회 중복 측정의 이점

- 개체 내 변동성의 측정 오류 감소
- 3회 평균이 단회 측정보다 신뢰도 크게 향상
- 4회 이상은 추가 이점 미미

### 5.3 일주기(Circadian) 효과 보정

- 위약군의 ECG를 약물군과 **동일 시각**에 측정하여 시간에 따른 QTc 자연 변동 통제
- 위약 투여 시간을 moxifloxacin 투여 시간과 동기화

### 5.4 기저치(Baseline) 정의

- 일반적으로 투여 당일 투여 전 시간 매칭 ECG
- 일부 경우 투여 전날 측정값 사용 가능

## 6. Concentration-QTc (C-QTc) 분석 — E14 Q&A R3 (2016) 및 E14/S7B Q&A (2022)

### 6.1 개념

TQT 시험의 전통적 시간 매칭 분석을 대체하는 **노출-반응 모델링** 접근법:

- 약물 혈장 농도와 QTc 변화 간의 관계를 선형 혼합 효과 모델로 분석
- "White paper 모델": 60%의 QTc 연장 약물에 적용 가능한 사전 지정 선형 혼합 효과 모델
- 최고 임상 관련 혈장 농도에서 ddQTcF의 1측 95% CI 상한이 10ms 미만이면 음성

### 6.2 C-QTc 기반 TQT 시험 면제 조건 (5.1 접근법, E14 Q&A 5.1)

**E14 Q&A 5.1** (2015년 도입, 2022년 강화): 1상 시험 데이터를 이용하여 TQT 시험 대체 가능.

| 요건 | 세부 사항 |
|------|----------|
| **노출 배수** | 임상 고노출의 **2배 이상** 달성 시 양성 대조군 면제 가능 |
| **ECG 품질** | 고품질 ECG 기록 필요 |
| **대조군** | 위약 대조군 필수 |
| **고용량 포함** | 예상 최고 임상 노출을 초과하는 용량 포함 |

### 6.3 "이중 음성(Double-Negative)" 통합 위험 평가 (2022 Q&A 신규)

임상 시험에서 충분한 노출 배수를 달성하지 못한 경우, 다음 비임상 증거로 보완 가능:

**구성 요소 1 (in vitro hERG):**
- QTc 연장으로 알려진 약물보다 hERG 안전 마진이 높아야 함
- ICH S7B Q&A 2에 따른 최선 실무 기준 준수

**구성 요소 2 (in vivo):**
- 임상 노출을 초과하는 수준에서 QTc 연장 없음을 확인
- ICH S7B Q&A 3에 따른 충분한 민감도의 in vivo 시험

### 6.4 6.1 접근법 (E14 Q&A 6.1)

- 임상 노출 달성 시 (2배 배수 불필요) 허용
- 기저 조정 QTcF 상한이 10ms 미만 + 실제 평균 효과가 20ms 미만일 가능성이 낮음을 입증

### 6.5 TQT 시험 사용 추이 (FDA 데이터, 2016~2024)

| 연도 | TQT 시험 비율 | C-QTc (5.1) 비율 |
|------|--------------|-----------------|
| 2016 | 75% | 6% |
| 2023 | 41% | 39% |
| 변화 | **-34%** | **+33%** |

C-QTc 분석 사용 시 표본 수 감소 효과:
- 평행 설계: 67% 감소
- 중첩 교차: 42% 감소  
- 교차 설계: 35% 감소

## 7. TQT 시험 면제 조건 (Waiver Conditions)

### 7.1 전통적 TQT 면제 (규제기관 협의 필요)

| 면제 사유 | 조건 |
|----------|------|
| 국소 적용 약물 | 전신 노출이 극히 낮음을 입증 |
| 비임상 데이터 | 비임상 패키지만으로는 TQT 면제 불가 (단독 사유로 인정 안 됨) |
| 수명 단축 질환 | 임상적 현실성 부족 시 수정 설계 허용 |

### 7.2 C-QTc 기반 TQT 대체 (현재 주요 경로)

위 **5.1 접근법** 및 **이중 음성 통합 위험 평가** 참조

### 7.3 설계 수정이 필요한 경우

| 상황 | 대안 |
|------|------|
| 긴 반감기 약물 | 반복 투여 교차 또는 평행 설계 |
| 건강인에게 안전성 문제 | 환자 집단으로 전환 |
| 초과치료 용량 달성 불가 | C-QTc + 비임상 보완 (5.1/이중 음성) |
| 만성 질환 | 복수 투여 설계 |

## 8. 통계 분석

### 8.1 일차 분석 방법

- 시간 매칭, 기저 조정, 위약 보정 ddQTcF의 평균 및 90% 신뢰구간 (각 시점별)
- 일측 97.5% 신뢰구간 상한 < 10ms → 음성 결과
- 선형 혼합 효과 모델 사용

### 8.2 민감도 분석

- 다양한 심박수 보정 방법 비교 (Fridericia, Bazett, 개인별 보정)
- 성별, 기저 심박수 등 공변량 탐색

### 8.3 탐색 분석

- 농도-반응 관계 시각화
- 이상치 분석 (QTcF > 450ms, > 480ms, > 500ms 또는 30ms 이상 증가)

## 9. 임상약리 관점에서의 실무 적용

### 9.1 임상약리 시험 (Phase 1) 계획 시 고려사항

| 요소 | 고려 사항 |
|------|----------|
| **용량 설계** | 임상 고노출의 2배 달성 가능 용량 포함 (5.1 경로 목표 시) |
| **ECG 측정 시점** | PK 채혈 시점과 시간 매칭, Tmax 전후 충분한 측정 시점 |
| **PK-PD 연계** | 혈장 농도와 QTc 동시 측정을 위한 채혈 및 ECG 일정 조정 |
| **위약 제어** | 무작위배정 교차 설계에서 위약 포함 |
| **기저 측정** | 투여 전 triplicate ECG 측정 |

### 9.2 권고 ECG 측정 시점 (일반 가이드라인)

- **투여 전**: 1~3회 측정 (기저값)
- **Cmax 전**: 점진적 측정 (30min, 1h)
- **Cmax 근방**: 집중 측정 (Tmax ±0.5h)
- **Cmax 이후**: 점진적 측정 (2h, 4h, 6h, 8h, 12h, 24h)
- **기타**: 약물 특성(반감기, 지연 효과)에 따라 조정

## 10. S7B와의 연계 (비임상 통합 평가)

ICH S7B: "비임상적 QT/QTc 간격 연장 및 부정맥 유발 잠재성 평가"

| 비임상 시험 | 내용 |
|------------|------|
| in vitro hERG 시험 | IKr 채널 억제 평가 |
| in vivo QTc 시험 | 동물에서 QTc 연장 평가 |
| 심장 이온 채널 패널 | 여러 이온 채널 효과 종합 평가 |
| 컴퓨터 모델링 | 통합 위험 예측 |

E14/S7B 통합 평가에서 비임상이 양성인 경우에도 일반적으로 임상 평가 필요.
