## 1. 약사법 제34조 — 임상시험의 계획 승인

### 조문 요지

**제34조(임상시험의 계획 승인 등)**

**제1항**: 의약품등으로 임상시험을 하려는 자는 임상시험계획서를 작성하여 **식품의약품안전처장의 승인**을 받아야 한다. 승인받은 사항을 변경하려는 경우에도 변경승인을 받아야 한다. 다만, 임상시험계획서 중 총리령으로 정하는 경미한 사항을 변경하려는 경우에는 식품의약품안전처장에게 보고하면 된다.

**제2항**: 임상시험 실시 시 준수 요건:
- 지정된 임상시험 실시기관(약사법 제34조의2)에서 실시
- 적합한 제조시설에서 제조된 의약품 사용
- 임상시험 참여 모집 공고 시 시험 명칭, 목적, 방법 등 공지
- 임상시험 대상자 피해보상을 위한 보험 또는 공제 가입

**제3항(승인 면제)**: 이미 시판 중인 의약품을 허가·신고 범위 내에서 임상적 효과를 관찰하는 경우 등 총리령이 정하는 임상시험은 승인을 받지 아니할 수 있다.

**제4항(승인 제한 및 중지)**: 식품의약품안전처장은 안전성·유효성 문제 시 승인을 제한하거나, 위반 발생 시 임상시험을 중지하거나 금지하는 등 필요한 조치를 명할 수 있다.

### 실무 적용

| 항목 | 내용 |
|------|------|
| 승인 기관 | 식품의약품안전처 (MFDS) — 임상약리과 또는 임상제도과 |
| 심사 기간 | 신청일로부터 30일 이내 (법정 심사 기간) |
| 보완 요청 | 보완 요청 시 기간 정지; 보완 후 재심사 |
| 변경 승인 | 프로토콜 변경, 대상자 수 변경 등 → 사전 변경 승인 신청 |
| 변경 보고 | 경미한 사항(식약처 고시 기준) → 사후 보고 |

## 2. 약사법 제34조의2 — 임상시험 실시기관의 지정

### 조문 요지

**제34조의2(임상시험 실시기관의 지정 등)**

임상시험을 실시하려는 기관은 식품의약품안전처장으로부터 지정을 받아야 한다. 지정 요건은 총리령(의약품등의 안전에 관한 규칙)으로 정한다.

### 지정 기관 요건

| 요건 | 내용 |
|------|------|
| **시설** | 입원 병상, 응급 장비(심폐소생 장비, AED 등), 처치실, 채혈실, 검체 보관 시설 |
| **인력** | 시험책임자(의사 자격자), GCP 교육 이수자, 임상시험 코디네이터(CRC) |
| **IRB** | 기관 내 또는 지역 IRB 보유 (독립적 운영) |
| **표준작업지침서(SOP)** | 임상시험 관리 SOP 보유 및 운영 |
| **GMP 시설** | 임상시험용 의약품 제조소의 경우 GMP 시설 요건 충족 |
| **QA 인력** | QA 담당 인력 필수 배치 (2024년 개정 반영) |

## 3. 의약품등의 안전에 관한 규칙 제30조~제36조 — 임상시험 관련 조항

> 본 규칙은 총리령으로, 의약품 임상시험 관리기준(KGCP)을 별표 4로 규정하고 있다.

### 제30조 (임상시험의 실시기준 등)

의약품 임상시험은 의약품등의 안전에 관한 규칙 **제30조** 및 그 **[별표 4] 의약품 임상시험 관리기준(KGCP)**에 따라 실시한다.

→ 상세 내용은 `korea_kgcp.md` 참조

### 제31조 (임상시험 실시기관의 지정 요건)

임상시험 실시기관 지정 신청에 필요한 서류 및 요건(제34조의2 위임 규정):
- 시설 기준: 입원 병상, 검체 보관, ECG 장비 등
- 인력 기준: 시험책임자 자격, 코디네이터 인원
- IRB 운영 기준: 위원 구성, 독립성, 운영 절차

### 제32조~제36조 (임상시험 보고 및 관리)

| 조항 | 내용 |
|------|------|
| **제32조** | 임상시험 실시기관 지정 취소 사유; 취소 절차 |
| **제33조** | 임상시험용 의약품의 관리 기준 (제조·보관·처방·반납) |
| **제34조** | 임상시험 결과 보고서 제출 요건 |
| **제35조** | 임상시험 대상자 피해보상 보험 기준 |
| **제36조** | 임상시험 관련 기록 보존 기준 |

## 4. 의약품 임상시험 계획 승인에 관한 규정 (IND 규정)

**식약처 고시 제2024-2호** (2024.1.16. 시행)

### 4.1 IND 신청 서류 목록

| 서류 | 비고 |
|------|------|
| 임상시험계획 승인 신청서 | 별지 제23호 서식 |
| 임상시험계획서 (Protocol) | ICH E6(R3) Annex 1 필수 요소 포함 |
| 임상시험자 자료집 (IB) | 비임상 및 임상 데이터 요약 |
| 동의설명서 초안 | KGCP 및 생명윤리법 요건 충족 |
| 비임상 독성 자료 | GLP 단회·반복투여 독성시험; NOAEL 명시 |
| 안전성 약리 자료 | ICH S7A(일반 안전성 약리), S7B(심독성) |
| CMC 자료 (제조·품질) | 임상시험용 의약품 규격·제조 방법; GMP 적합성 |
| 초기 용량 산출 보고서 | MRSD/MABEL 산출 근거 (FIH의 경우) |
| IRB 심의 확인서 | 심의 완료 또는 심의 예정 확인서 |
| 피해보상 보험 증명 | 보험증권 사본 |
| 임상시험실시기관 지정서 | 식약처 지정 기관 확인서 |
| 수수료 납부 영수증 | 규정 수수료 |

### 4.2 변경 승인이 필요한 주요 사항

| 변경 사항 | 처리 |
|----------|------|
| 대상자 선정·제외 기준 변경 | 변경 승인 |
| 투여 용량·용법 변경 | 변경 승인 |
| 1차 평가변수 변경 | 변경 승인 |
| 대상자 수 30% 이상 증가 | 변경 승인 |
| 시험 기간 대폭 연장 | 변경 승인 |
| 임상시험실시기관 추가/변경 | 변경 승인 또는 보고 (경중에 따라) |

### 4.3 변경 보고로 가능한 사항

| 변경 사항 | 처리 |
|----------|------|
| 행정적 사항(연락처, 주소 등) 변경 | 변경 보고 |
| 편의상 오류(오탈자) 수정 | 변경 보고 |
| 시험 기간 소폭 연장 | 변경 보고 |
| 대상자 수 소폭 변경 (30% 미만) | 변경 보고 (경우에 따라) |

## 5. 시험 개시·종료·중단 보고 요건

| 보고 사유 | 보고 시한 | 보고 기관 |
|----------|---------|---------|
| **시험 개시** (최초 대상자 투여) | 개시 후 지체 없이 | 식약처, IRB |
| **중간 안전성 보고** | SUSAR 발생 7일/15일 이내 | 식약처, IRB, 시험책임자 |
| **시험 조기 종료/중단** | 결정 즉시 → 30일 이내 이유 보고 | 식약처, IRB |
| **시험 완료** | 마지막 대상자 마지막 방문 후 | 식약처, IRB |
| **임상시험 결과 보고서 제출** | 시험 완료 후 6개월~1년 이내 | 식약처 |

## 6. 임상시험 관련 행정 처분

| 위반 사항 | 처분 |
|----------|------|
| IND 승인 없이 임상시험 실시 | 임상시험 중지 명령; 형사 처벌(약사법 제93조~제97조) |
| KGCP 위반 | 임상시험 중지 명령; 기관 지정 취소 |
| 자료 위조·변조 | 형사 처벌; 허가 취소 |
| 보고 의무 위반 | 행정 처분; 과태료 |

## 7. 임상시험 대상자 피해보상

| 항목 | 내용 |
|------|------|
| **법적 근거** | 약사법 제34조 제2항; 의약품등의 안전에 관한 규칙 제35조 |
| **의무** | 임상시험 의뢰자는 임상시험 참여로 인한 피해보상을 위해 보험 또는 공제 가입 의무 |
| **보상 범위** | 임상시험 참여로 인한 사망, 장애, 질병 치료비 등 |
| **동의서 명시** | ICF에 피해보상 내용, 보상 신청 절차 명시 의무 |
