## 1. 목적 및 원칙

KGCP는 임상시험의 과학적 질과 윤리성을 보증하고, 임상시험 대상자의 권리·안전·복지를 보호하며, 임상시험 결과의 신뢰성을 확보하기 위한 기준이다. ICH E6(R2) GCP와 실질적으로 동일하며, ICH E6(R3) 원칙을 반영하고 있다.

**KGCP 10대 원칙:**
1. 헬싱키 선언 및 GCP 원칙 준수
2. 임상시험 개시 전 위험-이익 평가 및 정당화
3. 대상자 권리·안전·복지가 과학적 이익보다 우선
4. 비임상 및 임상 정보가 계획 임상시험을 지지해야 함
5. 과학적으로 건전한 프로토콜 필요
6. IRB/IEC의 사전 승인 후 실시
7. 대상자 의료 결정은 자격 있는 의사/치과의사가 수행
8. 시험 참여 자격·전문성을 갖춘 의뢰자, 모니터요원, 시험자 필요
9. 자발적 서면 동의 (prior written informed consent) 필수
10. 모든 임상시험 정보 기록·처리·보존·보고의 정확성과 완전성 보장

## 2. 시험의뢰자(Sponsor) 의무

### 2.1 일반 의무

| 의무 | 내용 |
|------|------|
| **IND 신청** | 식약처 임상시험계획 승인 신청 및 유지 |
| **IB 작성·배포** | 임상시험자 자료집(IB) 작성; 개정 시 모든 관련 기관에 배포 |
| **프로토콜 작성** | ICH E6(R3) Annex 1 기반 프로토콜 작성·승인·배포 |
| **CRO 계약 및 감독** | CRO에 업무 위탁 시 계약서 작성; 위탁 업무 감독 의무 |
| **모니터링 계획** | 위험기반 모니터링(RBM) 또는 일반 모니터링 계획 수립·실행 |
| **안전성 보고** | SUSAR, DSUR 등 식약처·IRB에 적시 보고 |
| **임상시험용 의약품 관리** | 제조·품질(GMP), 포장·라벨링, 공급·회수, 온도 관리 |
| **데이터 관리** | EDC/CRF 시스템 구축; 데이터 완전성 보장; 21 CFR Part 11 또는 동등 기준 |
| **통계분석계획(SAP)** | 시험 종료 전 SAP 확정; 통계분석 감독 |
| **임상시험 보고서(CSR)** | 시험 완료 후 CSR 작성·제출 (ICH E3 형식) |
| **기록 보존** | 모든 임상시험 기록 규정 기간 동안 보존 |

### 2.2 연구자 임상시험(IIT) 시 의뢰자 역할

연구자 주도 임상시험(Investigator-Initiated Trial)의 경우 시험책임자가 의뢰자 역할도 수행하며, 동일한 의뢰자 의무를 이행해야 한다.

## 3. 시험책임자(Principal Investigator) 자격 및 의무

### 3.1 자격 요건

| 요건 | 내용 |
|------|------|
| **의사 자격** | 의사면허 소지자 (치과의사 허용 — 해당 전문 분야) |
| **교육·훈련** | GCP 교육 이수; 매년 1회 이상 보수 교육 의무 (1년 경과 시 IRB 심의 신청 불가) |
| **경험** | 해당 임상시험 분야 관련 교육·훈련 및 경험 보유 |
| **규정 숙지** | KGCP 및 관계 법령 숙지·준수 의무 |

### 3.2 시험책임자 주요 의무

| 의무 | 구체적 내용 |
|------|----------|
| **충분한 자원 확보** | 적합한 시설·인력·시간 확보; 임상시험 기간 중 시험 능력 유지 |
| **프로토콜 준수** | 승인된 프로토콜대로 시험 실시; 무단 이탈 금지 |
| **IRB 보고·승인** | 프로토콜 최초 심의; 연간 지속 심의; 중요 변경 심의 |
| **서면 동의 취득** | 모든 대상자로부터 서면 동의 취득 (동의 전 충분한 설명 의무) |
| **대상자 의료 관리** | 임상시험 관련 의학적 결정; 이상반응 치료 의무 |
| **이상반응 평가 및 보고** | 모든 AE 수집·평가; SAE 즉시 의뢰자 보고; 예상치 못한 SAE는 IRB 즉시 보고 |
| **임상시험용 의약품 관리** | 수령·보관·처방·반납·파기 기록 유지 |
| **모니터링·감사 협조** | 의뢰자 모니터요원, 감사담당자, 식약처 실태조사관에 협조 |
| **기록 유지** | 모든 CRF 및 원자료 정확하게 기록; 수정 시 원본 알아볼 수 있게 한 줄 그어 수정, 서명·날짜 |
| **보고서 제출** | 시험 종료 후 시험기관 보고서(Site Report) 작성; IRB에 최종 보고 |

### 3.3 시험담당자(Sub-Investigator)

- 시험책임자의 위임하에 임상시험 관련 업무 수행
- 위임장(Delegation Log)에 위임 업무 명시 필요
- GCP 교육 이수 의무

## 4. IRB(기관생명윤리위원회) 심의 요건

### 4.1 IRB 구성 요건

| 요건 | 내용 |
|------|------|
| **최소 위원 수** | 5명 이상 |
| **성별 다양성** | 남·녀 각 1명 이상 |
| **전문성** | 과학자 위원 + 비과학자 위원 최소 각 1명 |
| **외부 위원** | 기관 외부 비계열 위원 최소 1명 |
| **독립성** | 시험의뢰자·시험책임자로부터 독립 운영 |

### 4.2 심의 대상 및 유형

| 심의 유형 | 내용 |
|----------|------|
| **최초 심의** | 임상시험 개시 전 필수; 프로토콜, ICF, IB, 모집 광고 등 |
| **지속 심의 (연간)** | 최소 1년에 1회 임상시험 지속 여부 검토 |
| **변경 심의** | 프로토콜 또는 ICF 중요 변경 시 사전 심의 |
| **이상반응 심의** | SUSAR 또는 중대한 프로토콜 위반 시 긴급 심의 |
| **종료 심의** | 시험 완료 또는 조기 종료 시 최종 보고 |

### 4.3 ICF 심의 체크포인트

IRB는 ICF가 다음 KGCP 요건을 충족하는지 확인:
- 자발적 참여 및 철회 자유 명시
- 시험의 목적·절차·예상 이익 및 위험 기술
- 기밀 보장 범위 및 제3자 접근 가능성 기술
- 피해보상 내용 및 신청 절차
- 연락 가능한 시험책임자 정보
- 참여 거부 또는 철회 시 불이익 없음 명시

## 5. 서면 동의(Informed Consent) 취득 절차

### 5.1 동의 취득 원칙

| 원칙 | 내용 |
|------|------|
| **사전 동의** | 임상시험 관련 모든 절차 시행 전 동의 취득 |
| **충분한 설명** | 대상자가 이해할 수 있는 언어·어휘로 설명; 질문에 충분히 답변 |
| **서면 동의** | 대상자(또는 법정대리인) 및 설명자의 서명·날짜 기재 |
| **사본 제공** | 서명된 ICF 사본 대상자에게 제공 의무 |
| **철회 자유** | 언제든지 동의 철회 가능; 불이익 없음 보장 |
| **재동의** | ICF 중요 변경 시 새로운 ICF로 재동의 취득 |

### 5.2 취약한 대상자(Vulnerable Subjects)

| 대상 | 추가 요건 |
|------|---------|
| **미성년자** | 법정대리인(부모 또는 친권자) 동의 + 미성년자 동의(assent, 가능한 경우) |
| **의사결정 능력 없는 성인** | 법정대리인 동의; 가능하면 본인 동의도 취득 |
| **임신부** | 태아에 대한 위험성 설명; 추가 고려 사항 |
| **위계 관계 있는 자** | 과도한 영향 방지; 독립적 설명자 활용 권장 |
| **응급 상황** | 긴급 동의 면제 절차 (IRB 사전 승인 필요) |

## 6. 이상약물반응(ADR) 보고 체계

### 6.1 보고 기간 기준

| 이상반응 유형 | 보고 기간 | 보고 대상 |
|------------|---------|---------|
| **사망 또는 생명위협 SUSAR** | 최초 인지 후 **7일** 이내 초기 보고; 추가 정보 **8일** 이내(= 최초 인지 후 15일 이내) | 식약처, IRB, 시험책임자 |
| **기타 SUSAR** (비생명위협) | 최초 인지 후 **15일** 이내 | 식약처, IRB, 시험책임자 |
| **예상된 중대한 이상약물반응** | DSUR(연간 보고)에 포함 | 식약처 |
| **비중대 이상약물반응** | DSUR에 포함 | 식약처 |
| **SAE (현장)** | 시험책임자 → 의뢰자: **24시간** 이내 |
| **SAE (의뢰자)** | 의뢰자 → 식약처·IRB·시험책임자: 규정 기간 이내 |

### 6.2 SUSAR 정의

- **Serious**: 사망, 생명위협, 입원(또는 연장), 영구적 장애, 선천성 기형, 기타 의학적으로 중요
- **Unexpected**: IB에 기재되지 않았거나, IB 기재 내용과 성질·중증도 측면에서 다른
- **Adverse Drug Reaction**: 인과관계를 완전히 배제할 수 없는 이상반응

### 6.3 보고 방식

- 식약처: 전자 보고 시스템 (이약시스템 또는 지정 전자 방식)
- IRB: 기관별 양식으로 보고
- 주요 정보: 대상자 식별 코드, 시험 번호, 이상반응명(MedDRA PT), 발생일, 중증도, 인과관계, 결과

## 7. 기록 보존 요건

### 7.1 보존 기간

| 문서 종류 | 보존 기간 |
|---------|---------|
| **임상시험 기본 문서 (Essential Documents)** | 시험 완료 또는 조기 중단 후 최소 **15년** (또는 제품 허가 후 최소 2년 중 더 긴 기간) |
| **IRB 심의 기록** | IRB 보존 기간 기준 (통상 3년~15년) |
| **임상시험용 의약품 기록** | 해당 허가 만료 또는 시험 완료 후 최소 2년 |
| **대상자 동의서** | 시험 완료 후 최소 15년 |
| **CRF/원자료** | 시험 완료 후 최소 15년 |

### 7.2 필수 보존 문서 목록 (Essential Documents)

**시험 개시 전:**
- IND 승인서; IRB 승인서; IRB 위원 명단; ICF 최종본
- IB; 프로토콜; CRF 양식; 대상자 모집 광고
- 시험책임자 CV; 관련 인력 자격증; 위임장(Delegation Log)
- 임상시험 배상책임보험 증명서
- 임상시험용 의약품 이송 확인서; 기준서(specification); COA

**시험 진행 중:**
- 프로토콜 개정본; ICF 개정본; IRB 재심의 결과
- SAE 보고서; SUSAR 보고서; 모니터링 방문 보고서
- 방문 일지; CRF(작성 완료); 의약품 수령·불출·반납 기록
- 임상검사 결과지; ECG 등 검사 기록

**시험 완료 후:**
- 최종 CRF; 대상자 등록 목록; 무작위 배정 목록(맹검 해제 후)
- 통계분석 계획 및 결과; CSR 초안 및 최종본
- 감사 보고서 (해당 시)

## 8. 모니터링 요건 (Monitoring Requirements)

### 8.1 모니터링 목적

- 임상시험이 프로토콜, KGCP, 관계 법령에 따라 수행되고 있는지 확인
- 기록된 데이터가 원자료(source data)와 일치하는지 확인 (Source Data Verification, SDV)

### 8.2 위험기반 모니터링 (Risk-Based Monitoring, RBM)

ICH E6(R3)의 품질관리(QbD) 원칙에 따라:

| 모니터링 유형 | 내용 |
|------------|------|
| **현장 방문 모니터링** | 시험 기관 직접 방문; 원자료 확인; 시설·인력 점검 |
| **원격/중앙 모니터링** | EDC 데이터 원격 검토; 통계적 모니터링; 이상값 탐지 |
| **위험 기반 접근** | 위험이 높은 영역에 집중; 저위험 영역은 원격 모니터링으로 대체 가능 |

### 8.3 모니터링 빈도

| 시험 단계 | 최소 빈도 |
|---------|---------|
| **시험 개시 (SIV, Site Initiation Visit)** | 최초 대상자 등록 전 1회 |
| **진행 중 모니터링** | 위험 기반으로 결정; 통상 1~3개월에 1회 현장 방문 |
| **종료 방문 (CSV, Close-out Visit)** | 마지막 대상자 마지막 방문 후 |

### 8.4 모니터링 보고서

- 각 모니터링 방문 후 서면 보고서 작성 (의뢰자 기록)
- 발견 사항, 이탈(deviation), 권고 사항 기재
- 심각한 이탈(critical deviation)은 즉시 의뢰자 책임자에게 에스컬레이션

## 9. 임상시험용 의약품 관리

| 항목 | 요건 |
|------|------|
| **제조·품질** | GMP 기준 제조; COA(시험성적서) 동반 |
| **이중눈가림** | 눈가림 코드 봉인; 비상 코드 해제 절차 프로토콜 명시 |
| **온도 관리** | 포장·운반·보관 온도 모니터링 기록 유지 |
| **접수·불출 기록** | 수령량, 불출량, 반납량, 파기량 기록; 대사 일치 확인 |
| **미사용 의약품** | 사용 후 미사용분 반납 또는 현장 파기; 기록 유지 |
| **라벨링** | 임상시험용임을 명시; 프로토콜 번호, 용법·용량, 보관 조건 기재 |
