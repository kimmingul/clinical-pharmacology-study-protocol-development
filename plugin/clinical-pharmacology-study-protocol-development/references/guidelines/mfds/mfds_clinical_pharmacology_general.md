## 1. 임상약리시험의 분류 (Classification of Clinical Pharmacology Studies)

| 시험 유형 | 목적 | 통상 Phase |
|----------|------|-----------|
| **FIH (SAD/MAD)** | 최초 인체 적용; 안전성·내약성·기초 PK 확립 | Phase 1 |
| **생물학적동등성 (BE)** | 제네릭 의약품의 오리지널 대비 동등성 입증 | Phase 1 |
| **약물상호작용 (DDI)** | 병용약물의 PK/PD 영향 정량적 평가 | Phase 1 |
| **식이 영향 (Food Effect, FE)** | 음식이 약물 흡수·PK에 미치는 영향 | Phase 1 |
| **QTc 시험 (TQT/CQTS)** | 심장 재분극(QTc 연장) 위험 평가 | Phase 1 |
| **ADME** | 흡수·분포·대사·배설 프로파일 (방사성동위원소 사용 포함) | Phase 1 |
| **특수 집단 PK** | 간장애·신장애·고령자·소아 대상 PK 평가 | Phase 1~2 |
| **용량-반응 (D-R)** | 최적 용량 범위 탐색; PK-PD 관계 수립 | Phase 1~2 |
| **집단 PK (PopPK)** | 대규모 PK 데이터 기반 개체간 변동성 모델링 | Phase 2~3 |

## 2. IND(임상시험계획 승인) 신청 절차

### 2.1 법적 근거

- **약사법 제34조**: 임상시험을 하려는 자는 임상시험계획서를 작성하여 **식품의약품안전처장의 승인**을 받아야 한다.
- **의약품 임상시험 계획 승인에 관한 규정** (식약처 고시 제2024-2호, 2024.1.16. 시행): IND 신청 절차, 서류 목록, 심사 기간 규정

### 2.2 IND 신청 서류 목록

| 서류 | 내용 |
|------|------|
| **임상시험계획 승인 신청서** | 별지 제23호 서식 |
| **임상시험계획서 (Protocol)** | ICH E6(R3) Annex 1 기반 필수 요소 포함 |
| **임상시험자 자료집 (IB)** | 비임상 및 임상 데이터 요약 |
| **동의설명서 (ICF) 초안** | KGCP 및 생명윤리법 요건 충족 |
| **비임상 독성 자료** | GLP 독성시험 결과, NOAEL, 독성 프로파일 |
| **안전성 약리 자료** | 심혈관계, 중추신경계, 호흡기계 평가 |
| **CMC 자료** | 임상시험용 의약품 제조·품질 관리 자료 |
| **초기 용량 산출 보고서** | MRSD/MABEL 산출 근거 (FIH의 경우) |
| **IRB 심의 서류** | IRB 승인서 또는 심의 예정 확인서 |
| **피해보상 보험 증명** | 임상시험 배상책임보험 가입 증명 |
| **임상시험실시기관 지정서** | 약사법 제34조의2 지정 기관 확인 |

### 2.3 심사 기간 및 승인 유형

| 구분 | 내용 |
|------|------|
| **표준 심사** | 신청일로부터 30일 이내 승인 또는 보완 요청 |
| **조건부 승인** | 조건 부기 후 시험 진행 가능 |
| **변경 승인** | 중요 변경사항 (프로토콜 변경, 대상자 수 변경 등) → 사전 변경 승인 |
| **변경 보고** | 경미한 변경사항 → 사후 보고 (식약처 고시 기준에 따름) |
| **승인 면제** | 시판 중인 의약품으로 허가 범위 내 임상효과 관찰 등 총리령이 정하는 경우 |

### 2.4 IND 이후 관리

| 단계 | 요건 |
|------|------|
| **시험 개시 보고** | 최초 대상자 투여 후 지체 없이 식약처 보고 |
| **연간 보고 (DSURㅡ)** | 개발 단계 정기 안전성 업데이트 보고서 (식약처 제출) |
| **시험 종료 보고** | 최종 투여 후 일정 기간 내 식약처 보고 |
| **시험 결과 보고** | 임상시험 결과 보고서(CSR) 최종본 제출 |
| **조기 중단 보고** | 조기 종료·중단 시 식약처 및 IRB 즉시 보고, 사유 설명 |

## 4. 시험대상자 요건 (Subject Requirements)

### 4.1 건강한 자원자 대상 시험

| 항목 | MFDS 요건 |
|------|----------|
| **연령** | 만 19세 이상 (통상 19~55세; 노인 연구 시 조정) |
| **건강 상태** | 병력, 이학적 검사, 임상검사(혈액학, 생화학, 요검사, ECG)로 건강 확인 |
| **BMI** | 통상 18.5~30.0 kg/m² (시험별 조정 가능) |
| **선정/제외 기준** | 프로토콜에 명확히 기술 (ICH E6(R3) Annex 1 요건 충족) |
| **동의** | 서면 동의(ICF); 자발적 참여; 동의 철회 자유 보장 |
| **보상** | 합리적 범위 내 (과도한 금전적 유인 금지; IRB 검토 대상) |
| **washout** | 이전 약물 복용 후 충분한 washout 기간 |
| **임신 가능성** | 임신 가능성 있는 여성: 임신검사 음성 확인; 적절한 피임 요건 |
| **헌혈 금지** | 채혈 기간 중 헌혈 금지; 철결핍성 빈혈 예방 |

### 4.2 환자 대상 시험 (해당 시)

| 항목 | 추가 고려 사항 |
|------|-------------|
| **적응증** | 해당 질환의 진단 기준 명시 |
| **병용약물** | 허용/금지 병용약물 목록 명시; washout 요건 |
| **위험-이익 평가** | 건강 자원자 대비 환자 특이 위험성 강화 평가 |
| **취약한 대상자** | 추가적인 보호 조치 (KGCP 및 생명윤리법 요건) |

## 5. 약동학(PK) 평가 일반 원칙

### 5.1 PK 파라미터

| 파라미터 | 의미 | 산출 방법 |
|---------|------|---------|
| **Cmax** | 최고 혈중농도 | 실측값 |
| **Tmax** | 최고 혈중농도 도달 시간 | 실측값 |
| **AUC₀₋ₜ** | 최종 정량 시점까지 곡선하면적 | 선형-로그 사다리꼴법 |
| **AUC₀₋∞** | 무한대까지 곡선하면적 | AUC₀₋ₜ + Clast/λz |
| **t½ (λz)** | 말기 소실 반감기 | 0.693 / λz |
| **CL/F** | 겉보기 청소율 | Dose / AUC₀₋∞ |
| **Vd/F** | 겉보기 분포용적 | CL/F / λz |
| **AUCτ** | 정상상태 투여간격 AUC | 선형-로그 사다리꼴법 |
| **Css,max** | 정상상태 최고 혈중농도 | 실측값 |
| **Css,min** | 정상상태 최저 혈중농도 | 실측값 |
| **RA** | 누적비 | AUCτ,ss / AUC₀₋τ |

### 5.2 생체시료분석법

| 항목 | 요건 |
|------|------|
| **분석법 밸리데이션** | 식약처 생체시료분석법 밸리데이션 가이드라인 준수 (정확도, 정밀도, 선택성, 감도, 직선성, 회수율, 매질 효과, 안정성) |
| **분석 방법** | LC-MS/MS 원칙; 정당한 이유 있는 경우 다른 방법 허용 |
| **분석기관** | GLP 인증 분석기관 또는 밸리데이션된 분석법 보유 기관 |
| **블라인드 재분석(ISR)** | 시험 검체의 최소 5~10% 재분석; 재분석 결과 일치 확인 |

## 7. 안전성 평가 (Safety Assessment)

| 항목 | 요건 |
|------|------|
| **이상반응(AE) 수집** | 모든 이상반응 수집; MedDRA 코딩; CTCAE 등급; 인과관계 평가 |
| **중대한 이상반응(SAE)** | 정의: 사망, 생명위협, 입원, 장애, 선천성 기형, 기타 의학적으로 중요 |
| **SAE 보고** | 발생 즉시 IRB 보고; 식약처 7일(사망·생명위협)/15일(기타 예상치 못한) 이내 보고 |
| **SUSAR** | 예상하지 못한 중대한 이상약물반응: 15일 이내 보고 (사망·생명위협: 7일 이내 초기 보고 + 8일 이내 추가 정보) |
| **활력징후** | 투여 전후 정기적 측정 (혈압, 맥박, 체온, 호흡수) |
| **심전도(ECG)** | 12-lead ECG; 프로토콜에 정의된 시점에서 측정; QT/QTc 분석 |
| **임상검사** | 혈액학, 생화학(간기능, 신기능, 전해질), 요검사 — 투여 전후 비교 |

## 10. QTc 시험 (Thorough QT / Concentration-QT, TQT/CQTS)

ICH E14에 따른 QTc 시험 요건 (ICH S7B와 연계):

| 항목 | 요건 |
|------|------|
| **적용 시기** | Phase 2~3 진입 전 (또는 Phase 1에서 농도-QTc 분석) |
| **설계** | TQT: 4-arm(시험약, 위약, 양성 대조, 시험약 고용량); 또는 CQTS(집중 ECG + 집단 PK) |
| **ECG 분석** | Fridericia 보정(QTcF) 원칙; 과도한 QT 연장(>10ms) 시 양성 결과로 판단 |
| **양성 대조** | Moxifloxacin 400mg 경구 (TQT) |
| **CQTS 대체** | 충분한 농도-QTc 관계 데이터 있으면 TQT 대체 가능 (ICH E14 개정 반영) |

## 11. ADME 시험 (흡수·분포·대사·배설)

| 항목 | 요건 |
|------|------|
| **방사성 동위원소 표지** | [¹⁴C] 표지 약물 사용 원칙 (mass balance study) |
| **배설 평가** | 소변·대변·호기를 통한 누적 배설률 측정 |
| **대사체 프로파일** | 주요 대사체 구조 동정; 인체 특이 대사체 확인 |
| **허가 요건** | 인체 특이 대사체 > 10%인 경우 별도 독성 평가 필요 (ICH M3(R2)) |
| **수집 기간** | 방사능 회수율 95% 이상 또는 최소 7일 |

## 12. 특수 집단 PK 시험

| 집단 | 주요 평가 내용 |
|------|-------------|
| **신장애** | eGFR 기반 중증도 분류(정상/경증/중등도/중증/투석); AUC 변화 → 용량 조절 지침 |
| **간장애** | Child-Pugh 분류(A/B/C); 대사 기능 저하 → 용량 조절 지침 |
| **고령자 (≥65세)** | 신기능·체성분·병용약물 영향; 집단 PK 모형으로 대체 가능 |
| **소아** | MFDS 소아의약품 개발 가이드라인 별도 적용; 연령군별 용량 탐색 |
| **임산부** | 위험-이익 평가; 임신 레지스트리 권장 |
