## 1. 적용 범위 (임상시험 맥락)

임상시험에서 수집되는 개인정보(대상자 성명, 생년월일, 건강정보, 유전정보 등)는 PIPA의 적용을 받는다. 임상시험 의뢰자·수탁자·시험기관은 모두 개인정보처리자로서 의무를 이행해야 한다.

---

## 2. 개인정보 수집·이용 동의 — 제15조

### 2.1 수집·이용의 법적 근거 (제15조 제1항)

개인정보처리자는 다음 각 호 중 하나에 해당하는 경우 개인정보를 수집하고, 그 목적 범위에서 이용할 수 있다:

| 호 | 근거 | 임상시험 적용 |
|---|------|------------|
| **1호** | 정보주체의 **동의** | 임상시험 ICF에서 대상자 동의 취득 (가장 일반적) |
| **2호** | 법률에 특별한 규정 또는 법령상 의무 준수 | 약사법상 안전성 보고 의무 이행 |
| **3호** | 공공기관의 법령 소관 업무 수행 | 식약처 등 규제기관의 업무 |
| **4호** | 계약 체결·이행 | 연구 참여 계약 |
| **5호** | 급박한 생명·신체·재산 이익 | 응급 상황에서의 개인정보 처리 |
| **6호** | 정당한 이익 달성 (명백히 우선) | 임상시험에서는 제한적 적용 |
| **7호** | 공중위생 등 공공 안전 긴급 필요 | 팬데믹 등 특수 상황 |

### 2.2 동의 취득 시 필수 고지 항목 (제15조 제2항) — 6개 항목

| 번호 | 고지 항목 | 임상시험 ICF 적용 예 |
|-----|----------|------------------|
| **①** | 개인정보의 수집·이용 **목적** | "임상시험 참여 관리, 안전성·유효성 평가, 규제기관 보고" |
| **②** | 수집하려는 개인정보의 **항목** | "성명, 생년월일, 연락처, 건강정보(병력·검사치·유전정보), 혈액·소변 검체 분석 결과" |
| **③** | 개인정보의 **보유 및 이용 기간** | "임상시험 종료 후 15년 (KGCP 기준); 허가 후 최소 2년 중 긴 기간" |
| **④** | 동의를 거부할 **권리** 및 거부 시 **불이익** | "동의 거부 가능; 거부 시 임상시험 참여 불가" |

> **주의**: 항목 ①②③④가 모두 명시되어야 적법한 동의

## 3. 개인정보 제3자 제공 동의 — 제17조

### 3.1 제3자 제공 근거 (제17조 제1항)

- **1호**: 정보주체의 동의
- **2호**: 법적 의무(법률 제15조제1항 제2~3·5~7호)에 따른 수집 범위 내 제공

임상시험에서 개인정보를 의뢰자, 규제기관(식약처, 외국 규제기관), CRO, 중앙실험실, IRB 등에 제공할 때는 제3자 제공 동의가 필요하다.

### 3.2 제3자 제공 동의 시 필수 고지 항목 (제17조 제2항) — 5개 항목

| 번호 | 고지 항목 | 임상시험 ICF 적용 예 |
|-----|----------|------------------|
| **①** | 개인정보를 **제공받는 자** | "의뢰자(회사명), CRO(회사명), 식약처, 외국 규제기관(FDA, EMA 등), 중앙실험실" |
| **②** | 제공받는 자의 개인정보 **이용 목적** | "임상시험 데이터 관리, 통계분석, 안전성 평가, 규제 심사" |
| **③** | 제공하는 개인정보의 **항목** | "대상자 코드(pseudonym), 연령, 성별, 건강정보, PK 데이터, 이상반응 정보" |
| **④** | 제공받는 자의 개인정보 **보유 및 이용 기간** | "임상시험 종료 후 15년 (또는 허가 후 최소 2년 중 긴 기간)" |
| **⑤** | 동의를 **거부할 권리** 및 **불이익** | "동의 거부 가능; 거부 시 임상시험 참여 불가" |

### 3.3 다국가 임상시험에서의 국외 이전

국외 제3자(FDA, EMA, 해외 의뢰자 본사 등)에게 개인정보를 이전할 때:
- 별도 동의 또는 보호 수준 적정성 결정(adequacy decision) 국가 해당 여부 확인
- PIPA 제28조의8 (국외 이전) 요건 충족 필요
- ICF에 국외 이전 국가 및 기관 명시 권장

## 4. 민감정보 처리 — 제23조

### 4.1 민감정보 범위

임상시험과 관련된 주요 민감정보:

| 민감정보 종류 | 임상시험 적용 예 |
|------------|--------------|
| **건강에 관한 정보** | 병력, 진단명, 임상검사 결과, 이상반응, 복용 약물 |
| **유전자검사 등의 결과로 얻어진 유전정보** | 약물유전체 분석, 바이오마커 유전자형 |
| **범죄경력 자료** | (임상시험에서는 일반적으로 비해당) |
| **신체적 특징 (생체인식 정보)** | 망막, 지문 (일부 시험) |

### 4.2 민감정보 처리 요건

기본적으로 처리 금지; 다음의 경우에만 허용:

| 요건 | 내용 |
|------|------|
| **별도 동의** | 정보주체로부터 **별도 서면 동의** 취득 (수집·이용 동의와 분리하여 명시적 동의) |
| **법령 근거** | 법률에서 민감정보 처리를 명시적으로 요구하거나 허용하는 경우 |
| **안전성 확보 조치** | 민감정보가 분실·도난·유출·위조·변조·훼손되지 않도록 강화된 안전조치 |

### 4.3 임상시험 ICF에서의 민감정보 동의 문구 예시

```
[민감정보 처리 동의]
귀하의 건강정보(병력, 임상검사 결과, 이상반응 기록 등) 및
유전정보(해당 시: 약물유전체 분석 결과)는 민감정보로서
개인정보 보호법 제23조에 따라 별도 동의가 필요합니다.

위 민감정보의 수집·이용 및 제3자 제공에 동의하십니까?

□ 동의함  □ 동의하지 않음 (미동의 시 임상시험 참여 불가)
```

## 5. 동의 방법 — 제22조

### 5.1 적법한 동의 방법

| 방법 | 요건 |
|------|------|
| **서면 동의** | 임상시험에서 원칙 (KGCP 요건과 일치); 날인 또는 서명 |
| **전자 동의 (eConsent)** | 전자적 동의도 유효; 동의 내용 명확히 구분; 기록 보존 |
| **명확한 구분** | 수집·이용 동의, 제3자 제공 동의, 민감정보 동의를 각각 명확히 구분 |
| **선택 구조** | 개인정보 처리와 서비스 제공을 결합한 동의 강요 금지 |
| **명확한 표시** | 동의 여부를 명확히 알아볼 수 있도록 표시 (체크박스 등) |

### 5.2 임상시험 ICF 동의 구조 (권장)

```
1. 임상시험 참여 동의 (KGCP 요건)
   □ 동의함

2. 개인정보 수집·이용 동의 [필수]
   - 목적, 항목, 보유기간, 거부 시 불이익
   □ 동의함  □ 동의하지 않음

3. 개인정보 제3자 제공 동의 [필수]
   - 제공받는 자, 이용목적, 항목, 보유기간, 거부 시 불이익
   □ 동의함  □ 동의하지 않음

4. 민감정보(건강정보, 유전정보) 처리 동의 [필수]
   □ 동의함  □ 동의하지 않음

5. 개인정보 국외 이전 동의 [다국가 시험의 경우 필수]
   - 이전 국가, 이전받는 자, 목적, 기간
   □ 동의함  □ 동의하지 않음

6. 잔여 생체시료 2차 활용 동의 [선택]
   □ 동의함  □ 동의하지 않음
```

## 6. 개인정보 보유 기간 및 파기 — 제21조

### 6.1 보유 기간 원칙

| 원칙 | 내용 |
|------|------|
| **목적 달성 후 파기** | 수집·이용 목적 달성 후 지체 없이 파기 |
| **KGCP 보존 기간 우선** | KGCP는 15년 보존 요건 → 법령상 의무로서 보존 가능 (PIPA 제21조 단서) |
| **명시** | ICF에 보유 기간 구체적으로 명시 필요 |

### 6.2 임상시험 개인정보 보유 기간 설정 기준

| 목적 | 보유 기간 | 근거 |
|------|---------|------|
| 임상시험 참여 관리 | 임상시험 종료 후 15년 | KGCP 별표 4 |
| 이상반응 추적 관리 | 임상시험 종료 후 15년 | KGCP |
| 규제기관 보고 | 제품 허가 후 최소 2년 (KGCP와 긴 기간 적용) | 약사법 |
| 잔여 생체시료 관련 정보 | 생명윤리법 제39조 보존 기간과 일치 | 생명윤리법 |

### 6.3 파기 방법

| 파기 대상 | 방법 |
|---------|------|
| **전자적 기록** | 복구 불가능한 방법으로 영구 삭제 (덮어쓰기 7회 이상 또는 물리적 파기) |
| **종이 기록** | 분쇄 또는 소각 |
| **생체시료** | 생명윤리법 제39조 보존·폐기 절차에 따름 |

## 7. 동의 거부 시 불이익 금지

| 원칙 | 내용 |
|------|------|
| **기본 원칙** | 개인정보 처리 동의를 거부한다는 이유로 불이익을 주어서는 안 됨 |
| **임상시험 예외** | 임상시험의 특성상 안전성 평가에 필요한 필수 개인정보 동의 거부 시 → 시험 참여 불가 (이는 "불이익"이 아닌 시험 실시 불가 사유로 정당) |
| **선택적 동의** | 2차 활용, 추가 생체시료 보관 등 선택적 항목은 거부해도 시험 참여에 영향 없음 |
| **ICF 명시** | "이 항목에 동의하지 않으면 임상시험에 참여하실 수 없습니다" 또는 "이 항목에 동의하지 않아도 임상시험 참여에 영향이 없습니다"로 명확히 구분 |

## 8. 가명처리 및 익명처리

### 8.1 가명처리 — 제28조의2~7

| 항목 | 내용 |
|------|------|
| **정의** | 가명정보: 추가 정보 없이는 특정 개인을 알아볼 수 없도록 처리한 정보 |
| **허용 목적** | 통계 작성, **과학적 연구**, 공익적 기록 보존 등 → 임상시험 데이터 분석 포함 |
| **동의 불필요** | 가명정보는 정보주체 동의 없이 처리 가능 (제28조의2 제1항) |
| **제3자 제공** | 가명정보 제3자 제공 시 특정 개인 식별 가능 정보 포함 금지 |
| **금지 행위** | 가명정보를 특정 개인 재식별 목적으로 사용 금지; 위반 시 전체 매출 3% 과징금 |
| **안전성 조치** | 가명정보 처리 시 재식별 방지를 위한 기술적·관리적 조치 필요 |
| **처리 기록** | 가명정보 처리 목적, 방법, 기간 등 기록 의무 |

### 8.2 익명처리 — 제28조의7

| 항목 | 내용 |
|------|------|
| **정의** | 익명정보: 시간·비용·기술 등을 합리적으로 고려할 때 다른 정보를 사용해도 더 이상 특정 개인을 알아볼 수 없는 정보 |
| **PIPA 적용 제외** | 익명처리된 정보는 PIPA 적용 대상 아님 |
| **임상시험 데이터** | 완전 익명화 후 학술 발표·논문 공개 가능 |

### 8.3 임상시험에서의 가명처리 실무

| 단계 | 가명처리 방법 |
|------|------------|
| CRF 입력 | 대상자 번호(pseudonym code) 사용; 성명 대신 이니셜+생년 조합 등 |
| 데이터베이스 | 개인식별정보 분리 저장; 연계 테이블 별도 접근 통제 |
| 분석 데이터셋 | 통계 분석용 데이터셋에는 성명·주민번호 포함 금지 |
| 논문·발표 | 개인 식별 불가 수준의 익명화 후 공개 |

## 9. 정보주체의 권리

| 권리 | 내용 | 임상시험에서의 적용 |
|------|------|-----------------|
| **열람권 (제35조)** | 자신의 개인정보 처리 여부 및 내용 열람 요구 | 대상자는 자신의 CRF·검사 결과 열람 가능 |
| **정정·삭제권 (제36조)** | 부정확한 정보 정정 요구 | 오류 정보 정정 가능; 단, 임상 데이터 무결성 원칙과 충돌 시 별도 협의 |
| **처리정지권 (제37조)** | 개인정보 처리 정지 요구 | 동의 철회와 연계; 이미 처리된 데이터는 소급 삭제 불가 원칙 |
| **동의 철회권** | 언제든 동의 철회 가능 | ICF에 명시; 철회 시 이후 데이터 수집 중단; 이미 수집된 데이터의 처리 범위는 ICF에 명시 |
| **자동화 결정 거부권 (제37조의2)** | 자동화된 결정에 대한 거부·설명 요구 | 임상시험에서는 일반적으로 비해당 |

## 10. 개인정보 보호책임자(CPO) 및 안전성 확보 조치

| 항목 | 요건 |
|------|------|
| **개인정보 보호책임자** | 일정 규모 이상의 개인정보처리자는 CPO 지정 의무 |
| **개인정보 처리방침** | 공개 의무; 임상시험 기관 및 의뢰자 모두 해당 |
| **기술적 안전 조치** | 암호화(개인식별정보, 민감정보), 접근 통제, 접속 기록 보관 |
| **관리적 안전 조치** | 내부 관리계획 수립; 교육·훈련; 접근 권한 최소화 |
| **물리적 안전 조치** | 출입 통제; 서버·저장매체 물리적 보호 |
