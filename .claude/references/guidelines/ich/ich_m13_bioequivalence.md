## 1. M13 시리즈 구성

```
ICH M13 시리즈
├── M13A — 즉시방출 고형 경구 제형의 생물학적 동등성 [최종, 2024.07]
│   ← 표준 BE 시험 설계, 분석, BCS biowaiver (M9 연계)
├── M13B — 추가 강도 Biowaiver [Step 2, 2025.02]
│   ← 생물학적 동등성 시험 없이 추가 강도 허가
└── M13C — 고변동/NTI 약물/복잡 BE 설계 [개념 단계]
    ← 고변동(HVD), 좁은 치료 지수(NTI), 복잡 설계
```

---

## 2. M13A 범위 및 목적

### 2.1 적용 범위

- **대상 제형**: 즉시방출(IR) 고형 경구 제형 (정제, 캡슐 등)
- **평가 기준**: 약동학적(PK) 평가변수 기반 생물학적 동등성
- **적용 약물**: 새로운 경구 의약품 또는 기존 경구 의약품의 제네릭

### 2.2 핵심 목적

ICH 회원국(FDA/미국, EMA/유럽, PMDA/일본, 한국 MFDS 등) 간 BE 평가 기준을 조화하여:
- 중복 시험 감소
- 국제적으로 상호 수용 가능한 데이터 생성
- 규제 자원 효율화

---

## 3. BE 시험 설계 요건

### 3.1 표준 설계: 단회 투여 교차 설계

| 항목 | 요건 |
|------|------|
| **설계 유형** | 2-기간 2-처치 교차 설계 (표준) |
| **대상** | 건강한 성인 자원자 (원칙) |
| **최소 표본 수** | 12명 이상 |
| **세척 기간** | 최소 **5개 반감기** 이상 |
| **주요 분석약물** | 모약물(parent drug) (원칙) |
| **투여 조건** | 라벨 지시에 따름 (공복 또는 식후) |

### 3.2 대체 설계

| 설계 | 적용 조건 |
|------|----------|
| **평행 설계** | 긴 반감기 약물 또는 데포(depot) 제형 |
| **반복 교차(Replicate crossover)** | 개체 내 CV >30% 고변동 약물 |
| **복수 투여** | 최고 강도에서 안전하지 않은 경우, 또는 PK 선형성 미입증 시 |

### 3.3 단회 투여 vs 복수 투여

- **단회 투여 시험 선호**: 흡수 차이 탐지에 더 민감
- **복수 투여 허용**: 안전성 문제로 최고 강도 단회 투여 불가 시, 또는 환자 집단 필요 시

---

## 4. 약동학 지표 (PK Parameters)

### 4.1 일차 PK 지표

| 지표 | 의미 | 산출 방법 |
|------|------|----------|
| **AUC₀₋ₜ** | 최종 정량 시점까지의 흡수 정도 | Linear-log trapezoidal |
| **AUC₀₋∞** | 무한대까지 외삽한 흡수 정도 | AUC₀₋ₜ + Clast/λz |
| **Cmax** | 최대 혈장 농도 (흡수 속도 및 최대 노출) | 직접 관찰값 |

**AUC 요건**: AUC₀₋ₜ가 AUC₀₋∞의 80% 이상을 포함해야 함

### 4.2 이차 PK 지표

| 지표 | 의미 |
|------|------|
| **Tmax** | 최대 농도 도달 시간 (빠른 발현이 중요한 경우) |
| **t½** | 소실 반감기 |
| **kel** | 소실 속도 상수 |
| **부분 AUC** | 빠른 발현이 임상적으로 중요한 경우 (FDA, 캐나다 요구) |

---

## 5. 수용 기준 (Acceptance Limits)

### 5.1 표준 수용 기준

| 지표 | 기준 |
|------|------|
| **AUC (90% CI)** | **80.00% ~ 125.00%** |
| **Cmax (90% CI)** | **80.00% ~ 125.00%** |
| **분석 방법** | 자연로그 변환 후 선형 혼합 효과 모델 (ANOVA) |
| **신뢰구간** | 90% CI (two one-sided t-tests, TOST 등가) |

### 5.2 좁은 치료 지수(NTI) 약물

> **주의**: NTI 약물의 BE 기준은 M13C에서 다룰 예정 (현재 M13A에서 명시 없음)
- 현재 FDA: 더 좁은 기준 (90% CI: 90.00%~111.11%)
- 현재 EMA: 좁은 기준 별도 적용
- M13C에서 조화 목표

---

## 6. 식사 조건 (Fed/Fasted Study Requirements)

### 6.1 M13A의 주요 변경 사항

> **중요 변경**: 모든 제품에 공복+식후 2개 시험을 요구하던 일반 요건 **폐지**
> 대부분의 저위험 제품에는 단일 공복 시험으로 충분

### 6.2 식사 조건 결정 원칙

| 시험 조건 | 적용 기준 |
|----------|----------|
| **공복 시험만** | 공복 투여 라벨, 저위험 제품 (대부분의 경우) |
| **식후 시험만** | 식후 투여 라벨 제품 |
| **공복+식후 모두** | 고위험 제품 (저용해성, 식품 효과 가능성 높은 제품) |

**고위험 제품 (공복+식후 모두 필요한 경우):**
- 복합 제형 (마이크로에멀전, 나노입자, 고체분산체)
- 비선형 PK 약물
- 임상적으로 관련된 식품 효과가 있는 약물
- BCS Class II/IV 약물 (일부)

### 6.3 표준 식사 조건

- **고지방-고열량 식사**: 전신 생체이용률의 최대 섭동(perturbation) 보장
- 구체적: 약 800~1000 kcal, 지방 50~55% (약 50g)

---

## 7. BCS 기반 Biowaiver

### 7.1 BCS (생약제학적 분류 시스템)

| Class | 용해성 | 투과성 | Biowaiver 가능성 |
|-------|-------|-------|-----------------|
| I | 높음 | 높음 | **가능** (M9 기준 충족 시) |
| II | 낮음 | 높음 | 제한적 (일부 경우) |
| III | 높음 | 낮음 | **가능** (M9 기준 충족 시, 더 엄격한 조건) |
| IV | 낮음 | 낮음 | **불가** |

### 7.2 M13A와 M9의 관계

- M13A는 BCS biowaiver를 **ICH M9**에 위임
- M9 (BCS 기반 Biowaiver, 2020년 Step 4): BCS Class I, III 약물의 biowaiver 기준 상세 규정

**ICH M9 Biowaiver 주요 기준 (참고):**
- BCS Class I: 높은 용해성 + 높은 투과성 + 빠른 용해 (30분 이내 85% 이상)
- BCS Class III: 높은 용해성 + 유사한 부형제 조성

## 8. 기존 지역별 가이드라인과의 관계

### 8.1 M13A 조화 목표 달성 현황

| 항목 | 조화 결과 |
|------|----------|
| 단회 투여 교차 설계 | 합의 완료 |
| 건강인 최소 12명 | 합의 완료 |
| 모약물 측정 | 합의 완료 |
| 공복+식후 일반 요건 | FDA(미국)와 차이 — FDA는 여전히 양쪽 요구 가능 |
| 고변동 약물 기준 | 미해결 — M13C에서 다룰 예정 |
| 부분 AUC | 미해결 — FDA/캐나다만 요구 |

### 8.2 지역별 현황 비교

| 항목 | M13A (ICH 조화안) | FDA (미국) | EMA (유럽) | MFDS (한국) |
|------|-----------------|-----------|-----------|------------|
| 표준 수용 기준 | 80~125% | 80~125% | 80~125% | 80~125% |
| 식후 시험 | 위험 기반 | 라벨 무관 적극 요구 | 라벨 기반 | 라벨 기반 |
| NTI 기준 | M13C 예정 | 90~111.11% | 좁은 기준 적용 | 별도 기준 |
| 현지 인구 시험 | 규정 없음 | 해당 없음 | 해당 없음 | **현지 인구 우선** |

**MFDS(한국) 특이사항:**
- 한국인 피험자를 대상으로 BE 시험 수행 원칙
- 한국인 대상 시험 데이터가 있는 경우 ICH 데이터 수용

---

## 9. M13B: 추가 강도 Biowaiver (Step 2, 2025년 2월)

### 9.1 개념

최고 강도에서 BE가 입증된 후, 동일 제형의 **낮은 강도**는 in vivo BE 시험 없이 허가 가능한 조건 정의

### 9.2 주요 적용 조건 (예상)

- 동일 비율(proportional) 제형
- 동일 제조 공정
- 유사한 용해 프로파일 (in vitro)

---

## 10. M13C: 예정 가이드라인 (개념 단계)

다음 사항을 다룰 예정:
- **고변동 약물(HVD)**: 개체 내 CV >30%, 확장 기준 (scaled average BE)
- **좁은 치료 지수(NTI) 약물**: 더 엄격한 기준 (예: 90~111.11%)
- **복잡한 BE 설계**: 비선형, 시간 의존적 PK 등

---

## 11. 국내 적용 (MFDS 관련)

### 11.1 현황 (2026년 4월 기준)

- MFDS는 자체 BE 가이드라인 운용 중
- ICH M13A 조화 내용을 반영한 가이드라인 개정 논의 중
- 기본 수용 기준(80~125%)은 동일
- 현지 인구(한국인) BE 데이터 요구 유지

### 11.2 주요 차이점 (M13A vs MFDS)

| 항목 | M13A | MFDS |
|------|------|------|
| 시험 대상 인구 | 규정 없음 | 한국인 우선 |
| 식후 요건 | 위험 기반 | 개별 심사 |
| 고변동 약물 | M13C 예정 | 별도 지침 |

---

## 12. 실무 적용 포인트

### 12.1 임상약리 시험에서 BE 적용 시

| 상황 | M13A 권고 |
|------|----------|
| 표준 IR 정제/캡슐 | 2-기간 교차, 건강인 12명+, 단회 투여 |
| 공복 라벨 약물 | 공복 시험 (식후 시험 생략 가능) |
| 고지방식 라벨 약물 | 식후 시험 |
| 고변동 약물 | Replicate 교차 설계 고려, M13C 적용 예정 |
| BCS I/III 약물 | ICH M9 기준 biowaiver 검토 |

### 12.2 분석 시

- SAS, R 등 통계 소프트웨어로 ANOVA 기반 90% CI 산출
- 자연로그 변환 후 분석 (geometric mean ratio)
- AUC₀₋ₜ, AUC₀₋∞, Cmax 각각 90% CI 산출
