# Guideline on Bioanalytical Method Validation

## 메타 정보

| 항목 | 내용 |
|------|------|
| 발행 기관 | EMA CHMP (Committee for Medicinal Products for Human Use) |

| 문서 번호 | EMEA/CHMP/EWP/192217/2009 Rev. 1 Corr. 2 |
| 발행일 | 2011-08-01 (발행), 2012-02-01 (발효), 2015-06-03 (최종 수정) |

| ICH M10 | https://www.ema.europa.eu/en/documents/scientific-guideline/ich-guideline-m10-bioanalytical-method-validation-step-5_en.pdf |

## 적용 범위

- 생물학적 시료(혈장, 혈액, 소변, 조직 등)에서 약물 농도를 측정하는 생분석 방법의 밸리데이션
- 적용 대상 시험: 비임상 독성동태학(TK), 임상 약동학(PK), 생물학적동등성(BE), 약물 상호작용(DDI), 생체이용률(BA) 연구
- 크로마토그래피 분석법 및 리간드 결합 분석법(LBA) 포함
- 제외 범위: 약력학(PD) 바이오마커 정량화 방법 (별도 기준 적용)

---

## 1. 완전 밸리데이션 (Full Validation)

### 1.1 필수 밸리데이션 파라미터

| 파라미터 | 약어 | 정의 |
|----------|------|------|
| 선택성(Selectivity) | — | 기질 매트릭스의 내인성 성분 존재 하에서 분석물질을 선별적으로 측정하는 능력 |
| 특이성(Specificity) | — | 유사 구조물, 대사체, 다른 약물로부터 분석물질을 구별하는 능력 |
| 정확도(Accuracy) | RE% | 측정값의 공칭 농도 대비 편차 (평균 바이어스, Mean Bias) |
| 정밀도(Precision) | CV% | 반복 측정 간 변동 계수 |
| 매트릭스 효과(Matrix Effect) | ME | 매트릭스 성분에 의한 이온 억제/증대 영향 |
| 회수율(Recovery) | RE | 시료 처리 과정에서의 분석물질 회수 비율 |
| 검량 범위 / 검량 곡선 | — | 정량 하한(LLOQ)부터 정량 상한(ULOQ)까지의 농도-반응 관계 |
| 정량 하한(LLOQ) | LLOQ | 허용 기준 내에서 정량 가능한 최저 농도 |
| 안정성(Stability) | — | 다양한 조건에서 분석물질의 안정성 |

---

## 2. 검량 곡선 (Calibration Curve)

### 2.1 검량 기준

| 항목 | EMA 요건 |
|------|----------|
| 최소 농도 수준 | **6개 이상** (LLOQ + 5개 이상) |
| 농도 범위 | 예상 연구 시료 농도 범위를 포함 |
| 검량 표준 허용 기준 | 공칭 농도의 **±15%** 이내 역산 농도 (back-calculated) |
| LLOQ 허용 기준 | 공칭 농도의 **±20%** 이내 |
| 최소 통과 기준 | 최소 75% 이상 검량 표준 허용 기준 충족 |
| 모델 | 최적 반응 함수(linear, quadratic 등) 사용; 최소 가중법(weighting) 적용 가능 |

### 2.2 검량 표준 허용 기준 요약

- LLOQ 제외 모든 수준: 공칭 농도 **±15%** 이내
- LLOQ: 공칭 농도 **±20%** 이내
- 동일 분석 배치에서 최고·최저 검량 표준 포함 필수
- 허용 기준 미달 검량 표준 제거 후 최소 기준 충족 여부 재확인

---

## 3. 정확도 및 정밀도 (Accuracy & Precision)

### 3.1 허용 기준

| 구분 | 정확도(RE%) | 정밀도(CV%) |
|------|------------|------------|
| LLOQ | ±20% | ≤20% |
| 그 외 모든 QC 수준 | **±15%** | **≤15%** |

### 3.2 평가 방법

**배치 내(Within-run) 정확도·정밀도:**
- 최소 1개 분석 배치에서 각 농도 수준당 **최소 5개** QC 시료 분석
- QC 농도 수준: **최소 4개** (LLOQ, 저농도 QC, 중농도 QC, 고농도 QC)
- 권고 사항: LLOQ에 가까운 저농도 QC는 LLOQ의 3배 이하

**배치 간(Between-run) 정확도·정밀도:**
- 최소 3개 이상의 별도 분석 배치에서 평가
- 각 배치는 서로 다른 날에 수행 권고

### 3.3 QC 시료 농도 수준

| QC 수준 | 권고 농도 범위 |
|---------|----------------|
| LLOQ QC | LLOQ 농도 |
| 저농도 QC (LQC) | LLOQ ~ 3× LLOQ |
| 중농도 QC (MQC) | 범위 중간 부근 |
| 고농도 QC (HQC) | ULOQ의 75–90% |
| 희석 QC (DQC) | ULOQ 초과 (희석 후 분석) |

---

## 4. 선택성 및 특이성 (Selectivity & Specificity)

### 4.1 선택성 평가

- **최소 6개** 독립적 공백(blank) 매트릭스 로트(lot) 분석
  - 특수 집단 (신기능장애, 간기능장애 등) 포함 고려
- 각 공백 매트릭스에서의 내인성 간섭 신호 확인
- 허용 기준:
  - LLOQ에서의 분석물질 신호 대비 공백 매트릭스 간섭 신호 ≤20%
  - 내부 표준물질(IS) 신호 대비 공백 매트릭스 간섭 신호 ≤5%

### 4.2 특이성 평가

- 예상 대사체, 유사 구조물, 병용 약물에 의한 간섭 평가
- 키랄 약물: 다른 거울상이성질체의 간섭 평가

---

## 5. 매트릭스 효과 (Matrix Effect)

### 5.1 평가 요건

- LC-MS/MS 등 이온화 방법 사용 시 **필수** 평가
- 최소 6개 독립적 매트릭스 로트 사용
- 특수 집단 매트릭스 포함 (지질혈증, 용혈 시료 등)

### 5.2 평가 방법

- Post-column infusion 방법 또는
- 동위원소 표지 내부 표준(SIL-IS) 사용 시 정규화 접근법

### 5.3 허용 기준

- 내부 표준으로 정규화된 매트릭스 인자(MF) 변동 계수(CV) ≤15%

---

## 6. 회수율 (Recovery)

- 추출 과정에서의 분석물질 회수율 평가
- 절대 회수율보다 **일관성(consistency)**이 중요
- 모든 농도 수준에서 유사한 회수율 또는 정밀도 내 변동
- 내부 표준의 회수율도 함께 평가

---

## 7. 안정성 시험 (Stability Testing)

### 7.1 필수 안정성 시험 항목

| 안정성 유형 | 조건 | 요건 |
|-------------|------|------|
| 동결-해동(Freeze-Thaw) | 실제 연구 시료 동결-해동 횟수 이상 | 최소 3 사이클; 동결 사이클 간 ≥12시간 동결 |
| 벤치탑(Bench-top) | 실온 | 실제 시료 처리 시간 이상 (최소 4–24시간) |
| 오토샘플러(Processed sample) | 오토샘플러 온도 | 실제 시료 대기 시간 이상 |
| 장기 동결(Long-term) | 연구 시료 보관 온도(-70°C 또는 -20°C) | 연구 시료 보관 기간 이상 |
| 스톡 용액(Stock solution) | 보관 조건 | 검량 표준 및 QC 제조에 사용된 모든 기간 |
| 추출물(Extract) | 오토샘플러 또는 재냉동 | 실제 분석 절차 조건 |

### 7.2 안정성 허용 기준

- 공칭 농도 대비 ±15% 이내 (LLOQ는 ±20%)
- 저농도(LQC) 및 고농도(HQC) QC 수준에서 평가

---

## 8. 정량 하한 (LLOQ)

| 항목 | 기준 |
|------|------|
| 정확도(Accuracy) | 공칭 농도의 ±20% 이내 |
| 정밀도(Precision) | CV ≤20% |
| S/N 비 | ≥5 (권고) |
| 특이성 | 공백 매트릭스 간섭 ≤20% (분석물질 기준) |

---

## 9. 희석 완전성 (Dilution Integrity)

- ULOQ를 초과하는 농도의 시료가 희석 후에도 허용 기준 내 정확도·정밀도 유지 여부 확인
- 최소 5개 시료로 각 희석 배수 평가
- 허용 기준: ±15% 이내 정확도, ≤15% CV

---

## 10. 분석물질 안정성에 관한 연구 시료 취급

- 전혈에서 혈장으로의 분리 절차 표준화
- 항응고제 선택 및 시료 전처리 조건 밸리데이션
- 혈액 세포와 분리 전 안정성 평가

---

## 11. 발생 시료 재분석 (Incurred Sample Reanalysis, ISR)

### 11.1 ISR 요건

| 항목 | EMA 요건 |
|------|----------|
| 적용 대상 | 모든 임상 PK 시험: BA, BE, PK, DDI, 비교 연구 |
| 비임상 | 독성동태학(TK) 시험 |
| ISR 시료 수 | 처음 1,000개: **10%** 재분석; 1,000개 초과분: **5%** 추가 재분석 |
| 최소 시료 수 | 최소 10개 (시험이 소규모인 경우) |
| 농도 선택 | Cmax 주변 + tmax 전후 시점의 시료 우선 선택 |

### 11.2 ISR 허용 기준

**소분자 약물:**
- 재분석 농도가 원 농도와 평균의 ±20% 이내인 시료가 **최소 67%(2/3)** 이상

**거대 분자(macromolecule) / LBA 방법:**
- 재분석 농도가 원 농도와 평균의 ±30% 이내인 시료가 **최소 67%(2/3)** 이상

### 11.3 ISR 실패 시 조치

- ISR 실패 원인 조사 (시료 불안정, 방법 오류 등)
- 필요 시 추가 시료로 ISR 반복
- 연구 결과의 신뢰성에 영향 시 규제기관에 보고

---

## 12. 리간드 결합 분석법 (Ligand Binding Assay, LBA) 특이 요건

### 12.1 크로마토그래피법 vs. LBA 주요 차이

| 항목 | 크로마토그래피 | LBA |
|------|---------------|-----|
| 정확도·정밀도 기준 | ±15% (LLOQ ±20%) | ±20% (LLOQ ±25%) |
| 선택성 | 이온 신호 기반 | 결합 친화도 기반 |
| 매트릭스 효과 | LC-MS/MS에서 이온 억제 | 매트릭스 단백질에 의한 간섭 |
| 특이성 | 분자량 기반 구별 | 항체 특이성에 의존 |

### 12.2 LBA 특이 밸리데이션 항목

- **Hook effect** 평가 (고농도에서 신호 감소 현상)
- 약물 표적 복합체 존재 시 측정 방법 검토
- 면역원성 간섭 평가 (ADA가 분석에 미치는 영향)
- 동물 혈장 다종 평가

---

## 13. 부분 밸리데이션 (Partial Validation) 요건

**다음 변경 사항 발생 시 부분 밸리데이션 필요:**

| 변경 유형 | 필요 밸리데이션 범위 |
|-----------|---------------------|
| 검량 범위 변경 | 범위 확장 또는 축소에 해당하는 수준 재밸리데이션 |
| 분석 장비 변경 | 정확도·정밀도, 검량 곡선 확인 |
| 매트릭스 변경 | 선택성, 매트릭스 효과, 안정성 재평가 |
| 전처리 방법 변경 | 회수율, 매트릭스 효과 재평가 |
| 내부 표준 변경 | 완전 또는 부분 밸리데이션 (변경 범위에 따라) |
| 종(species) 변경 | 선택성, 매트릭스 효과 재평가 |

---

## 14. 교차 밸리데이션 (Cross-Validation)

- 동일 분석물질을 두 가지 이상의 방법으로 측정하는 경우
- 또는 다른 연구 기관 간 방법 비교
- QC 시료 또는 발생 시료를 두 방법으로 분석하여 결과 비교
- 허용 기준: 두 방법 간 평균 차이 ±15% 이내

---

## 15. EMA 가이드라인과 ICH M10의 주요 차이점

| 항목 | EMA 가이드라인 (2012) | ICH M10 (2022) |
|------|----------------------|----------------|
| 선택성/특이성 | 혼용 사용 | **선택성(Selectivity)과 특이성(Specificity) 분리** 정의 |
| 매트릭스 효과 | 6개 로트 평가 | 더 정밀한 지침; 개별 매트릭스 소스별 정확도·정밀도 평가 |
| ISR 기준 | 10% (처음 1,000개) + 5% (초과분) | 동일 |
| LBA 허용 기준 | ±20% (LLOQ ±25%) | 동일 유지 |
| LLOQ 허용 기준 | ±20% | 동일 |
| 적용 범위 | BA/BE/PK/DDI 등 | 더 포괄적 (비임상 포함 전 단계) |
| 안정성 | 개략적 기술 | 더 구체적 지침 |
| 보고 요건 | 일반적 | 더 상세한 보고 체계 |

---

## 16. EMA 가이드라인과 FDA 가이드라인 주요 차이점

| 항목 | EMA CHMP/EWP/192217/2009 | FDA 2018 BMV Guidance |
|------|--------------------------|----------------------|
| **실험 기술** | 더 상세한 실험 절차 기술 | — |
| **보고 요건** | 보다 간략 | 더 포괄적 보고 지침 |
| **안정성 기간** | 실제 연구 기간 이상 | 동일 원칙 |
| **ISR 빈도** | 10% (처음 1,000개) + 5% | 동일 |
| **LBA 기준** | ±20%/±25% | ±20%/±25% (유사) |
| **매트릭스 효과** | 6개 로트 (평가 필수) | 평가 필수 (로트 수 덜 명시적) |
| **접근 방식** | 원칙 기반; 유연성 있음 | 더 처방적 |

---

## 17. 현행 적용 기준 (2023년 이후)

- **ICH M10**: 2022년 5월 Step 5 최종화, EU 2023년 1월 1일 발효
- EMA의 EMEA/CHMP/EWP/192217/2009는 ICH M10으로 **공식 대체**됨
- 신규 제출: ICH M10 기준 적용 권고
- 기존 연구: EMA 또는 FDA 기준으로 수행된 연구는 해당 기준 계속 적용 가능
