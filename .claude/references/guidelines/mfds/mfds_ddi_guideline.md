## 1. 적용 범위 (Scope)

약물상호작용(DDI, Drug-Drug Interaction) 시험은 신약 개발 과정에서 시험약이 다른 약물의 약동학(PK) 또는 약력학(PD)에 미치는 영향, 또는 다른 약물이 시험약에 미치는 영향을 정량적으로 평가하는 임상시험이다.

| 항목 | 내용 |
|------|------|
| **대상 약물** | 신약 및 적응증 확대 의약품으로서 DDI 평가가 필요한 경우 |
| **평가 유형** | ① 대사효소(CYP) 기반 DDI, ② 수송체(transporter) 기반 DDI, ③ 약력학적(PD) DDI |
| **단계적 접근** | In vitro 평가 → In vivo 임상시험 (단계적 risk-based approach) |
| **목적** | 허가사항(용법용량, 사용상 주의사항)에 DDI 정보 반영 |

---

## 2. In Vitro 평가 (In Vitro Assessment)

### 2.1 평가 대상 효소 및 수송체

| 구분 | 평가 대상 |
|------|----------|
| **CYP 억제 (가역적)** | CYP1A2, 2B6, 2C8, 2C9, 2C19, 2D6, 3A4 (7종 필수) |
| **CYP 억제 (시간의존적, TDI)** | 위 7종에 대해 시간의존적 억제 평가 필수 |
| **CYP 유도** | CYP1A2, 2B6, 3A4 (3종 필수) |
| **P-당단백 (P-gp)** | 기질/억제제/유도제 여부 평가 |
| **BCRP** | 기질/억제제 여부 평가 |
| **OATP1B1, OATP1B3** | 간 수송체; 기질/억제제 여부 |
| **OAT1, OAT3** | 신장 유기음이온 수송체 |
| **OCT2** | 신장 유기양이온 수송체 |
| **MATE1, MATE2-K** | 신장 수송체 |

### 2.2 In Vivo 시험 결정 기준

| 기준 | 판단 |
|------|------|
| **가역적 억제**: [I]/Ki > 0.1 또는 R-value > 1.02 | → In vivo 시험 필요 |
| **시간의존적 억제(TDI)**: KI, kinact 기반 기준 초과 | → In vivo 시험 강력 권고 |
| **유도**: AhR/PXR/CAR 활성화 양성 | → In vivo 시험 고려 |
| **수송체**: [I]/IC₅₀ 기반 기준 초과 | → In vivo 시험 고려 |

## 3. 임상 DDI 시험 설계 (Clinical DDI Study Design)

### 3.1 일반 원칙

| 항목 | MFDS 요건 |
|------|----------|
| **시험 유형** | 전향적(prospective) 전용 DDI 시험 원칙 |
| **대상자** | 건강한 성인 자원자 (특수한 경우 환자 가능) |
| **기본 설계** | 교차설계(crossover) 권장 |
| **병렬설계** | 약물 특성상 교차설계 불가 시 허용 |
| **고정순서(one-sequence)** | 유도제/억제제의 긴 반감기 등으로 교차설계 불가 시 허용 |

### 3.2 설계별 권장 사항

| 설계 | 적용 상황 | 비고 |
|------|----------|------|
| **2×2 교차** | 가역적 억제, 짧은 반감기 약물 | 표준 설계; ANOVA 분석 |
| **고정순서(fixed sequence)** | 효소 유도, 비가역적 억제, 긴 반감기 | 가장 흔한 DDI 설계 |
| **Williams 4×4** | 다중 용량 수준 평가, 복수 상호작용제 비교 | 복잡한 다중처리 설계 |
| **병렬** | 교차 불가 시 (예: 긴 반감기 + 강한 유도) | 대상자 수 증가 필요 |

### 3.3 투여 조건

| 항목 | 요건 |
|------|------|
| **기질(substrate) 투여** | 단회투여 원칙; 다회투여 필요 시 정상상태(steady-state) 확인 |
| **억제제 투여** | 정상상태(steady-state) 도달 후 기질 투여 (통상 5~7일 전투여) |
| **유도제 투여** | 정상상태 도달 후 기질 투여 (최소 7~14일 전투여) |
| **용량** | 승인 용량 범위 내 최대 상호작용 예상 용량 사용 |
| **공복/식후** | 약물의 허가 용법(또는 worst-case 상황)에 따름 |

### 3.4 채혈 및 PK 설계

| 항목 | 요건 |
|------|------|
| **채혈 시점** | 기질 투여 전 + 기질 PK 완전 포착을 위한 충분한 시점 |
| **대사체 측정** | 주요 대사체가 있는 경우 병행 측정 권장 |
| **정상상태 확인** | 억제제/유도제의 trough 농도 측정으로 정상상태 확인 |

## 4. 평가변수 (Endpoints)

| 평가변수 | 역할 | 비고 |
|---------|------|------|
| **AUC₀₋∞** | **1차 평가변수** | 대사 억제/유도 정도 평가의 핵심 |
| **AUC₀₋ₜ** | 1차 평가변수 (보조) | AUC₀₋∞ 산출 불가 시 사용 |
| **Cmax** | 2차 평가변수 | 흡수 단계 상호작용 평가 |
| **t½** | 탐색적 | 반감기 변화 확인 |
| **CL/F** | 탐색적 | 청소율 변화로 기전 이해 |
| **대사체/모약물 비율** | 탐색적 | 대사 경로 변화 평가 |

## 5. 통계 분석 (Statistical Analysis)

### 5.1 교차설계 DDI

| 항목 | MFDS 요건 |
|------|----------|
| **분석 방법** | ANOVA (BE 분석과 동일 모형) |
| **변환** | 자연로그(ln) 변환 후 분석 |
| **보고** | 기하평균비(GMR) 및 90% 신뢰구간(CI) |
| **no-effect 기준** | 90% CI가 **80.00%~125.00%** 이내 → 임상적으로 유의한 DDI 없음 |

### 5.2 고정순서(One-sequence) DDI

| 항목 | 방법 |
|------|------|
| **주 분석** | Paired t-test (로그변환) 또는 ANOVA (기간 효과) |
| **보고** | GMR 및 90% CI |
| **추가 분석** | 개별 대상자 fold-change scatter plot |

## 6. DDI 결과 해석 지침 (Interpretation)

### 6.1 억제 분류 (AUC 기준)

| GMR 범위 | 해석 | 조치 |
|----------|------|------|
| 0.80 ≤ GMR ≤ 1.25 | 임상적으로 유의한 DDI 없음 | 용량 조절 불필요 |
| 1.25 < GMR ≤ 2.0 | 약한(weak) 억제 | 주의 관찰 또는 라벨링 |
| 2.0 < GMR ≤ 5.0 | 중등도(moderate) 억제 | 용량 조절 권고; 라벨 명시 |
| GMR > 5.0 | 강한(strong) 억제 | 병용 금기 또는 대폭 감량 |

### 6.2 유도 분류

| GMR 범위 | 해석 | 조치 |
|----------|------|------|
| 0.50 ≤ GMR < 0.80 | 약한(weak) 유도 | 주의 관찰 |
| 0.20 ≤ GMR < 0.50 | 중등도(moderate) 유도 | 용량 증량 고려 |
| GMR < 0.20 | 강한(strong) 유도 | 용량 조절 또는 병용 금기 |

## 7. 특수 상황 (Special Considerations)

| 상황 | MFDS 입장 |
|------|----------|
| **다중 경로 대사** | 주 대사 경로 각각에 대한 DDI 평가; 분수 기여도(fm) 고려 |
| **수송체 매개 DDI** | P-gp, OATP 등에 대한 in vivo 시험 설계 동일 원칙 적용 |
| **PBPK 모델링** | 보조 근거로 활용 가능; 단독 대체는 제한적 수용 |
| **복합 DDI** | 억제+유도 동시 발생 가능 시 개별 평가 또는 통합 PBPK |
| **라벨링 표기** | DDI 결과에 따라 허가사항(용법용량, 사용상 주의사항, 병용 금기)에 반영 |
| **환자 대상 DDI** | 건강 자원자에서의 DDI 평가가 불가하거나 비윤리적인 경우 허용 |

## 8. IND 신청 시 DDI 자료 요건

| 단계 | 요건 |
|------|------|
| **Phase 1 이전** | In vitro CYP 억제/유도 데이터; 수송체 기질 여부 평가 |
| **Phase 1~2 중** | 주요 대사 경로 확인 후 임상 DDI 시험 계획 |
| **허가 신청 시** | 임상 DDI 시험 결과 보고서; 라벨링 문구 제안 |
