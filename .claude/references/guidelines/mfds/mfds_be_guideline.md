## 1. 적용 범위 (Scope)

생물학적동등성(BE, Bioequivalence)시험은 제네릭 의약품이 오리지널 의약품(대조약, Reference Product)과 생체이용률이 동등함을 통계적으로 입증하는 시험이다. 의약품 품목허가·신고 시 동등성 자료로 활용되며, 이하 기준은 경구 고형제를 중심으로 한다.

---

## 2. 시험 설계 (Study Design)

| 항목 | MFDS 요건 |
|------|----------|
| **기본 설계** | 2×2 교차설계 (2-sequence, 2-period crossover) 원칙 |
| **반복 교차설계** | 고변동 약물(HVD)에 한해 반복 교차설계(replicate crossover) 허용 |
| **무작위화** | 순서 무작위 배정 필수; 각 군 대상자 수 균형 유지 |
| **눈가림** | 필수는 아니나 가능한 경우 권장 |
| **대조약 (Reference)** | 국내 시판 중인 오리지널 품목 원칙; 수입품은 원산국 공인 품목 사용 가능 |
| **추가시험** | 1회에 한해 허용; 통계적 일관성 입증 필요 |

---

## 3. 시험대상자 (Subjects)

| 항목 | MFDS 요건 |
|------|----------|
| **대상** | 건강한 성인 자원자 (원칙적으로 만 19세 이상) |
| **최소 대상자 수** | 평가 가능 대상자 최소 12명 이상 (각 군); 통계적 근거로 산출 |
| **성별** | 원칙적으로 남녀 모두 포함; 약물 특성(예: 피임약)에 따라 단일 성별 허용 |
| **BMI** | 통상 18.5~30.0 kg/m² (관행; 가이드라인 명시 아님) |
| **건강 상태** | 병력·이학적 검사·임상검사(혈액학, 생화학, 요검사)로 건강 확인 |
| **금연·금주** | 시험 전 일정 기간 흡연·음주 제한 (프로토콜에 명시) |
| **washout** | 이전 임상시험 참여 후 적절한 washout 기간 확보 |

## 4. 시험 조건 (Study Conditions)

| 항목 | MFDS 요건 |
|------|----------|
| **공복 시험 (기본)** | 최소 10시간 공복 후 투여; 투여 후 4시간까지 금식 |
| **식후 시험** | 고지방식이 조건 필요 시 별도 시험 시행 (FDA는 공복+식후 모두 권장, MFDS는 공복 우선) |
| **투여 방법** | 물 240 mL(표준 용량, standing water)와 함께 투여 |
| **휴약기 (Washout period)** | 반감기(t½)의 5배 이상, 또는 최소 1주일 중 긴 기간 적용 |
| **활동 제한** | 투여 당일 격렬한 운동·카페인 제한 |

## 5. 채혈 시점 및 PK 분석 (Sampling and PK Analysis)

| 항목 | MFDS 요건 |
|------|----------|
| **채혈 시점 수** | Cmax 전후로 충분한 시점 설계; 곡선하면적(AUC)의 80% 이상 포착 필요 |
| **채혈 기간** | 투여 후 최소 3 × t½ 이상 (AUC 외삽 비중 최소화) |
| **1차 평가변수** | AUC₀₋ₜ (최종 정량 가능 시점까지), Cmax 필수 |
| **2차 평가변수** | AUC₀₋∞ (참고용), Tmax, t½, CL/F, Vd/F |
| **다회투여 시 평가변수** | AUCτ (정상상태 투여간격), Css,max |
| **분석법** | 밸리데이션된 생체시료분석법 (LC-MS/MS 등) — 식약처 생체시료분석법 가이드라인 준수 |
| **약동학 분석** | 비구획분석(NCA, Non-Compartmental Analysis) 원칙 |
| **소프트웨어** | 검증된 PK 소프트웨어 사용 (Phoenix WinNonlin, R/PKNCA 등) |

## 6. 동등성 판정 기준 (BE Criteria)

### 6.1 일반 약물 판정 기준

| 항목 | MFDS 요건 |
|------|----------|
| **동등성 한계** | 자연로그(ln) 변환 후 **90% 신뢰구간(CI)이 log 0.8 ~ log 1.25 이내** |
| **실제 수치** | 역변환 후 **80.00% ~ 125.00%** |
| **1차 평가변수** | AUC₀₋ₜ 및 Cmax **동시** 충족 필요 |
| **통계 방법** | 혼합효과 분산분석(Mixed-effects ANOVA): 순서(sequence), 대상자 within sequence(변량), 기간(period), 제형(formulation) 효과 |
| **변환** | 자연로그(ln) 변환 후 분석; 결과는 기하평균비(GMR, Geometric Mean Ratio)로 보고 |
| **AUC₀₋∞** | 참고 자료로만 제시 (판정 기준 아님) |

### 6.2 엄격 판정 기준 (일부 서방제/복합제 등 특수 경우)

로그변환 후 평균치 차이가 log 0.9 ~ log 1.11 (= 90%~111%) 범위 이내이고, 비교용출시험이 동등하며, 총 24명 이상 시험한 경우에 적용될 수 있음 (고시 제17조 제2항).

## 7. 고변동 약물 (Highly Variable Drugs, HVD)

### 7.1 정의 및 적용 기준

| 항목 | MFDS 요건 |
|------|----------|
| **정의** | 개체 내 변동계수(intra-subject CV) > 30% |
| **적용 대상** | Cmax의 intra-subject CV가 30% 이상인 약물 |
| **증명 방법** | 반복 교차설계(replicate crossover) 시험을 통해 CV 입증 후 기준 확장 신청 |

### 7.2 확장 한계 (Scaled Average BE, MFDS 자체 방식)

Cmax의 intra-subject CV에 따른 확장 한계 (AUC는 확장 불가):

| Intra-subject CV (Cmax) | 확장 허용 한계 (90% CI) |
|------------------------|------------------------|
| 30% | 80.00% ~ 125.00% (기본) |
| 35% | 77.23% ~ 129.48% |
| 40% | 74.62% ~ 134.02% |
| 45% | 72.15% ~ 138.59% |
| ≥50% | **69.84% ~ 143.19%** (최대 확장 한계) |

> **주의**: AUC(AUC₀₋ₜ, AUCτ)는 확장 한계 적용 **불가** — 항상 80.00%~125.00% 내에서 판정

### 7.3 추가 요건

| 항목 | 요건 |
|------|------|
| **점추정치 조건** | 확장 한계 적용 시에도 Cmax 점추정치(GMR)가 80.00%~125.00% 이내 필수 |
| **설계** | 반복 교차설계(2×3 또는 2×4) 시험 필수 |
| **FDA 비교** | FDA는 RSABE(Reference-Scaled ABE) 방식 사용; MFDS는 확장 한계 방식(고정 확장 테이블) 적용 |

## 8. 특수 상황 (Special Cases)

| 상황 | MFDS 입장 |
|------|----------|
| **서방제(ER/MR)** | 공복 + 식후 시험 모두 필요; 경우에 따라 다회투여(steady-state) 시험 요구 |
| **복합제** | 각 활성성분에 대해 개별적으로 BE 입증 필요 |
| **국소제(topical)** | PK 기반 BE 외 별도 가이드라인 적용 가능 |
| **좁은 치료역 약물(NTI)** | 더 엄격한 동등성 기준 적용 가능 (90.00%~111.11%); 별도 검토 필요 |
| **추가 시험** | 1차 시험 결과 불충분 시 1회에 한해 추가 시험 허용 |

## 9. 통계 분석 절차 (Statistical Analysis Procedure)

### 분산분석 모형 (2×2 교차설계)

```
ANOVA 요인:
  - 순서 (Sequence): 고정 효과, RT vs TR
  - 대상자 (Subject nested in Sequence): 변량 효과
  - 기간 (Period): 고정 효과
  - 제형 (Formulation): 고정 효과 (T vs R)
```

### 분석 절차

1. PK 파라미터(AUC₀₋ₜ, Cmax)를 자연로그(ln) 변환
2. 혼합효과 ANOVA(SAS PROC MIXED 또는 동등 프로그램) 실행
3. 제형 효과의 최소제곱평균(LSM) 차이 및 90% CI 산출
4. 역변환(exponentiation)으로 GMR 및 90% CI(%) 보고
5. 90% CI가 80.00%~125.00% 이내이면 동등 판정

## 12. IND 승인 후 생동성 시험 흐름

```
IND 신청 (식약처)
  → 승인 (30일 내 심사; 또는 단계적 심사)
  → IRB 심의 및 승인 (임상시험 실시기관)
  → 시험 실시 (Phase 1 시험기관에서 입원 관찰)
  → 생체시료 분석 (GLP 인증 분석기관)
  → 통계 분석 (SAP 기반)
  → 생동성시험 결과 보고서 작성
  → 허가 신청 (NDA/ANDA에 첨부)
```
