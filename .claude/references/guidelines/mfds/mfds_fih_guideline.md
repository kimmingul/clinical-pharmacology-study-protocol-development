## 1. 적용 범위 (Scope)

초회 인체적용(FIH, First-in-Human) 임상시험은 비임상 시험을 완료한 후 최초로 사람에게 신규 물질을 투여하는 시험이다. 안전성·내약성 및 기초 약동학(PK) 특성 확립이 주목적이다.

| 항목 | 내용 |
|------|------|
| **대상** | 사람에게 최초로 투여되는 신규 물질 (소분자 합성의약품, 바이오의약품 포함) |
| **시험 유형** | SAD(Single Ascending Dose), MAD(Multiple Ascending Dose), 또는 SAD+MAD 통합 설계 |
| **대상자** | 건강한 성인 자원자 원칙; 세포독성 약물·중증 희귀질환 치료제는 환자 대상 허용 |
| **시험 기관** | 식약처 지정 임상시험 실시기관 (입원 관찰 가능한 1상 임상시험센터) |

## 2. 비임상 자료 요건 (Preclinical Data Requirements)

FIH 시험 IND 신청 시 필요한 비임상 자료:

| 자료 종류 | 내용 |
|----------|------|
| **단회투여 독성** | 설치류+비설치류 2종 이상; GLP 준수; NOAEL 확인 |
| **반복투여 독성** | 최소 2주 반복투여 독성시험 (FIH 기간에 상응하는 기간 이상) |
| **안전성 약리** | 심혈관계(hERG, in vivo QT), 중추신경계, 호흡기계 평가 (ICH S7A/S7B) |
| **효력시험** | In vitro/in vivo 약력학(pharmacodynamics) 데이터 |
| **PK 데이터** | 동물 PK; ADME 예비 데이터 |
| **유전독성** | Ames test + in vitro 염색체이상시험 (또는 in vivo 마이크로뉴클레어스) |
| **CMC** | 임상시험용 의약품 제조·품질 자료 |
| **초기 용량 산출 보고서** | NOAEL/MABEL 기반 MRSD 산출 근거 상세 기술 |

## 3. 초기 용량 산출 (Starting Dose Calculation)

### 3.1 소분자 의약품 — NOAEL-HED 방법

#### 기본 절차

```
NOAEL (동물에서 독성이 없는 최대 용량, mg/kg)
  ↓ × Km 환산계수 (체표면적 기반 종간 환산)
HED (Human Equivalent Dose, mg/kg)
  ↓ × 안전계수 (통상 1/10)
MRSD (Maximum Recommended Starting Dose, mg/kg)
  ↓ × 평균 체중 (예: 60 kg)
초기 투여 용량 (mg/person)
```

#### 종별 Km 환산계수

| 동물 종 | Km 값 | HED 산출 |
|--------|-------|----------|
| 마우스 | 3 | NOAEL(mg/kg) × (3/37) |
| 랫드 | 6 | NOAEL(mg/kg) × (6/37) |
| 기니피그 | 8 | NOAEL(mg/kg) × (8/37) |
| 토끼 | 12 | NOAEL(mg/kg) × (12/37) |
| 개 | 20 | NOAEL(mg/kg) × (20/37) |
| 원숭이(영장류) | 12 | NOAEL(mg/kg) × (12/37) |
| 인간 | 37 | (기준) |

> Km = 체표면적(m²)당 체중(kg) / 체표면적(m²)  
> HED = Animal NOAEL × (Animal Km / Human Km)

#### 안전계수 선택 기준

| 조건 | 권장 안전계수 |
|------|-------------|
| 표준 독성 프로파일, 가역성 독성 | 1/10 (기본) |
| 비가역적 독성, 표적 장기 독성 | 1/10 이상 (추가 안전계수) |
| 좁은 치료역(narrow TI) | 추가 상향 |
| 비선형 PK, 용량 비례성 없음 | 보수적 접근 |
| 비임상과 임상의 종간 차이 크다는 근거 | 추가 상향 |

### 3.2 바이오의약품 — MABEL 방법

| 항목 | MFDS 요건 |
|------|----------|
| **기본 방법** | MABEL(Minimum Anticipated Biological Effect Level) 권고 |
| **MABEL 산출** | In vitro EC₅₀(수용체 점유율), in vivo 최소 효과 농도, 종간 비교 약리학 데이터 통합 |
| **NOAEL 병용** | MABEL과 NOAEL-HED 중 **더 낮은 용량** 선택 |
| **적용 대상** | 면역조절제, 면역치료제, 단클론항체, 세포·유전자치료제 등 |
| **특별 주의 물질** | 면역조절제·mAb: 추가 안전계수 적용; cytokine 계열은 cytokine storm 위험 고려 |

### 3.3 추가 고려 사항

| 항목 | 내용 |
|------|------|
| **PAD (약리학적 활성용량)** | In vivo 약력학 최소유효용량을 HED로 환산; MRSD와 비교하여 낮은 용량 선택 |
| **비교 검토** | NOAEL-HED, MABEL, PAD 세 가지 값 모두 산출하여 비교 |
| **최종 선택** | 세 가지 중 가장 낮은 값에 안전계수 적용한 용량을 MRSD로 확정 |

## 4. 용량 증량 설계 (Dose Escalation Design)

### 4.1 SAD (Single Ascending Dose)

| 항목 | 요건 |
|------|------|
| **코호트 구성** | 통상 코호트당 6~8명 활성약 + 1~2명 위약 (이중눈가림) |
| **증량 방법** | Modified Fibonacci (100%→67%→50%→33%→33%...) 또는 고정 배율(2배, 1.5배) |
| **센티넬 투여** | 각 코호트 최초 1~2명 선행 투여 → 최소 24시간 안전성 관찰 후 나머지 투여 |
| **코호트 간 검토** | SRC/DSMB에서 이전 코호트 완료 후 다음 코호트 증량 결정 |
| **최대 용량** | 프로토콜에 사전 정의된 최대 용량 (MTD 또는 노출 상한) |

### 4.2 MAD (Multiple Ascending Dose)

| 항목 | 요건 |
|------|------|
| **전제 조건** | SAD 완료 후 안전성 확인 (일반적으로 SAD와 MAD를 순차 진행) |
| **투여 기간** | 프로토콜 정의 (통상 7~14일; 약물 특성에 따라 조정) |
| **정상상태 확인** | Trough 농도 측정으로 정상상태(steady-state) 도달 확인 |
| **채혈 설계** | 1일차(단회) + 정상상태(다회) 각각 완전한 PK 프로파일 |
| **코호트 구성** | SAD와 동일 원칙 |

### 4.3 증량 방법 비교

| 방법 | 설명 | 적용 |
|------|------|------|
| **Modified Fibonacci** | 증량 비율 점감: 2×, 1.67×, 1.5×, 1.33×, 1.33×... | 전통적 Phase 1 |
| **3+3 설계** | 코호트당 3명, DLT 발생 시 3명 추가 (항암제 특화) | 항암 Phase 1 |
| **고정 배율** | 일정 배율(예: 2배, 1.5배)로 증량 | 초기 저용량 범위 |
| **PK-guided** | 이전 코호트 PK 기반 용량 결정 (적응적 설계) | Bayesian adaptive |
| **BOIN/CRM** | 통계 모형 기반 MTD 추정 | 현대 항암 Phase 1 |

### 4.4 증량 중단 기준 (Stopping Rules)

| 수준 | 기준 |
|------|------|
| **개별 대상자** | SAE 발생; CTCAE Grade 3 이상 이상반응; 사전 정의된 개별 중지 기준 위반 |
| **코호트 수준** | DLT 발생 비율 초과 (예: 3명 중 2명 이상 DLT); SRC 결정 |
| **시험 전체** | 예상치 못한 중대한 안전성 신호; MTD 도달; DSMB 권고 |

## 5. 안전성 관리 (Safety Management)

### 5.1 센티넬 투여 (Sentinel Dosing)

| 항목 | MFDS 요건 |
|------|----------|
| **적용** | 각 용량 코호트의 처음 1~2명에게 선행 투여 |
| **관찰 기간** | 최소 24시간 (약물 반감기·작용 기전에 따라 48~72시간으로 연장) |
| **판단 주체** | 시험책임자 또는 SRC |
| **판단 기준** | 안전성 이상 없으면 나머지 대상자 투여 진행 |
| **기록** | 결정 근거 문서화 필수 |

### 5.2 안전성 검토 위원회 (SRC/DSMB)

| 항목 | 요건 |
|------|------|
| **구성** | 독립적인 임상약리학자, 내과 전문의, 통계학자 등; 시험과 무관한 외부 전문가 포함 |
| **검토 시점** | 각 용량 코호트 완료 후, 다음 코호트 증량 전 필수 |
| **검토 항목** | 이상반응(AE/SAE), 활력징후, 심전도(QTc), 임상검사, PK 데이터(가용 시) |
| **문서화** | SRC/DSMB 회의록; 증량·중단 결정 근거 기록 |
| **MFDS 보고** | 중대한 안전성 결정(예: 시험 중단) 즉시 식약처 보고 |

### 5.3 안전성 평가 항목

| 항목 | 시점 |
|------|------|
| **활력징후** | 투여 전 + 투여 후 1h, 2h, 4h, 8h, 24h (최소) |
| **12-lead ECG** | 투여 전 + Tmax 전후 (QTc 연장 위험 시 집중 모니터링) |
| **혈액학 검사** | 투여 전, 투여 후 24h, 72h, 퇴원 전 (최소) |
| **생화학 검사** | 간기능(ALT, AST, ALP, T-Bil), 신기능(SCr, BUN), 전해질 |
| **요검사** | 투여 전, 퇴원 전 |
| **이상반응 평가** | 지속 모니터링; MedDRA 코딩, CTCAE 등급, 인과관계 평가 |

## 6. 약동학 평가 (PK Assessment)

### 6.1 SAD PK

| 파라미터 | 산출 방법 |
|---------|---------|
| Cmax | 실측값 |
| Tmax | 실측값 |
| AUC₀₋ₜ | 선형-로그 사다리꼴법 |
| AUC₀₋∞ | AUC₀₋ₜ + Clast/λz |
| t½ (λz) | 말기 소실 반감기 |
| CL/F | Dose / AUC₀₋∞ |
| Vd/F | CL/F / λz |

### 6.2 MAD PK (정상상태)

| 파라미터 | 산출 방법 |
|---------|---------|
| AUCτ | 정상상태 투여간격 AUC |
| Css,max (Cmax,ss) | 정상상태 최고 혈중농도 |
| Css,min (Ctrough,ss) | 정상상태 최저 혈중농도 |
| 누적비 (Accumulation Ratio) | AUCτ,ss / AUCτ,1day |

### 6.3 용량 비례성 (Dose Proportionality)

| 방법 | 수식 |
|------|------|
| **Power model** | AUC = a × Dose^β; β의 90% CI가 0.7~1.43이면 비례성 있다고 판단 |
| **GMR 방법** | 용량 배수 증가 시 AUC 비율이 동일 배수 내 있는지 확인 |

## 7. 프로토콜 필수 포함 사항

| 항목 | 내용 |
|------|------|
| **초기 용량 근거** | NOAEL, HED, MABEL 산출 근거; 안전계수 선택 근거 상세 기술 |

## 8. IND 신청 절차 (IND Application)

```
비임상 패키지 완료 (GLP 독성, 안전성약리, CMC)
  ↓
MRSD/MABEL 산출 보고서 작성
  ↓
임상시험계획서(Protocol) 작성 (위 7항 필수 요소 포함)
  ↓
임상시험자 자료집(IB) 작성
  ↓
IND 신청서 제출 (식약처) — 규정 서류 첨부
  ↓
식약처 심사 (30일 이내; 보완 요청 시 연장)
  ↓
IND 승인
  ↓
IRB 심의·승인 (임상시험 실시기관 IRB)
  ↓
시험 개시 (Site Initiation Visit → 대상자 모집 → 투여)
```
