# ICF Drafting — 동의설명서/동의서 작성

임상시험 계획서를 기반으로 시험대상자용 동의설명서(ICF), 동의서, 개인정보 동의서를 작성한다.

> 이 스킬은 **별도의 `/icf` 명령**으로만 실행된다. Protocol 작성 후 자동 실행되지 않는다.

## 전제 조건

- `_workspace/03_protocol_draft.md`가 존재해야 한다 (Protocol 미완성 시 실행 불가)

## 작성 전 확인

1. **계획서** (`_workspace/03_protocol_draft.md`)를 Read로 읽어 시험 내용을 파악한다
2. **자료 보고서** (`_workspace/01_research_report.md`)를 Read로 읽어 안전성 정보를 보충한다
3. 계획서에서 다음 항목을 반드시 추출한다:
   - 시험 절차 (방문별), 채혈 횟수 및 총 채혈량
   - 입원 기간, 선정/제외 기준
   - 알려진 위험/부작용, 시험약 정보
   - 금지 사항(약물, 식품, 활동)

## 동의설명서 필수 요소

헬싱키 선언, ICH E6(R3) Annex 1, MFDS 가이드라인이 요구하는 필수 항목. 상세 템플릿은 `.claude/skills/icf-drafting/references/icf-template.md`를 Read로 참조한다.

| 필수 요소 | 내용 |
|----------|------|
| 시험의 목적 | 왜 이 시험을 하는지 |
| 시험 절차 | 무엇을 하게 되는지 (방문별 상세) |
| 시험 기간 | 얼마나 걸리는지 |
| 예상 위험/불편 | 어떤 위험이 있는지 |
| 예상 이익 | 어떤 이익이 있는지 (과장 금지) |
| 대안적 치료 | 시험 외 다른 선택지 |
| 비밀보장 | 개인정보 보호 방법 |
| 손상 시 보상 | 시험 관련 손상 발생 시 보상 |
| 비용 및 보상 | 시험대상자 부담 비용, 참여 보상 |
| 자발적 참여 | 언제든 철회 가능함 |
| 연락처 | 시험책임자, IRB/IEC 연락처 |
| 새로운 정보 제공 | 시험 중 새 정보 발생 시 알림 |

## 쉬운 언어 변환 원칙

| 전문 용어 | 쉬운 표현 |
|----------|----------|
| 무작위 배정 | 무작위(동전 던지기와 같은 방법)로 배정 |
| 이중 맹검 | 시험대상자와 시험자 모두 어떤 약을 받는지 모르는 방법 |
| 위약 | 시험약과 겉모습이 같지만 유효 성분이 없는 약 |
| 약동학 | 약이 몸에서 어떻게 흡수되고 배출되는지 |
| 코호트 | 같은 용량의 약을 받는 그룹 |
| 세척 기간 | 이전 약의 영향이 사라지도록 기다리는 기간 |
| 이상반응 | 원하지 않는 증상이나 변화 |

## 임상약리 시험 특유 설명 사항

1. **빈번한 채혈**: 총 횟수와 총량 명시, 일상 비유로 양감 전달
2. **입원 생활**: CRU 입실~퇴실 동안의 제한 (식이, 활동, 외출)
3. **용량 증량**: 이전 코호트 결과에 따라 용량이 결정됨
4. **센티넬 도징**: 2명에게 먼저 투약 후 안전성 확인 (해당 시)
5. **식이 제한**: 금식, 표준 식이, 수분 제한
6. **금지 사항**: 자몽, 카페인, 알코올, 흡연, 격렬한 운동
7. **방사성 물질 투여** (ADME 시험): 방사선 노출량, 제한 사항 설명

## 취약한 시험대상자 고려

간/신기능 장애 환자(Special Population) 대상 시험:
- 질환 상태에 따른 추가 위험을 별도로 설명
- 법정대리인 동의가 필요한 경우 서명란 포함
- 취약한 대상자 보호 조치 명시 (입회인 참여 등)

## 개인정보 보호법(PIPA) 동의

동의설명서와 별도로 개인정보 수집·이용·제3자 제공 동의서를 작성한다:
- 수집 항목: 성명, 생년월일, 연락처, 건강 정보, 검사 결과 등
- 수집 목적: 임상시험 수행 및 안전성 보고
- 제3자 제공: 의뢰자, 규제기관(MFDS, IRB), CRO
- 보유 기간: 시험 종료 후 법정 보관 기간
- 동의 거부 시 불이익 없음을 명시 (단, 시험 참여 불가 가능)

## 생명윤리법 요건 (해당 시)

인체유래물 연구(유전체 분석, 잔여 검체 보관) 해당 시:
- 인체유래물의 종류 및 수집 목적
- 보관 기간 및 보관 장소
- 폐기 절차
- 2차 활용(다른 연구 목적) 동의: 별도 서명란
- 동의 철회 시 검체 폐기 절차

## 전자동의(eConsent) (선택)

사용자가 전자동의를 요청한 경우에만 추가 작성:
- MFDS 전자동의 가이드라인 준수
- 전자 서명 방법, 본인 확인 절차
- 데이터 보안 및 저장 방법
- 종이 동의서와 동등한 법적 효력 명시

## Gotchas

- **계획서-동의설명서 불일치**: 가장 빈번한 문제. 채혈 횟수, 방문 횟수, 금지 사항이 계획서와 반드시 일치해야 한다
- **위험 정보 누락**: 계획서 안전성 섹션의 이상반응이 동의설명서에 빠지면 Critical 지적
- **과장된 이익**: Phase 1 임상약리 시험에서 시험대상자에게 직접적 치료 이익은 거의 없다. 솔직하게 기술
- **전문 용어 잔류**: 쉬운 언어 변환이 불완전하면 시험대상자가 이해하지 못한다
- **별도 /icf 명령 필수**: Protocol 작성 후 자동으로 ICF가 시작되지 않는다
