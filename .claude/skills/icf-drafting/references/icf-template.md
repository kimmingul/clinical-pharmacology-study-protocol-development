## 2. 본문 섹션별 작성 지침

### 2.1 시험에 대한 소개

**예시:**
```
귀하를 이 임상시험에 참여하도록 초대합니다. 참여 여부를 결정하시기 전에,
이 설명서를 주의 깊게 읽어주시기 바랍니다. 궁금한 사항은 언제든지
시험 담당자에게 질문하실 수 있으며, 충분히 이해하신 후에 참여 여부를
결정하시기 바랍니다. 참여는 완전히 자발적이며, 참여하지 않기로
결정하셔도 불이익은 전혀 없습니다.
```

### 2.2 시험의 목적
- 어떤 약(시험약)을 시험하는지
- 왜 필요한지 (쉬운 언어)
- Phase 1의 의미: "사람에게 처음 또는 초기에 투여하여 안전성을 확인하는 단계"
- 참여 인원, 시험기관

### 2.3 시험 참여 조건
선정/제외 기준을 일반인 언어로 변환. 예:
- "건강한 만 19세~55세 성인"
- "체질량지수(BMI) 18.5~30.0 범위"
- "현재 복용 중인 약이 없는 분"

취약한 대상자(간/신기능 장애 환자) 대상 시험의 경우:
- 질환 상태에 대한 쉬운 설명
- 추가 위험 요소 명시

### 2.4 시험 절차 및 방문 일정

**방문별 표:**
| 방문 | 시기 | 장소 | 주요 내용 | 소요 시간 |
|------|------|------|----------|----------|

**채혈 설명 (임상약리 특화):**
```
약이 몸에서 어떻게 흡수되고 배출되는지 확인하기 위해 혈액 검사가 필요합니다.
투약 전부터 투약 후 약 N시간까지 총 약 N회의 채혈이 예정되어 있습니다.
1회 채혈량은 약 NmL이며, 전체 시험 기간 동안 총 채혈량은 약 NmL입니다.
이는 건강 검진 시 일반적으로 채혈하는 양과 비슷한 수준입니다.
```

### 2.5 시험약에 대한 설명
- 시험약 이름, 형태, 투여 방법
- 위약 설명 (해당 시)
- ADME 시험: 방사성 물질([14C]-표지) 사용 여부, 방사선 노출량, 안전성

### 2.6 예상되는 위험과 불편
**반드시 포함:**
- 채혈로 인한 통증, 멍, 어지러움
- 입원 생활의 불편
- 시험약의 알려진/예상되는 부작용 (IB 기반)
- 동일 계열 약물의 알려진 부작용
- "아직 알려지지 않은 부작용이 발생할 가능성"
- 임신에 대한 위험 (해당 시)

### 2.7 예상되는 이익
- Phase 1: 직접적 치료 이익 기대하기 어려움을 명시
- 건강 검진을 통한 간접적 이익
- 의약품 개발에 대한 기여

### 2.8 다른 치료 방법
### 2.9 비밀보장
- 코드화, 잠금 보관
- 자료 열람 가능자 (의뢰자, IRB, 규제기관)

### 2.10 시험 관련 손상에 대한 보상
- 건강 손상 시 치료 제공
- 배상 책임 보험

### 2.11 시험 참여에 따른 비용과 보상
- 시험대상자 부담 비용: 없음
- 참여 보상: "[금액]원" 또는 "별도 안내"

### 2.12 자발적 참여와 중도 철회
```
이 시험의 참여는 완전히 자발적입니다. 참여에 동의하신 후에도
이유를 밝히지 않고 언제든지 참여를 철회하실 수 있습니다.
참여를 철회하시더라도 귀하에게 어떠한 불이익도 없습니다.
```

## 4. 개인정보 수집·이용·제3자 제공 동의서

> 개인정보 보호법(PIPA)에 따라 별도 동의가 필요하다.

```
개인정보 수집·이용·제3자 제공 동의서

1. 수집하는 개인정보 항목
   - 일반 정보: 성명, 생년월일, 성별, 연락처, 주소
   - 건강 정보: 병력, 현재 복용 약물, 신체 계측(키, 체중, BMI)
   - 시험 관련 정보: 실험실 검사 결과, 심전도, 활력징후, 이상반응

2. 수집·이용 목적
   - 임상시험 수행 및 결과 분석
   - 시험대상자 안전성 모니터링
   - 규제기관 보고 및 감사 대응

3. 제3자 제공
   | 제공받는 자 | 제공 목적 | 제공 항목 |
   |-----------|----------|----------|
   | 의뢰자 ({의뢰자명}) | 시험 데이터 분석, 안전성 보고 | 코드화된 시험 데이터 |
   | 식품의약품안전처(MFDS) | 규제 심사, 실태조사 | 시험 관련 자료 전체 |
   | IRB ({기관명}) | 시험대상자 보호, 윤리 감독 | 안전성 관련 자료 |
   | 수탁기관(CRO) ({해당 시}) | 시험 운영 지원 | 코드화된 시험 데이터 |

4. 보유 및 이용 기간
   - 시험 종료 후 관련 법령에 따른 보관 기간 (최소 3년)

5. 동의 거부 권리
   - 귀하는 개인정보 수집·이용·제공에 동의하지 않을 권리가 있습니다.
   - 다만, 동의하지 않으시는 경우 임상시험에 참여하실 수 없습니다.

□ 위 내용을 충분히 이해하였으며, 개인정보 수집·이용에 동의합니다.
□ 위 내용을 충분히 이해하였으며, 개인정보 제3자 제공에 동의합니다.

서명: _______________  날짜: ____년 __월 __일
```

## 5. 선택 동의 사항

### 5.1 유전체 분석 동의 (생명윤리법)
```
이 시험에서는 선택적으로 유전체 분석(DNA 검사)을 수행할 수 있습니다.

- 수집 목적: 약물 반응과 유전적 특성의 관계 연구
- 인체유래물 종류: 혈액에서 추출한 DNA
- 보관 기간: [N]년 (보관 기간 만료 시 폐기)
- 보관 장소: [기관명/장소]
- 폐기 절차: 보관 기간 만료 또는 동의 철회 시 [방법]으로 폐기
- 2차 활용: □ 향후 다른 연구 목적으로의 활용에 동의합니다 / □ 동의하지 않습니다
- 동의 철회 시: 보관 중인 검체는 지체 없이 폐기됩니다

□ 유전체 분석에 동의합니다 / □ 동의하지 않습니다

서명: _______________  날짜: ____년 __월 __일
```

### 5.2 잔여 검체 보관 동의 (생명윤리법)
```
시험 후 남은 혈액 검체를 향후 연구에 사용할 수 있습니다.

- 보관 기간: [N]년
- 보관 장소: [기관명]
- 폐기 절차: [방법]
- 2차 활용: □ 동의합니다 / □ 동의하지 않습니다

서명: _______________  날짜: ____년 __월 __일
```

### 5.3 연구 결과 통보 동의
```
연구 결과 중 귀하의 건강과 관련된 중요한 소견이 발견되면 알려드릴 수 있습니다.

□ 통보를 희망합니다 / □ 통보를 희망하지 않습니다

서명: _______________  날짜: ____년 __월 __일
```

## 6. 전자동의(eConsent) 추가 사항

> 사용자 요청 시에만 추가. MFDS 전자동의 가이드라인 준수.

```
전자동의 안내

이 임상시험에서는 전자적 방법으로 동의를 받을 수 있습니다.

- 전자 서명 방법: [태블릿/웹 시스템 등]
- 본인 확인 절차: [인증 방법]
- 데이터 보안: [암호화, 접근 제한 등]
- 저장 방법: [서버/시스템]
- 종이 동의서와 동등한 법적 효력을 가집니다
- 동의서 사본을 전자 파일(PDF)로 제공받으실 수 있습니다
```
