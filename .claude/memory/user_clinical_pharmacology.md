임상약리 분야 전문가. 시험 유형별 입력 요건을 구분:
- **FIH (First-in-Human)**: IB(시험자자료집)가 필수 — 신약이라 기존 정보 없음
- **DDI, BE, Food Effects 등**: 약물명만 있으면 기존 문헌/공개 데이터로 충분 — 이미 허가된 약물 대상
