# 시험 정보

## 기본 정보
- **약물명**: Amlodipine besylate 5mg 정제
- **시험 유형**: BA/BE (생물학적동등성)
- **적응증**: 고혈압(hypertension), 협심증(angina pectoris)
- **시험 단계**: Phase 1
- **대상 집단**: 건강한 성인 남성 지원자
- **의뢰자명**: [E2E 테스트용]
- **투여 경로/제형**: 경구(oral) / 정제(tablet)

## 시험 목적
시험약(제네릭 amlodipine besylate 5mg 정제)과 대조약(원개발사 amlodipine besylate 5mg 정제)의 생물학적동등성을 평가한다.

## 추가 정보
- **IB 제공 여부**: 불필요 (허가 약물)
- **유전체/인체유래물 연구**: 미포함
- **약물 계열**: Dihydropyridine calcium channel blocker (CCB)
- **작용 기전**: L-type calcium channel 차단 → 혈관 평활근 이완 → 혈압 강하

## 알려진 PK 특성 (참고)
- 반감기(t½): 약 30-50시간
- Tmax: 6-12시간
- 생체이용률: 약 64-90%
- 주요 대사: CYP3A4 (간대사)
- BCS 분류: Class I (고용해성, 고투과성)
