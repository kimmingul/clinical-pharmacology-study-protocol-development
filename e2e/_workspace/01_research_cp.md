## 2. PK/PD 문헌 요약

### 2.1 약동학(PK) 파라미터

아래 파라미터는 FDA 공인 처방 정보(Norvasc® 라벨) 및 복수의 임상약리 문헌에서 수집하였다.

| PK 파라미터 | 값 | 비고 |
|------------|-----|------|
| Tmax | 6–12시간 (중앙값 약 8시간) | 단회 경구 투여 후 |
| Cmax (5mg 단회, 공복) | 약 3.9–4.0 ng/mL | 건강한 중국인 지원자 [Wang et al., 2020, PMID: 32034814] |
| AUC0-t (5mg 단회) | 보고값 상이 (약 150–250 ng·h/mL 범위) | 집단·조건에 따라 상이 [출처 미확인 — 검증 필요: 수치는 복수 문헌 기반 추정] |
| AUC0-∞ (S-amlodipine, 한국인) | 175.3 ± 45.1 ng·h/mL (대조약) | [Kim et al., 2007, PMID: 17213004] |
| t½ (반감기, terminal) | 30–50시간 (평균 약 35–45시간) | FDA 라벨 [Norvasc® Label, FDA, 2013]; 정맥 투여 후 약 34시간, 경구 후 약 36시간 [Abernethy et al., 1993, PMID: 1534713] |
| 생체이용률 (F) | 64–90% (절대값) | 저 first-pass 효과가 특징 [FDA 라벨] |
| 단백결합률 | ~93% (고혈압 환자) | FDA Norvasc® 라벨 [PMID 확인 필요 — 라벨 원문 기준] |
| 분포용적 (Vd) | 21 L/kg | 광범위 분포 [Abernethy et al., 1993, PMID: 1534713] |
| 청소율 (CL) | 약 7 mL/min/kg | 정맥 단회 10mg 투여 기준 [Abernethy et al., 1993, PMID: 1534713] |
| 식이 영향 | 없음 (Cmax, AUC 불변) | BCS 특성 및 FDA 라벨 확인 |

**핵심 PK 특징:**
- 긴 반감기(30–50시간)로 인해 steady-state 도달에 약 7–8일 소요
- 고단백결합(93%), 광범위 분포(Vd = 21 L/kg)
- 식사 영향 없음 → 공복/식후 별도 시험이 필요하나, 일부 규제당국은 공복 단독 허용 [출처 미확인 — 검증 필요]
- Tmax 6–12시간으로 흡수 단계가 길어 채혈 프로토콜 설계 시 충분한 초기 밀집 채혈 필요

### 2.2 대사 경로 (CYP, 수송체)

| 항목 | 내용 |
|------|------|
| 주요 대사 효소 | CYP3A4 (주), CYP3A5 (부) |
| 대사 형태 | Dihydropyridine moiety의 산화적 탈수소화(dehydrogenation) → 비활성 pyridine 유도체(M9 주성분) |
| 활성 대사체 | 없음 — 약 90%가 비활성 대사체로 전환 |
| 주요 비활성 대사체 | M9 (pyridine 유도체); 이후 O-demethylation, O-dealkylation, 산화적 탈아미노화(oxidative deamination)로 추가 대사 |
| First-pass 효과 | 없음 (No significant presystemic/first-pass metabolism) — 다른 dihydropyridine 계열과 구별되는 특징 |
| 간 추출율 | 낮음 (low hepatic extraction ratio) |
| 배설 경로 | 소변으로 약 10%가 미변화체; 나머지 대사체로 배설 |
| 수송체 관여 | [출처 미확인 — 검증 필요: P-gp 관여 가능성 보고 있으나 임상적 유의성 미확립] |

**CYP3A4 임상적 함의 (BE 시험 관련):**
- CYP3A4 억제제(예: ketoconazole, clarithromycin) 또는 유도제(예: rifampicin)와 병용 금지/주의
- BE 시험 대상자 선정 시 CYP3A4 억제/유도 약물 복용 제외 기준 반드시 포함
- CYP3A4 다형성(poor/rapid metabolizer) 고려: 한국인 CYP3A4 표현형 분포는 caucasian과 유사하여 별도 보정 불필요

**참고 문헌:**
- [Nakamura K et al., 2013, PMID: 24301608] — Amlodipine metabolism in human liver microsomes and roles of CYP3A4/5
- [Pyo J et al., 2020, PMID: 32256614 — PMC7216797] — Effects of CYP450 oxidoreductase genotypes on amlodipine PK in healthy Korean subjects
- [Meredith PA and Elliott HL, 1992, PMID: 1532771] — Clinical pharmacokinetics of amlodipine

### 2.3 BCS 분류

| 항목 | 내용 |
|------|------|
| BCS 분류 | Class III (보수적 분류) — 고용해성(high solubility), 저투과성(low permeability, inconclusive) |
| 일부 문헌 분류 | BCS Class I (고용해성, 고투과성) — 일부 문헌 및 FDA 제출 자료 |
| 용해성 | 생리적 pH 범위(1.2–6.8) 전 구간에서 고용해성 (high solubility) |
| 투과성 | In vitro 투과성 데이터 불확실; 그러나 인체에서 흡수분율(fraction absorbed) ≥96%로 보고 → 실질적으로 고투과성에 가까움 |
| Biowaiver 적용 가능성 | BCS 기반 biowaiver 신청 가능 (IR 제형); 다수의 IR 제형에서 생동성 확인됨 |

**BCS 분류 논쟁 요약:**
- 규제 문서에서는 in vitro 투과성 데이터의 불확실성을 이유로 보수적 Class III 지정
- 그러나 인체 흡수율 ≥96%, Tmax 6–12시간, 식이 영향 없음 등은 Class I에 부합하는 특성
- 본 BE 시험에서는 표준 2×2 교차 설계 in vivo 생동성 시험이 요구되며, biowaiver 적용은 해당 없음

**참고 문헌:**
- [Biowaiver monograph for amlodipine besylate, PMC12974563] — BCS 분류 상세 근거
- [Rosenberger J et al., 2007, PMID: 19876745] — Provisional BCS classification of world's top oral drugs
- [FDA Chemistry Review, Katerzia NDA 211340, 2019] — FDA BCS Class 1 언급 (내부 제출 자료)

### 3.2 주요 시험 상세 분석

#### NCT02974439 — 아mlodipine vs. Norvasc® BE 시험 (중국, 2016)
[출처: Wang et al., 2020, PMID: 32034814 / PMC7307347; clinicaltrials.gov/study/NCT02974439]

**시험 설계:**
- 유형: Open-label, 무작위배정, 단회 투여, 2-way crossover
- Washout: 15일 (반감기 약 45시간의 약 8배)
- 투여 조건: 공복 연구 1건 + 식후 연구 1건 (별개 시험)
- 용량: Amlodipine 5mg 정제 단회 경구 투여
- 대조약: Norvasc® (원개발사 amlodipine besylate 5mg)

**대상자:**
- 공복 시험: 건강한 지원자 24명 (여성 16/24명, 연령 18–54세, BMI 19.5–26.0)
- 식후 시험: 건강한 지원자 24명 (여성 16/24명, 연령 20–49세, BMI 19.1–25.0)

**주요 PK 결과 (공복 조건):**

| 파라미터 | 시험약(T) | 대조약(R) |
|----------|----------|----------|
| Cmax (ng/mL) | 3.881 ± 0.982 | 4.042 ± 1.147 |
| Tmax (중앙값, 시간) | 6.0 | 6.0 |

**주요 PK 결과 (식후 조건):**

| 파라미터 | 시험약(T) | 대조약(R) |
|----------|----------|----------|
| Cmax (ng/mL) | 3.312 ± 0.789 | 3.392 ± 0.902 |
| Tmax (중앙값, 시간) | 6.0 | 5.0 |

**생동성 판정:** 공복 및 식후 모두 생동성 확인 (90% CI AUC, Cmax 모두 80–125% 내)

**분석 방법:** LC-MS/MS, 비구획 분석(noncompartmental model)

## 4. PK 변동성 — Sample Size 계산 핵심 파라미터

### 4.1 개체내 변동계수(Intra-subject CV%)

| 파라미터 | Intra-subject CV% 범위 | 비고 |
|----------|----------------------|------|
| AUC0-t | 7.5–11.6% (일부 보고 10–18%) | Wang et al., 2020, PMID: 32034814; 복수 문헌 |
| AUC0-∞ | 10.2% (일부 보고 10–18%) | 복수 문헌 |
| Cmax | 12.4–18% | 복수 문헌; 가장 높은 변동성 파라미터 |

**핵심 결론:**
- Amlodipine은 고변동성 약물(Highly Variable Drug, CV% > 30%)에 해당하지 않음
- Cmax 기준 보수적 CV% = 18% 적용이 권장됨 (Sample size 계산 시)
- 보수적 추산(CV% = 18%, α = 0.05, power = 80%, acceptance limit 80–125%): 최소 21명의 유효 대상자 필요
- 탈락률 고려 20% 보정 시: 최소 26명 등록 권장 [출처 미확인 — 검증 필요: 구체 계산은 biostatistician 확인 필요]

**참고 문헌:**
- [Wang et al., 2020, PMID: 32034814 / PMC7307347]

- 48–144시간: Terminal elimination 특성화 (AUC0-∞ 신뢰성 확보)
- 144시간 이후: 일반적으로 정량 한계 이하로 예상되어 추가 채혈 불필요

**Washout 기간:** 최소 5×t½ = 175–225시간 → **실용적 적용: 14–21일(약 336–504시간)**, 실제 시험에서는 15일이 표준으로 사용됨 [NCT02974439, Wang 2020]
