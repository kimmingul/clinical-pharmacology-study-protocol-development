## 2. 규제 가이드라인 요약

### 2.1 MFDS 생물학적동등성시험 가이드라인 — 핵심 요건

**Reference**: [생물학적동등성시험 기준, 식품의약품안전처 고시, 최신판] — https://nedrug.mfds.go.kr

#### 2.1.1 시험 설계 (Study Design)

| 항목 | MFDS 요건 | Amlodipine 5mg BE 시험 적용 |
|------|----------|--------------------------|
| **기본 설계** | 2×2 교차설계 (2-sequence, 2-period crossover) 원칙 | 적용: 단순 2×2 교차설계 적절 |
| **반복 교차** | 고변동 약물(intra-subject CV >30%)에 한해 허용 | Amlodipine CV ~10%이므로 불필요 |
| **무작위화** | 순서 무작위 배정 필수, 군 간 균형 | 필수 |
| **대조약(Reference)** | 오리지널(RLD, Reference Listed Drug) 원칙 | Norvasc® 또는 국내 허가 오리지널 제품 |
| **눈가림** | 필수 아님, 가능한 경우 권장 | 공개 설계 허용 |

#### 2.1.2 시험대상자 (Subjects)

| 항목 | MFDS 요건 |
|------|----------|
| **대상** | 건강한 성인 자원자 (만 19세 이상) |
| **최소 대상자 수** | 최소 12명 (평가 가능); 통계적으로 산출된 수 적용 |
| **성별** | 원칙적으로 남녀 모두 포함; 단일 성별 허용 가능 (Amlodipine 시험: 남성 대상 가능) |
| **BMI** | 통상 18.5~30.0 kg/m² |
| **washout** | 반감기 5배 이상 또는 최소 1주 (더 긴 기준 적용) |

**Amlodipine 특이사항**: 반감기 30~50시간 → washout 기간 최소 150~250시간 = **약 7~11일 이상** 필요. 실무적으로 **21일 이상**(약 3~4주)의 휴약기를 적용하는 것이 일반적.

#### 2.1.3 시험 조건 (Study Conditions)

| 항목 | MFDS 요건 | Amlodipine 적용 |
|------|----------|---------------|
| **공복 시험** | 원칙 — 최소 10시간 공복 후 투여, 투여 후 4시간 금식 | 필수 (1차 시험) |
| **식후 시험** | 필요 시 별도 시행 (MFDS는 공복 우선) | Amlodipine: 식이 영향 없음 — 공복 단독 시험으로 충분할 수 있음 |
| **투여 조건** | 240 mL 물과 함께 투여 | 적용 |

#### 2.1.4 채혈 및 PK 분석 (Sampling and PK)

| 항목 | MFDS 요건 | Amlodipine 5mg 적용 |
|------|----------|-------------------|
| **채혈 기간** | 투여 후 최소 3반감기(t½) 이상 | 3 × 50시간 = 150시간 → 최소 **96~120시간** 이상 권장 |
| **채혈 시점 수** | Cmax 전후 충분한 시점 포함, AUC의 80% 이상 포착 | Tmax 6~12시간 전후 집중 채혈 필요; 총 72~120시간 추적 |
| **평가변수** | AUC₀₋ₜ, AUC₀₋∞ (참고), Cmax 필수 | AUC₀₋ₜ + Cmax (1차); AUC₀₋∞, Tmax (참고) |
| **분석법** | 밸리데이션된 생체시료분석법 (LC-MS/MS 권장) | 적용 |
| **PK 분석** | 비구획분석(NCA) 원칙 | 적용 (WinNonlin, R 등) |

#### 2.1.5 동등성 판정 기준 (BE Acceptance Criteria)

| 항목 | MFDS 요건 |
|------|----------|
| **동등성 한계** | 로그 변환 후 90% CI가 **80.00%~125.00%** 이내 |
| **1차 평가변수** | AUC₀₋ₜ **및** Cmax 동시 충족 |
| **통계 방법** | ANOVA (sequence, subject nested in sequence, period, formulation) |
| **변환** | 자연로그(ln) 변환 후 분석 |
| **결과 보고** | GMR (기하평균비) + 90% CI |
| **AUC₀₋∞** | 참고 자료 (판정 기준 아님) |

**Amlodipine 고변동약물 여부**: Amlodipine의 intra-subject CV는 Cmax 약 10.4%, AUClast 약 9.4%로 **고변동약물(CV >30%)에 해당하지 않음**. 따라서 표준 BE 기준(80~125%) 적용, 반복 교차설계 또는 확장 한계 적용 불필요.

**Reference**: [생물학적동등성시험 기준, 식품의약품안전처 고시, 발행일 확인 필요]; [Current Bioequivalence Study Designs in South Korea, Frontiers in Pharmacology, 2021, PMC8147690]

### 2.2 MFDS 통계 가이드라인 — BE 시험 핵심 요건

**Reference**: [생물학적동등성시험 통계분석 가이드라인, 식품의약품안전처, 발행일 확인 필요]

#### 2.2.1 Sample Size 산출

| 항목 | 요건 | Amlodipine 적용 |
|------|------|---------------|
| **Intra-subject CV** | 선행 연구 또는 문헌 근거 | CV(Cmax) ≈ 10.4%, CV(AUC) ≈ 9.4% |
| **GMR 가정** | 통상 95%~105% 또는 문헌 근거 | 95% 가정 가능 |
| **검정력 (Power)** | ≥80% 원칙 (90% 권장) | 80% 이상 |
| **동등성 한계** | 80~125% | 적용 |
| **탈락률 보정** | 예상 탈락률 반영 | 10~20% 보정 권장 |
| **최소 대상자** | 12명 이상 (평가 가능) | CV 낮아 이론적 최소값 도달 가능; 규정상 최소 12명 |

**산출 결과 예시 (참고)**: CV(Cmax) = 10.4%, GMR = 0.95, 동등성 한계 80~125%, 검정력 80% 기준 → 약 10~15명 산출 → 최소 규정 12명 충족; 탈락률 20% 보정 시 **15~18명** 모집 권장.

#### 2.2.2 통계 분석 절차

1. AUC₀₋ₜ, Cmax 자연로그(ln) 변환
2. 혼합효과 ANOVA 실행 (sequence, subject[sequence], period, formulation)
3. 제형 효과 LSM 차이 및 90% CI 산출
4. 역변환 → GMR 및 90% CI 보고
5. 90% CI 80.00~125.00% 이내: 동등성 판정

## 4. MFDS 임상시험 승인현황 — Amlodipine BE 시험

> **한계 명시**: 의약품안전나라(nedrug.mfds.go.kr)에 구조화된 공개 API가 없어 WebSearch를 통한 간접 조사를 실시함. 구체적 승인 번호 및 건수는 의약품안전나라 직접 접속으로 확인 권장.

### 4.1 국내 BE 시험 현황 요약

한국에서 amlodipine besylate 관련 BE 시험은 활발하게 수행되어 왔으며, 공개 문헌 및 의약품안전나라 등록 자료를 통해 다음과 같은 일반적 동향이 확인된다:

| 항목 | 내용 |
|------|------|
| **승인 기관** | 식품의약품안전처(MFDS) + 기관 IRB |
| **시험 기관 유형** | 국내 지정 임상시험 실시기관 (CTC/CRC 보유 의료기관) |
| **시험 설계** | 건강한 성인 남성 대상, 2×2 교차설계, 공복 조건 원칙 |
| **대상자 수** | 통상 15~24명 (Amlodipine 낮은 CV 반영) |
| **Washout** | 통상 3~4주 (21~28일) |
| **등록 의약품 수** | 암로디핀베실산염 5mg 제네릭 제품 다수 의약품안전나라 등재 확인 |

### 4.2 공개 문헌에서 확인된 국내 BE 시험 사례

### 4.3 국내 승인 현황 (의약품안전나라 기반)

- 의약품안전나라 검색 결과, 산도스암로디핀정5밀리그램, 동화암로디핀베실산염정, 아드정(암로디핀베실산염) 등 다수의 제네릭 제품이 등재되어 있으며, 이는 과거 BE 시험 완료 후 허가된 제품들임.

## 5. 규제 고려사항 요약

### 5.1 Amlodipine 5mg BE 시험의 핵심 규제 요건

| 항목 | 요건 | 근거 |
|------|------|------|
| **IND 신청** | 식약처 IND 신청 (프로토콜, 임상시험용 의약품 자료, CMC 등) | KGCP, 약사법 제34조 |
| **IRB 승인** | 기관생명윤리위원회 심의 및 승인 | 생명윤리법, KGCP |
| **시험 기관** | 식약처 지정 임상시험 실시기관 | KGCP |
| **시험 설계** | 2×2 교차설계, 공복, 무작위 | MFDS BE 가이드라인 |
| **대조약** | 국내 허가 오리지널 제품(Norvasc® 또는 동등) | MFDS BE 가이드라인 |
| **평가변수** | AUC₀₋ₜ + Cmax (90% CI: 80~125%) | MFDS BE 가이드라인 |
| **채혈 기간** | ≥96시간 이상 (t½ × 3 이상) | MFDS BE 가이드라인 |
| **Washout** | ≥21일 (t½ × 5 = 약 150~250시간 = 7~10일 최소, 실무 3~4주) | MFDS BE 가이드라인 |
| **통계** | ANOVA, 로그변환, 90% CI 산출 | MFDS 통계 가이드라인 |
| **분석법 밸리데이션** | LC-MS/MS 등 밸리데이션 필수 | MFDS 생체시료분석법 가이드라인 |
| **SAE 보고** | 24시간 이내 식약처/IRB 보고 | KGCP |
| **시험 결과 보고** | 종료 후 식약처에 결과보고서 제출 | KGCP, 약사법 |

### 5.2 주요 규제 리스크 및 고려사항

1. **Washout 기간**: Amlodipine의 긴 반감기(30~50시간)로 인해 washout 기간이 최소 3~4주 필요. 시험 기간이 길어져 대상자 탈락 위험 증가 → 탈락 보정 모집 필요
2. **채혈 기간 및 시점**: Tmax(6~12시간) + 이후 긴 소실 기간(96시간 이상) → 채혈 부담 크므로 채혈 스케줄 최적화 필요
3. **공복 요건**: MFDS는 공복 우선. Amlodipine은 식이 영향이 없으므로 공복 단독 시험으로 충분함(식후 시험 추가 불필요 가능성 높음)
4. **5mg Biowaiver 가능성**: FDA PSG는 10mg 기준 in vivo 시험 후 5mg biowaiver 허용. MFDS도 유사 원칙 적용 가능하나 사전 식약처 확인 필요
5. **ICH E6(R3) Annex 1 준수**: 계획서 필수 요소는 Annex 1(비기술적 원칙)에 정의. R2 구조와 상이하므로 주의
