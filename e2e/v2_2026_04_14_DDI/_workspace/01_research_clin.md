# 임상의 안전성 조사 보고서 (Clinician Research Report)

**시험**: Clopidogrel-Omeprazole DDI 시험 (건강한 성인 남성, Phase 1)  
**작성일**: 2026-04-14  
**담당**: Clinician Agent  

---

## 섹션 1: Clopidogrel 안전성 프로파일

> 상세: `_workspace/01_references/safety/AE_profile_clopidogrel.md`

### 1-1. 주요 이상반응 (CAPRIE 기반, n=9,599)

| 이상반응 | 빈도 | 비고 |
|---------|------|------|
| 발진(Rash) | 6.0% | 아스피린보다 유의하게 높음 (p<0.001) |
| 설사(Diarrhea) | 4.46% | 아스피린보다 유의하게 높음 (p<0.001) |
| GI 출혈 | 1.99% | 아스피린(2.66%)보다 낮음 |
| 두개내 출혈 | 0.33% | 아스피린(0.47%)보다 낮음 |
| 호중구감소증 | 0.10% | 아스피린(0.17%)과 유사 |
| TTP | Very rare (~1/200,000) | 주로 2주 이내 발병 |

**근거**: Harker et al. 1999 (PMID 10514023), CAPRIE 1996 (PMID 8918275)

### 1-2. 건강인 Phase 1 시험 데이터

- **한국인 건강인 30명** (Shim 2010, PMID 20974324): 300mg 부하 + 75mg × 6일 — **이상반응 없음**
- **건강한 남성 27명** (Kubitza 2012, PMID 24281379): clopidogrel 300mg 단회 — SAE 없음, 출혈 시간 약 2배 연장
- **건강인 14명** (Price & Teirstein 2008, PMID 18774008): 600mg 부하 — 혈소판 억제 8.5~97.7% 개인 간 변동성

### 1-3. 혈소판 기능 회복 (Washout 근거)
- 일 약 17% 지수적 회복 → **14일 이상 washout** 권장 (Price & Teirstein 2008)
- 혈소판 수명(8-10일) 기준 완전 회복: 7-14일

## 섹션 2: Omeprazole 안전성 프로파일

> 상세: `_workspace/01_references/safety/AE_profile_omeprazole.md`

### 2-1. 주요 이상반응 (단기 투여 기준)

| 이상반응 | 빈도 | 근거 |
|---------|------|------|
| 두통 | ~4.7% | Joelson 1992 (PMID 1397750) |
| 설사 | ~3.2% | Joelson 1992 |
| 오심/복통 | ~2-3% | 라벨, 임상시험 |
| 발진 | <1% | Uncommon |
| SAE | 없음 | 장기 사용에서도 SAE 미확인 |

### 2-2. 고용량(80mg) 사용 시 고려
- 표준 용량(20-40mg) 대비 2-4배 고용량 — GI 증상 빈도 증가 가능
- 단기 사용(7-14일)이므로 장기 합병증(저마그네슘혈증 등) 무관
- DDI perpetrator 역할 목적으로 타당: CYP2C19 억제 강도 극대화

## 섹션 3: 병용 안전성 (Clopidogrel + Omeprazole)

> 상세: `_workspace/01_references/literature/PMID_25949846.md`

### 3-1. 이상반응 프로파일 변화

| 항목 | 병용 시 변화 | 임상적 의미 |
|------|------------|-----------|
| 항혈소판 효과 | 감소 (활성 대사체 형성 감소) | DDI 평가의 목적 자체 |
| GI 출혈 위험 | **감소** (RR 0.13, meta-analysis) | 안전성 이점 |
| 심혈관 이벤트 | 변화 없음 (COGENT 결과) | 단기 건강인 시험에서 무관 |
| 두통/GI 증상 | 각 약물 이상반응 중첩 가능 | 경미, 지지 치료로 관리 |

**핵심 결론**: Clopidogrel + Omeprazole 병용은 건강인 단기 시험에서 **안전성 문제가 크지 않으며**, GI 출혈 측면에서 오히려 보호 효과. 항혈소판 효과 감소는 시험 목적(DDI 평가) 자체이므로 의도된 현상.

## 섹션 4: P2Y12 억제제 계열 효과 (Class Effect)

> 상세: `_workspace/01_references/safety/class_effect_P2Y12_inhibitors.md`

| Class Effect | 기전 | Clopidogrel 특이점 |
|------------|------|-----------------|
| 출혈 위험 증가 | ADP 수용체 차단 → 혈소판 응집 억제 | 동 계열 중 상대적으로 낮은 출혈 강도 |

| 설사 | Thienopyridine GI 이상반응 | 4.46% |

## 섹션 5: 선정/제외 기준 근거 후보

### 5-1. 필수 제외 기준 (안전성 근거)

| 제외 기준 | 근거 | 문헌 |
|---------|------|------|
| 출혈 병력 (GI, 두개내, 안구 내 등) | 출혈 위험 상가적 증가 | CAPRIE, PMID 8918275 |
| 혈소판 질환 또는 TTP 과거력 | TTP 재발 위험 | Bennett 2000, PMID 10852999 |
| 항혈전제 복용 (아스피린, NSAIDs, 항응고제) | 출혈 시간 상가적 연장 | Kubitza 2012, PMID 24281379 |
| 혈소판 수 <150 × 10⁹/L | TTP 조기 발견 기저치 확보 | Bennett 2000 |
| 혈소판 수 이상(>450 × 10⁹/L) | 혈전 위험 | 표준 임상 기준 |
| 활동성 GI 질환 (궤양, 염증성 장질환) | GI 출혈 위험 증가 | CAPRIE |
| 수술/처치 예정 (시험 기간 ±2주 내) | 출혈 시간 연장 상태에서 수술 위험 | Kubitza 2012 |
| 알코올/약물 남용 | 간 기능 영향, 출혈 위험 | 표준 임상 기준 |
| Omeprazole 금기 사항 | 과민반응 | FDA 라벨 |
| PPI 투여 중인 자 | 기저 CYP2C19 억제 상태 영향 | DDI 설계 타당성 |

### 5-2. CYP2C19 PM 포함/제외 여부 임상 근거

**DDI 평가 목적 측면**
- CYP2C19 PM: 활성 대사체 기저 형성 매우 낮음 → Omeprazole의 추가 억제 효과 측정 어려움
- **PM 제외 권장**: 1차 평가변수(활성 대사체 AUC의 GMR)의 통계적 검정력 확보
- **EM 또는 IM 선정**: DDI 효과 탐지 민감도 최대화

**안전성 측면**
- PM에서 항혈소판 효과 감소 → 혈전 위험 증가 (건강인에서 임상적 의미 낮음)
- PM에서 출혈 위험 오히려 낮음 (활성 대사체 적음): Ganoci 2024, Jung 2025

**결론**: **CYP2C19 PM 제외** (DDI 평가 적절성 이유가 주된 근거, 안전성 문제보다)

### 5-3. 추가 선정 기준 고려

| 기준 | 근거 |
|------|------|
| BMI 18.5-27 kg/m² | 건강인 표준 범위, PK 변동성 최소화 |
| 연령 19-45세 | 연령 관련 혈소판 기능 변화 최소화 |
| 남성 한정 | 여성 호르몬의 혈소판 기능 영향 배제, 시험 정보 일치 |
| 흡연자 제외 | 혈소판 반응성 증가 영향 |
| CYP2C19*17 (UM) 선정 고려 | 활성 대사체 초과 형성 → 과도한 억제 위험 여부 판단 필요 |

## 섹션 6: 모니터링 계획 요약

> 상세: `_workspace/01_references/safety/safety_monitoring_rationale.md`

### 6-1. 모니터링 일정 (요약)

| 항목 | 시점 | 근거 |
|------|------|------|
| 혈소판 수 | Screening, Day 1, 5-7, 14 | TTP 조기 발견 (Bennett 2000) |
| CBC 전체 | Screening, Day 1, 5-7, 14 | 호중구감소, 빈혈 |
| LDH | Screening, 이상 시 즉시 | TTP 용혈 마커 |
| AST/ALT | Screening, 종료 | Omeprazole 간독성 |
| VerifyNow P2Y12 | 기저, Tmax, 24h, 종료 | 혈소판 기능 평가 |
| 활력징후 | 투여 전, 1, 2, 4, 6h | 출혈 징후 |
| 출혈 문진 | 매 방문 | 임상 이상반응 |

### 6-2. 응급 대응 절차

1. **TTP 의심** (혈소판감소 + LDH 상승 + Coombs 음성 빈혈):
   - 즉시 약물 중단 → 혈액과 긴급 자문 → 혈장교환 평가
   
2. **TIMI Major 출혈** (두개내, Hgb ≥5 g/dL 감소):
   - 즉시 중단 → 응급실 이송 → 지혈 처치

3. **아나필락시스**:
   - 에피네프린 0.5mg IM → 응급실 이송

## 임상의 의견 요약

1. **Clopidogrel 건강인 단기 투여 안전성**: 문헌 근거상 안전하며 내약성 양호. 한국인 건강인 데이터(Shim 2010)에서도 이상반응 없음. 단, 출혈 시간 연장 및 혈소판 기능 억제는 예상된 약리 효과.

2. **핵심 위험: TTP**: 매우 드물지만(~1/200,000) 치명적. 투여 2주 이내가 주요 위험 기간 → 혈소판 수/LDH 집중 모니터링 필수. 시험 기간 중 혈소판 <100×10⁹/L 발생 시 즉시 중단 기준 설정 권고.

3. **CYP2C19 PM 제외**: DDI 평가 타당성(활성 대사체 기저 농도 낮아 Δ측정 어려움)이 주요 이유. PM 안전성은 오히려 양호(출혈 위험 낮음).

4. **병용 안전성**: Clopidogrel + Omeprazole 병용 — 항혈소판 효과 감소는 시험 목적이며, GI 안전성은 PPI 보호로 오히려 향상. 건강인 단기 시험에서 심각한 안전성 우려 없음.

5. **Washout 14일 이상**: 혈소판 기능 완전 회복 필요 (Price & Teirstein 2008 근거).
