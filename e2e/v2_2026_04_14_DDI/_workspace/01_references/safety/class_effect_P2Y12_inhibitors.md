# Class Effect — P2Y12 수용체 억제제

**약물 계열**: P2Y12 수용체 억제제 (Thienopyridine 및 Non-thienopyridine)  
**해당 약물**: Clopidogrel (thienopyridine), Prasugrel (thienopyridine), Ticagrelor (cyclopentyl triazolopyrimidine), Ticlopidine (thienopyridine, 현재 거의 미사용)  
**작성일**: 2026-04-14  
**근거 문헌**: Fahmy et al. 2019 (PMID 30738497), Bennett 1999 (PMID 10573042), 기타

---

## 1. P2Y12 억제제 계열 공통 이상반응 (Class Effect)

### 핵심 기전
모든 P2Y12 억제제는 ADP에 의한 혈소판 응집을 억제 → **출혈 위험 증가**가 계열 공통 효과

| 이상반응 | Class Effect 근거 | 약물별 차이 |
|---------|-----------------|-----------|
| **출혈 (모든 부위)** | 모든 P2Y12 억제제 공통. 혈소판 ADP 수용체 차단 기전에 기인 | Clopidogrel < Ticagrelor ≈ Prasugrel (출혈 강도) |
| **GI 출혈** | ADP 수용체 억제 → 점막 혈소판 응집 억제 | Clopidogrel GI 입원율 가장 높음 (FAERS, Fahmy 2019) |
| **두개내 출혈** | 공통 위험. FAERS: clopidogrel 치명률 상대적 낮음 | Prasugrel > Clopidogrel ≈ Ticagrelor |
| **혈전성 혈소판감소자반증(TTP)** | Thienopyridine 계열 한정 class effect. ADAMTS13 억제 기전 제안 | Ticlopidine >> Clopidogrel >> Prasugrel (매우 드묾) |
| **피부 발진(Rash)** | Thienopyridine 계열에서 보고 | Clopidogrel 6%, Ticlopidine 더 높음 |
| **설사(Diarrhea)** | Thienopyridine 계열 GI 이상반응 | Clopidogrel 4.46% |

## 2. 약물별 이상반응 프로파일 비교

| 이상반응 | Clopidogrel | Prasugrel | Ticagrelor |
|---------|-------------|-----------|------------|
| Major bleeding | 기준 | 높음 (TRITON) | 유사-높음 |
| TIMI minor bleeding | 낮음 | 높음 | 중간 |
| GI 출혈 | 1.99% (CAPRIE) | 유사 | 유사 |
| 두개내 출혈 | 0.33% | 유사 | 유사 |
| 치명적 출혈 | 상대적 낮음 | 높음 | 중간 |
| 호흡곤란(Dyspnea) | 드묾 | 드묾 | **최대 15%** (ticagrelor 특이) |
| TTP | Very rare | Very rare | 보고 있음 (ticagrelor) |
| 피부 발진 | 6.0% | 드묾 | 드묾 |
| 설사 | 4.46% | 드묾 | 드묾 |

**출처**: Fahmy et al. 2019 (PMID 30738497) — FAERS 분석 (clopidogrel 2,252건, ticagrelor 2,450건, prasugrel 549건)

## 3. Thienopyridine 계열 특이 SAE: TTP

### 계열 내 비교
| 약물 | TTP 발생 빈도 (추정) | 발병 시기 |
|------|-------------------|---------|
| Ticlopidine | 1/2,000~1/4,000 | 2-12주 |
| Clopidogrel | 1/200,000 (추정) | 주로 2주 이내 |
| Prasugrel | Very rare (보고 드묾) | 불명 |

- Bennett 1999 (PMID 10573042): Ticlopidine TTP 98건 분석으로 thienopyridine class effect 확립
- Clopidogrel은 ticlopidine 대비 TTP 빈도 극히 낮으나 zero risk 아님

## 4. P2Y12 억제제의 출혈 위험 임상적 의미

### 건강인 대상 시험에서의 위험 평가
- **침습적 처치**: 혈소판 기능 억제 상태에서 채혈, 주사, 카테터 삽입 시 출혈 시간 연장
  - Kubitza 2012 (PMID 24281379): clopidogrel 단독으로 출혈 시간 1.96배 연장
- **활동 제한 필요**: 격렬한 운동, 접촉 스포츠 등 외상 위험 활동 제한
- **병용 금지 약물**: 아스피린, NSAIDs, 항응고제 — 출혈 위험 상가적 증가

### 역설적 보호 효과 (Clopidogrel + Omeprazole 맥락)
- PPI 병용은 GI 출혈 위험 감소 (RR 0.13, meta-analysis)
- 본 DDI 시험: Omeprazole 병용이 오히려 GI 안전성 향상 기대

## 5. 한국인에서의 특별 고려사항

- **CYP2C19 Poor Metabolizer (PM) 비율**: 한국인 ~15% vs 백인 2-3%
- PM에서 clopidogrel 활성 대사체 형성 감소 → 혈소판 억제 효과 감소
- **출혈 위험**: PM에서 항혈소판 효과 감소 → 출혈 위험이 오히려 낮거나 유사
  - Ganoci 2024: PM/IM 군 출혈 13.1% vs EM/UM 군 16.6-20.5% — PM 출혈 낮음
  - Jung 2025: CYP2C19 LOF 보유자 출혈 0.6% vs 비보유자 0.8% — 차이 없음
- **DDI 시험 관련성**: PM 대상자는 DDI 효과 평가에 적합하지 않으나 안전성은 유리
