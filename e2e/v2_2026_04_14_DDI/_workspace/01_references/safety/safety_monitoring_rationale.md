# 안전성 모니터링 시점·항목 과학적 근거

**시험**: Clopidogrel-Omeprazole DDI 시험 (건강인 Phase 1)  
**작성일**: 2026-04-14  
**근거 문헌**: Price & Teirstein 2008 (PMID 18774008), Kubitza 2012 (PMID 24281379), Bennett 2000 (PMID 10852999), Shim 2010 (PMID 20974324)

---

## 1. 모니터링 항목별 근거

### 1-1. 혈소판 수(Platelet Count)

**왜 측정하는가**
- Clopidogrel 투여 후 TTP 발생 시 혈소판감소증이 초기 징후 (Bennett 2000, PMID 10852999)
- TTP 10명 중 10명이 14일 이내 발병 → 투여 초기 집중 모니터링 필요

**측정 시점 근거**
| 시점 | 근거 | 목적 |
|------|------|------|
| Screening (Day -14~-7) | 기저치 확립. 혈소판감소증 선별 | 기저 혈소판 이상 배제 |
| Day 1 (투여 전) | Period 시작 시 기저치 재확인 | 교차기간 이상 배제 |
| Day 5-7 (투여 중) | TTP 발병 위험 기간 (1~2주 내) | TTP 조기 발견 |
| Day 14 (추적) | 투여 종료 후 지연 발현 가능성 | 지연 TTP 배제 |

**중지 기준 임계값**
- 혈소판 <100 × 10⁹/L 또는 기저치 대비 >25% 감소: 즉시 약물 중단 + 혈액과 자문
- 혈소판 <50 × 10⁹/L: 응급 평가 (TTP 의심 시 혈장교환 고려)

---

### 1-2. 혈소판 기능 검사(Platelet Function Test, VerifyNow P2Y12)

**왜 측정하는가**
- 본 시험의 1차 PD 평가변수 (PRU; P2Y12 Reaction Units)
- 출혈 위험 개인차 반영: Price & Teirstein 2008 (PMID 18774008) — 600mg 부하 후 1일 억제율 8.5~97.7% 매우 큰 범위
- 혈소판 억제 정도와 출혈 시간 상관 관계 확인 (Kubitza 2012, PMID 24281379)

**측정 시점 근거**
| 시점 | 근거 | 목적 |
|------|------|------|
| 기저치 (투여 전) | 개인별 기저 혈소판 반응성 확립 | 정상 범위 확인, 고반응성 선별 |
| Tmax 시점 (75mg 투여 후 2-4시간) | 활성 대사체 Cmax ≈ 0.5-2시간 → 혈소판 억제 최대 시점 | 약력학적 평가 |
| 투여 후 24시간 | 일일 유지 상태 평가 | 안정 상태 억제 수준 |
| 각 Period 종료 시 | 혈소판 기능 회복 확인 | Washout 충분성 검증 |

**혈소판 기능 회복 시간 (Washout 설계 근거)**
- Price & Teirstein 2008: 일일 약 17% 지수적 회복 → 7-10일 후 기저치의 ~50%, 14일 후 ~85% 회복
- **혈소판 수명(8-10일)**을 고려한 완전 회복: 7-14일
- Washout 14일 이상 권장 근거: 혈소판 수명 고려 완전 기능 회복

---

### 1-3. 출혈 시간(Bleeding Time) 또는 PFA-100

**왜 측정하는가**
- Kubitza 2012 (PMID 24281379): clopidogrel 300mg 단독 투여 후 출혈 시간 기저치 대비 1.96배 연장
- 임상적 출혈 위험의 대리 지표 (surrogate marker)

**제한점**: 출혈 시간은 변동성이 크고 표준화 어려움 — VerifyNow P2Y12가 더 표준적

---

### 1-4. 활력징후(Vital Signs)

**왜 측정하는가**
- 출혈 이상반응 시 혈압 저하, 맥박 증가 → 조기 발견
- Omeprazole: 심혈관 이상반응 없으나 기저치 확립 필요

**측정 시점**
| 시점 | 내용 |
|------|------|
| 각 투여 전 | 기저치 확립 |
| 투여 후 1, 2, 4, 6시간 | Clopidogrel 활성 대사체 Tmax 전후 |
| 퇴원 시 | 안정성 확인 |
| 이상반응 발생 시 | 즉시 측정 |

---

### 1-5. 혈액검사(CBC, Chemistry)

**CBC (Complete Blood Count)**
- WBC (백혈구): 호중구감소증(0.10%), 면역 반응 모니터링
- Hgb/Hct: 출혈 발생 시 빈혈 조기 발견
- Platelet: TTP 모니터링 (상기)
- 말초혈액도말(peripheral smear): 혈소판감소 시 schistocyte 확인 → TTP 진단

**Chemistry (혈청 화학)**
- LDH (Lactate Dehydrogenase): TTP 시 상승 (용혈 마커) — 혈소판감소 동반 시 즉시
- AST/ALT: Omeprazole 간독성 모니터링 (드물지만)
- BUN/Creatinine: 신기능 (TTP 신침범 평가)
- Haptoglobin: TTP 용혈 확인 (감소)

**측정 시점**
| 시점 | 이유 |
|------|------|
| Screening | 기저치, 선정 기준 확인 |
| Day 1 각 Period | 교차 사이 회복 확인 |
| Day 5 ± 2 | TTP 위험 기간 (clopidogrel 투여 중) |
| 시험 종료/추적 | 기저치 회복 확인 |
| 증상 발생 시 즉시 | 이상 징후 대응 |

---

### 1-6. GI 증상 모니터링

**왜 중요한가**
- Omeprazole 80mg: 설사(3.2%), 두통(4.7%) 발생 가능
- Clopidogrel + 설사 4.46%
- 병용 시 GI 증상 중복 가능성

**모니터링 방법**
- 매 방문 구조화 문진: 복통, 설사, 오심, 구토, 흑색변(melena), 혈변(hematochezia)
- 흑색변 발생 시: 즉시 입원 + 위장관 내시경 평가

---

### 1-7. 출혈 징후 문진 (Bleeding Symptom Checklist)

**매 방문 시 확인 항목**
1. 피부: 자반(purpura), 혈종(hematoma), 멍(ecchymosis)
2. 점막: 잇몸 출혈, 비출혈(epistaxis)
3. 비뇨기: 혈뇨(hematuria)
4. GI: 흑색변, 혈변
5. 두통: 갑작스럽고 심한 두통 → 두개내 출혈 의심

## 3. 입원 기간 임상적 적절성

**입원 필요성 근거**
- 집중 혈소판 모니터링 (TTP 조기 발견)
- 출혈 발생 시 즉각 의료 대응
- PK 채혈 스케줄 준수 (Tmax 전후 빈번한 채혈)

**권장 입원 기간**: 약물 투여일 전날 입원 → 최종 PK 채혈 완료 후 퇴원
- 단기 입원 정당화: PK/PD 목적 + 안전성 모니터링 동시 수행
