# LTA (Light Transmittance Aggregometry) — ADP 유도 혈소판 응집 억제율

## 기본 정보

| 항목 | 내용 |
|------|------|
| 측정 대상 | ADP 유도 혈소판 응집 억제율 (IPA, % Inhibition of Platelet Aggregation) |
| 시료 | PRP (Platelet-Rich Plasma) — 구연산나트륨 항응고 혈액 원심분리 |
| 분석 방법 | 광투과율 응집 측정법 (turbidimetric aggregometry) |
| 자극 농도 | ADP 5 μmol/L 또는 10 μmol/L (표준화) |
| 단위 | % IPA 또는 최대 혈소판 응집률 (% PA, maximum aggregation) |
| 검증 상태 | 확립된 참조 방법(gold standard), 완전 검증 |

## 작동 원리

1. PRP를 광원과 광검출기 사이 큐벳에 넣고 교반
2. 기준선: PRP(흐린 상태) — 광투과율 낮음 (0% 기준)
3. ADP 첨가 → 혈소판 활성화 및 응집 → 광투과율 증가
4. 최대 광투과율 상승 = 최대 혈소판 응집 (% PA)
5. % IPA = 100% − % PA (억제제 투여 후) / % PA (기저치)

## 시험약과의 연관성 (Clopidogrel DDI)

- Clopidogrel H4가 P2Y12를 비가역적 차단 → ADP 자극에 대한 응집 억제 → % IPA 증가
- Omeprazole DDI → H4 형성 감소 → ADP 응집 억제 감소 → % PA 증가, % IPA 감소
- 본 시험에서 2차 PD 지표로 활용

## 표준화 고려사항

- **ADP 농도**: 10 μmol/L 선호 (표준화). 5 μmol/L과 20 μmol/L는 결과 매우 유사 (r > 0.95; PMID: 21192314)
- **PPP (Platelet-Poor Plasma)**: 100% 광투과율 기준으로 사용
- **혈소판 수 보정**: 혈소판 수가 낮으면 응집 반응 감소 — 일부 연구에서 보정 권장 (PMID: 17957573)
- **온도·시간 조건**: 채혈 후 2시간 이내 검사 권장, 37°C 유지

## VerifyNow PRU와의 비교

| 항목 | LTA | VerifyNow PRU |
|------|-----|---------------|
| 분류 | 참조 방법(gold standard) | Point-of-care |
| 시료 | PRP (원심분리 필요) | 전혈 (즉시 사용) |
| 자극 물질 | ADP 단독 | ADP + PGE1 |
| 검사 시간 | 30–60분 | ~5분 |
| 특이성 | 중간 (P2Y1도 반응) | 높음 (P2Y12 특이적) |
| 표준화 | 어려움 | 표준화 카트리지 |
| 상관성 | LTA vs VerifyNow: r = 0.82–0.90 | — |
| 연구 신뢰도 | 높음 (오래된 참조 데이터) | 높음 (최신 임상 연구) |

## 측정 시점 권고 (본 DDI 시험)

- 기저 (Pre-dose Day 1): 5 또는 10 μmol/L ADP, 응집 기저값 확인
- Clopidogrel 단독 정상 상태 (Day 5–7, 복용 2–4 hr 후)
- Omeprazole 병용 정상 상태 (Day 12–14, 복용 2–4 hr 후)

## 한계

- 표준화 어려움 (기관 간 변동 크음)
- PRP 제조 과정에서 혈소판 활성화 가능
- 결과 해석에 전문성 필요
- 검사 시간이 길어 point-of-care 부적합
- 혈소판 수가 낮거나(< 100,000/μL) 헤마토크릿 이상 시 결과 영향
