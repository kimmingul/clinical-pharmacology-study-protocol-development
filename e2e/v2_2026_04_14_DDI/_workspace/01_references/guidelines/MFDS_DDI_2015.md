## 핵심 요건 요약

### 1. 적용 범위

| 항목 | 내용 |
|------|------|
| 대상 약물 | 신약 및 적응증 확대 의약품으로서 DDI 평가가 필요한 경우 |
| 평가 유형 | ① 대사효소(CYP) 기반 DDI, ② 수송체(transporter) 기반 DDI, ③ 약력학적(PD) DDI |
| 단계적 접근 | In vitro 평가 → In vivo 임상시험 (단계적 risk-based approach) |
| 목적 | 허가사항(용법용량, 사용상 주의사항)에 DDI 정보 반영 |

### 2. In Vitro 평가 체계

#### 평가 대상 효소 및 수송체

| 구분 | 평가 대상 |
|------|----------|
| CYP 억제 (가역적) | CYP1A2, 2B6, 2C8, 2C9, 2C19, 2D6, 3A4 (7종 필수) |
| CYP 억제 (시간의존적, TDI) | 위 7종에 대해 TDI 평가 필수 |
| CYP 유도 | CYP1A2, 2B6, 3A4 (3종 필수) |
| P-gp | 기질/억제제/유도제 여부 평가 |
| BCRP | 기질/억제제 여부 평가 |
| OATP1B1, OATP1B3 | 간 수송체; 기질/억제제 여부 |
| OAT1, OAT3 | 신장 유기음이온 수송체 |
| OCT2 | 신장 유기양이온 수송체 |
| MATE1, MATE2-K | 신장 수송체 |

#### In Vivo 시험 결정 기준

| 기준 | 판단 |
|------|------|
| 가역적 억제: [I]/Ki > 0.1 또는 R-value > 1.02 | In vivo 시험 필요 |
| TDI: KI, kinact 기반 기준 초과 | In vivo 시험 강력 권고 |
| 유도: AhR/PXR/CAR 활성화 양성 | In vivo 시험 고려 |
| 수송체: [I]/IC₅₀ 기반 기준 초과 | In vivo 시험 고려 |

### 3. 임상 DDI 시험 설계

#### 일반 원칙

| 항목 | MFDS 요건 |
|------|----------|
| 시험 유형 | 전향적(prospective) 전용 DDI 시험 원칙 |
| 대상자 | 건강한 성인 자원자 (특수한 경우 환자 가능) |
| 기본 설계 | 교차설계(crossover) 권장 |
| 고정순서(one-sequence) | 반감기 긴 경우, 비가역적 억제, 유도제 시 허용 |

#### 설계별 권장사항

| 설계 | 적용 상황 |
|------|----------|
| 2×2 교차 | 가역적 억제, 짧은 반감기 약물 — 표준 설계; ANOVA 분석 |
| 고정순서(fixed sequence) | 효소 유도, 비가역적 억제, 긴 반감기 — 가장 흔한 DDI 설계 |
| Williams 4×4 | 다중 용량 수준, 복수 상호작용제 비교 |
| 병렬 | 교차 불가 시 (긴 반감기 + 강한 유도) |

#### 투여 조건

| 항목 | 요건 |
|------|------|
| 기질(substrate) 투여 | 단회투여 원칙; 다회투여 필요 시 정상상태 확인 |
| 억제제 투여 | 정상상태(steady-state) 도달 후 기질 투여 (통상 5~7일 전투여) |
| 유도제 투여 | 정상상태 도달 후 기질 투여 (최소 7~14일 전투여) |
| 용량 | 승인 용량 범위 내 최대 상호작용 예상 용량 사용 |
| 공복/식후 | 약물 허가 용법 또는 worst-case 상황에 따름 |

### 4. 평가변수

| 평가변수 | 역할 |
|---------|------|
| **AUC₀₋∞** | **1차 평가변수** — 대사 억제/유도 정도 평가 핵심 |
| AUC₀₋ₜ | 1차 보조 (AUC₀₋∞ 산출 불가 시) |
| **Cmax** | **2차 평가변수** — 흡수 단계 상호작용 |
| t½, CL/F | 탐색적 |
| 대사체/모약물 비율 | 탐색적 — 대사 경로 변화 평가 |

### 5. 통계 분석

#### 교차설계 DDI

| 항목 | MFDS 요건 |
|------|----------|
| 분석 방법 | ANOVA (BE 분석과 동일 모형) |
| 변환 | 자연로그(ln) 변환 후 분석 |
| 보고 | GMR 및 90% 신뢰구간(CI) |
| No-effect 기준 | 90% CI **80.00%~125.00%** 이내 → 임상적으로 유의한 DDI 없음 |

#### 고정순서(One-sequence) DDI

| 항목 | 방법 |
|------|------|
| 주 분석 | Paired t-test (로그변환) 또는 ANOVA (기간 효과) |
| 보고 | GMR 및 90% CI |
| 추가 분석 | 개별 대상자 fold-change scatter plot |

### 6. DDI 결과 해석 기준

#### 억제 분류

| GMR 범위 | 해석 | 조치 |
|----------|------|------|
| 0.80 ≤ GMR ≤ 1.25 | 임상적으로 유의한 DDI 없음 | 용량 조절 불필요 |
| 1.25 < GMR ≤ 2.0 | 약한(weak) 억제 | 주의 관찰 또는 라벨링 |
| 2.0 < GMR ≤ 5.0 | 중등도(moderate) 억제 | 용량 조절 권고; 라벨 명시 |
| GMR > 5.0 | 강한(strong) 억제 | 병용 금기 또는 대폭 감량 |

#### 유도 분류

| GMR 범위 | 해석 |
|----------|------|
| 0.50 ≤ GMR < 0.80 | 약한(weak) 유도 |
| 0.20 ≤ GMR < 0.50 | 중등도(moderate) 유도 |
| GMR < 0.20 | 강한(strong) 유도 |

### 7. IND 신청 시 DDI 자료 요건

| 단계 | 요건 |
|------|------|
| Phase 1 이전 | In vitro CYP 억제/유도 데이터; 수송체 기질 여부 평가 |
| Phase 1~2 중 | 주요 대사 경로 확인 후 임상 DDI 시험 계획 |
| 허가 신청 시 | 임상 DDI 시험 결과 보고서; 라벨링 문구 제안 |

## 본 시험(Clopidogrel-Omeprazole DDI)에 적용되는 항목

| 항목 | 적용 내용 |
|------|----------|
| 시험 설계 | Fixed-sequence 교차설계 — CYP2C19 저해제(Omeprazole, perpetrator) 정상상태(5~7일 전투여) 후 Clopidogrel(victim) 단회 투여 |
| 투여 용량 | Clopidogrel 75 mg (허가 용량); Omeprazole 80 mg (trial_info 기재, FDA DDI 시험에 사용된 worst-case 용량) |
| 1차 평가변수 | Clopidogrel 활성 대사체(H4) AUC₀₋ₜ 및 AUC₀₋∞의 GMR 및 90% CI |
| 2차 평가변수 | Cmax GMR; 모약물 AUC/Cmax; PD (PRU, LTA) |
| No-effect 기준 | 90% CI 80~125% — 이 범위를 벗어나면 임상적으로 유의한 DDI 존재 |
| 통계 분석 | Paired t-test (ln 변환) 또는 ANOVA; GMR 및 90% CI |
| CYP2C19 분층 | 한국인 CYP2C19 PM 비율 (~15%) 고려, 표현형별 층화 분석 권장 |
| 대상자 | 건강한 성인 남성 (19-45세, trial_info 기재) |
| Washout | 활성 대사체 t½ 짧음(0.5~1 hr)이나 혈소판 수명(~10일) 고려하여 ≥14일 washout |
| PBPK | MFDS는 보조 근거로만 인정. 임상 시험 생략 근거로 사용 불가 |

## 다른 가이드라인과의 차이점

| 항목 | MFDS (본 가이드) | FDA | EMA |
|------|---------------|-----|-----|
| 가이드라인 구조 | In vitro + In vivo 통합 | In vitro/In vivo 별도 | 통합 |
| PBPK 수용도 | 보조 근거 제한적 | 임상 시험 대체 허용 | 지지적 도구 |
| No-effect 기준 | **80~125%** | **80~125%** | 0.5~2.0배 (서술적) |
| 농도 지표 | 불명확 (총 농도 관행) | 비결합 권장 | 비결합 명시 |
| BSEP | 별도 언급 없음 | 없음 | **필수** |
| 유도 평가 효소 | 3종 | 3종 | 5종 |
| 라벨 반영 | 허가사항(용법용량, 주의사항) 직접 반영 | Drug label (PI) 반영 | SmPC 반영 |
