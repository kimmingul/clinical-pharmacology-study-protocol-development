## 4. 약물상호작용 — Clopidogrel 관련 핵심 (DRUG INTERACTIONS 및 WARNINGS)

### Section 5.7: Clopidogrel과의 상호작용 (WARNINGS에 기재)

> **"Avoid concomitant use of omeprazole with clopidogrel."**
>
> Clopidogrel은 prodrug. 혈소판 응집 억제는 전적으로 활성 대사체에 의한다. **Omeprazole과 같이 CYP2C19 활성을 억제하는 약물의 병용은 clopidogrel의 활성 대사체로의 대사를 장애시킬 수 있다.**

**임상 데이터 (라벨 기재)**:
- Omeprazole 80 mg/일 병용: clopidogrel 활성 대사체 노출 **현저히 감소**
- 12시간 간격 투여도 동일 효과 → 분리 투여로 상호작용 회피 불가
- 권고: 대체 항혈소판 치료 고려

### 주요 DDI 목록 (CYP2C19 기반)

| 병용약 | 기전 | 결과 | 권고 |
|--------|------|------|------|
| **Clopidogrel** | CYP2C19 TDI 억제 | 활성 대사체 감소 → 항혈소판 효과 저하 | **병용 금지** |
| Citalopram | CYP2C19 억제 → citalopram 노출 증가 | QT 연장 위험 | 최대 20 mg/일로 제한 |
| Cilostazol | CYP2C19 억제 | 노출 증가 | 50 mg bid로 감량 |
| Phenytoin | CYP2C19 억제 | 혈중 농도 증가 | 모니터링 및 용량 조절 |
| Diazepam | CYP2C19 억제 | AUC 증가 | 필요 시 용량 감소 |
| Warfarin | CYP2C9 억제(미약) | 출혈 시간 연장 보고 | INR/PT 모니터링 |
| Rilpivirine | CYP3A4 유도 → rilpivirine 감소 | HIV 치료 실패 | **금기** |
| Atazanavir / Nelfinavir | pH 상승 → 흡수 감소 | 항바이러스 효과 저하 | **병용 금지** |
| Methotrexate | 신장 배설 억제 가능 | 독성 증가 | 주의 |

### CYP2C19 기질 (omeprazole 자신)

Omeprazole은 CYP2C19의 **기질이자 시간의존적 억제제(TDI)**:

> "Omeprazole은 CYP2C19의 time-dependent inhibitor로서 **자가 억제(autoinhibition)**를 일으키고 **비선형 약동학**을 보인다."
>
> **정상상태(steady-state)에서 농도 비례 이상 증가**:
> - 40 mg/일: Cmax 118% 증가, AUC 175% 증가 (반복 투여 시)

## 5. 임상약리 (CLINICAL PHARMACOLOGY)

### 5.1 작용 기전

Omeprazole은 H+/K+-ATPase(proton pump) **비가역적 억제제**. 산 분비 최종 단계를 차단.

### 5.2 대사 경로 (CYP2C19/CYP3A4)

| 대사 경로 | 생성 대사체 |
|---------|-----------|
| CYP2C19 (주) | Hydroxyomeprazole |
| CYP3A4 (부) | Omeprazole sulfone |

**PM(CYP2C19 poor metabolizer)에서**:
- 주 대사 경로(CYP2C19) 비기능 → CYP3A4가 대안 경로
- 혈중 농도 현저히 높음 (EM 대비 AUC 수배 증가)

### 5.3 PK 파라미터

| 파라미터 | 값 |
|---------|---|
| Tmax | ~0.5~3.5 hr |
| t½ | ~0.5~1 hr (EM); PM에서 더 길어짐 |
| 단백 결합 | ~95% |
| 생체이용률 | ~30~40% (초회 통과 효과; 반복 투여 시 60%까지 증가) |
| 대사 | CYP2C19 (주) + CYP3A4 (부) |
| 배설 | 소변 77% (대사체로); 대변 18% |

## 7. 본 시험 설계에의 시사점

| 항목 | 내용 |
|------|------|
| 시험 용량 근거 | 본 시험에서 Omeprazole 80 mg/일 사용 — FDA 기존 DDI 시험에서 사용된 용량; 최대 상호작용(worst-case) 시나리오 |
| 투여 기간 | 정상상태(steady-state) 도달: t½ ~1 hr이나 TDI autoinhibition으로 5~7일 전투여 후 정상상태 확인 필요 |
| Trough 측정 | Omeprazole 정상상태 확인을 위한 trough 혈중농도 측정 권장 |
| CYP2C19 표현형 | Omeprazole 자체가 CYP2C19 기질 — PM에서 omeprazole 농도 크게 높음 → DDI 크기도 달라질 수 있음 |
| 투여 시간 | 식사 전 30분 투여 (허가 용법 준수) |
| 라벨 경고 인용 | 계획서 배경에 "Section 5.7: Avoid concomitant use of omeprazole with clopidogrel" 인용 필수 |
| 분리 투여 불가 | 12시간 간격 투여도 효과 감소 동일 → 병용 회피 불가 확인 |
