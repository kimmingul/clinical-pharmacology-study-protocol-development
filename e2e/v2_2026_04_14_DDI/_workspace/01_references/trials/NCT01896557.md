# NCT01896557 — Ranitidine vs Omeprazole in Patients Taking Clopidogrel

## 기본 정보

| 항목 | 내용 |
|------|------|
| NCT 번호 | NCT01896557 |
| 시험명 | Ranitidine Versus Omeprazole in Patients Taking Clopidogrel |
| 의뢰자 | University of Sao Paulo General Hospital; 협력: InCor Heart Institute |
| 단계 | Phase 4 |
| 상태 | Completed (results posted May 2016) |
| 시작일 | October 2011 |
| 완료일 | June 2013 |
| 시험 유형 | Active comparator, parallel-group (PD 중심) |

## 연구 목적

- 안정적 관상동맥 질환 환자에서 Clopidogrel 병용 시 Omeprazole vs Ranitidine의 혈소판 기능 억제 효과 차이 비교
- CYP2C19 다형성이 Omeprazole DDI에 미치는 영향 평가

## 시험 설계

| 항목 | 내용 |
|------|------|
| 설계 유형 | Double-blind, double-dummy, randomized, parallel-group |
| 맹검 | Double-blind |
| 대조군 | Ranitidine (active comparator) |
| 기간 | 1주 치료 |
| 전처치 | 1주 clopidogrel 단독 투여 후 무작위배정 |

## 투여 방법

| 약물 | 용량 |
|------|------|
| Omeprazole | 20 mg BID × 1주 |
| Ranitidine | 150 mg BID × 1주 |
| Clopidogrel | 75 mg QD (모든 대상자, open-label) |
| Aspirin | 100 mg QD (모든 대상자) |

## 선정/제외 기준

**선정:**
- 18세 이상 안정적 관상동맥 질환 환자 (이전 MI, 성형술, CABG, 또는 혈관조영 ≥50% 폐색)
- Aspirin 100 mg/day 복용 중

**제외:**
- 최근 항혈소판제/항응고제 복용
- 이전 7일 내 PPI/ranitidine 복용
- 활성 출혈
- 임신
- 비정상 헤모글로빈/혈소판/크레아티닌 청소율
- 최근 PCI/CABG/급성관상동맥증후군
- 악성종양, 소화성궤양, 알레르기

## 대상자 수

| 항목 | 수치 |
|------|------|
| 계획 | 100명 |
| 실제 등록 | 92명 |
| 완료 | 85명 (Omeprazole 44, Ranitidine 48) |

## 1차 평가변수

1. VerifyNow P2Y12 Reactivity Units (PRU) — 무작위배정 1주 후
2. Inhibition of Platelet Aggregation (IPA) %

## 2차 평가변수

- Bioimpedance aggregometry
- PFA-100 (Collagen/ADP)
- 하위군 분석: CYP2C19 다형성, 연령, 성별, 흡연, BMI, 당뇨병, 스타틴

## 임상약리학적 시사점 (본 DDI 시험 설계 관련)

1. **실제 환자군 vs 건강인**: 본 시험은 관상동맥 질환 환자 대상 — 건강인 대상 시험과 대조군으로 활용 가능
2. **저용량 Omeprazole (20 mg BID)에서도 PD 영향 입증**: 본 설계 시험의 Omeprazole 80 mg이 과도한 용량이 아님을 간접 지지
3. **PD 평가변수 참고**: VerifyNow PRU + IPA — PD 평가변수 표준으로 널리 사용됨 (본 시험의 2차 평가변수로 채택 가능)
4. **CYP2C19 하위군 분석**: 한국인 CYP2C19 PM 비율이 높음을 고려한 하위군 설계의 근거
