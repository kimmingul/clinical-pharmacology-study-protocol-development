# NCT00557921 — COGENT-1: Clopidogrel + Omeprazole vs Clopidogrel 단독 (Phase 3)

## 기본 정보

| 항목 | 내용 |
|------|------|
| NCT 번호 | NCT00557921 |
| 시험명 | Clopidogrel and the Optimization of Gastrointestinal Events (COGENT-1) |
| 공식명 | A Randomized, Double-Blind, Double-Dummy, Parallel Group, Phase 3 Efficacy and Safety Study of CGT-2168 Compared With Clopidogrel to Reduce Upper Gastrointestinal Events |
| 의뢰자 | Cogentus Pharmaceuticals |
| 단계 | Phase 3 |
| 상태 | Terminated (조기 종료, 의뢰자 결정) |
| 시작일 | December 2007 |
| 종료 예정 | November 2009 |
| 시험 유형 | GI 보호 목적 복합제 vs 단독제 비교 (PD/임상 결과 중심) |

## 연구 목적

- Clopidogrel 75 mg + Omeprazole 20 mg 복합 고정용량 제제(CGT-2168)가 Clopidogrel 단독 대비 상부 소화관 사건 감소 여부 평가
- 동시에 Omeprazole 병용이 심혈관 사건에 미치는 영향 평가 (DDI 우려 검증)

## 시험 설계

| 항목 | 내용 |
|------|------|
| 설계 유형 | Randomized, double-blind, double-dummy, parallel-group |
| 맹검 | Triple masking (대상자, 조사자, 평가자) |
| 대조군 | Active comparator (Clopidogrel 75 mg 단독) |
| 기간 | ≥12개월 |

## 투여 방법

| 군 | 약물 |
|----|------|
| Experimental | CGT-2168 (Clopidogrel 75 mg + Omeprazole 20 mg) + Aspirin |
| Comparator | Plavix (Clopidogrel 75 mg) + Aspirin |

## 선정 기준

- ACS 또는 관상동맥 스텐트 삽입으로 ≥12개월 dual antiplatelet therapy 필요 환자

**주요 제외:**
- 만성 PPI 복용자
- 최근 대수술
- 출혈 기왕력
- 심인성 쇼크
- 기대 여명 <12개월

## 대상자 수

- 계획: 5,000명 (조기 종료로 미달성)

## 1차 평가변수

**상부 소화관 복합 임상 사건**: 위십이지장 출혈, 증상 있는 위십이지장 궤양, 지속 통증 + 다발성 위미란, 폐쇄, 천공

## 2차 평가변수

- 개별 GI 사건
- 약물 중단
- GERD/식도염
- 소화불량
- 심혈관 사건 (MACE)

## 주요 결과 (조기 종료 후 게재)

- **GI 사건**: Omeprazole 복합제 군에서 소화관 사건 유의하게 감소
- **심혈관 사건**: 두 군 간 유의한 차이 없음 (Omeprazole DDI가 임상 결과에 미치는 영향은 입증되지 않음)
- 조기 종료로 통계적 검정력 부족

## 임상약리학적 시사점 (본 DDI 시험 설계 관련)

1. **임상적 우려 배경**: COGENT-1 결과에도 불구하고, 규제 기관(FDA/EMA/MFDS)은 PK 기반 DDI 우려를 유지 → PK DDI 평가의 독립적 가치를 지지
2. **대상 집단**: COGENT-1은 실제 환자 대상; 본 DDI PK 시험은 건강인 대상 → PK 변동 요소 최소화
3. **임상 용량**: Omeprazole 20 mg 사용 (본 시험 80 mg은 최대 저해 조건으로 worst-case DDI 평가)
4. **규모 대조**: 5,000명 Phase 3 vs 본 시험 ~24명 Phase 1 PK 시험 — 목적이 다름
