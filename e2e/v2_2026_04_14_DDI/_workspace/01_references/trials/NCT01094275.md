# NCT01094275 — CYP2C19 다형성과 Clopidogrel-Omeprazole 약물상호작용

## 기본 정보

| 항목 | 내용 |
|------|------|
| NCT 번호 | NCT01094275 |
| 시험명 | Role of CYP2C19 Polymorphism in the Drug Interaction Between Clopidogrel and Omeprazole |
| 의뢰자 | Neil Kleiman, MD / Methodist Hospital Research Institute |
| 단계 | Phase (해당 없음 — observational) |
| 상태 | Completed (verified January 2018) |
| 시작일 | January 2010 |
| 완료일 | June 2014 |
| 시험 유형 | Observational, case-crossover, prospective, CYP2C19 genotype-stratified |

## 연구 목적

- CYP2C19 기능 상실(loss-of-function, LOF) 대립 유전자 보유 여부에 따라 Omeprazole 병용이 clopidogrel의 혈소판 억제 효과에 미치는 차이 평가
- CYP2C19 LOF 보유자와 wild-type 대조군 비교

## 시험 설계

| 항목 | 내용 |
|------|------|
| 설계 유형 | Observational, case-crossover, prospective |
| 층화 | CYP2C19 LOF 변이 보유 여부 |
| 기간 | Crossover 방식, 각 기간 1주 |
| 총 추적 | 3주 |

## 투여 방법

| 기간 | 처치 |
|------|------|
| 기간 1 | Clopidogrel 75 mg 단독 |
| 기간 2 | Clopidogrel 75 mg + Omeprazole 20 mg |

## 선정/제외 기준

**선정:**
- 18–65세 건강한 지원자
- CYP2C19 유전형 확인 가능

**제외:**
- 혈소판 기능에 영향하는 약물 복용
- 심장 질환 기왕력
- 출혈 장애
- 연구 약물 알레르기
- 임신/수유

## 대상자 수

| 항목 | 수치 |
|------|------|
| 스크리닝 | 75명 |
| 실제 등록 | 32명 |
| LOF 변이 보유군 | 16명 |
| Wild-type 대조군 | 16명 |

## 1차 평가변수

- Omeprazole 병용이 clopidogrel 혈소판 억제 효과를 감소시키는지 여부 (LOF 변이 보유자에서, 3주)

## 2차 평가변수

- Omeprazole이 LOF 변이 보유자에서 clopidogrel 활성 대사체 전환에 미치는 영향 (3주)

## 임상약리학적 시사점 (본 DDI 시험 설계 관련)

1. **대상자 층화**: 이 시험은 CYP2C19 LOF 변이 여부로 층화 — 본 시험 설계에서 CYP2C19 EM 대상자 선정 전략의 근거로 참고
2. **소규모 설계**: 32명으로 CYP2C19 효과 탐색 — 본 시험 ~24명 규모 설계와 유사한 샘플 크기 범위 지지
3. **PD 평가변수**: 혈소판 억제율을 2차 평가변수로 — 본 시험의 2차 PD 평가변수 설계에 참고
4. **한국인 적용**: 한국인 CYP2C19 PM 빈도 ~15%로 높음 → 건강인 대상 DDI 시험에서 CYP2C19 EM 선정이 매우 중요함
