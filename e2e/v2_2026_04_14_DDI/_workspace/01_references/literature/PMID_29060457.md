# PMID 29060457 — Tangamornsuksan et al. 2017: Clopidogrel-Omeprazole PK 모델링

| 항목 | 내용 |
|------|------|
| PMID | 29060457 |
| 제목 | A pharmacokinetic model of drug-drug interaction between clopidogrel and omeprazole at CYP2C19 in humans |
| 저자 | Tangamornsuksan W, Thiansupornpong P, Morasuk T, Lohitnavy O, Lohitnavy M |
| 학술지 | Annual International Conference of the IEEE Engineering in Medicine and Biology Society (EMBC) |
| 출판연도 | 2017 |

## 연구 목적

- CYP2C19에서의 경쟁적 억제(competitive inhibition) 기전을 적용한 수학적 PK-DDI 모델 개발
- 6개 임상 연구에서 농도-시간 데이터를 추출하여 모델 검증
- Clopidogrel 단독 및 Omeprazole 병용 시 농도-시간 경과 시뮬레이션

## 방법

- PubMed에서 clopidogrel-omeprazole 인간 PK 연구 체계적 검색
- 선택된 6개 데이터셋에서 농도-시간 데이터 추출
- ACSLX 소프트웨어를 이용한 3개 수학 모델 개발:
  1. Clopidogrel 단독 모델
  2. Omeprazole 단독 모델
  3. CYP2C19 경쟁적 억제를 통한 DDI 상호작용 모델

## 주요 결과

- 개발된 DDI 모델이 6개 데이터셋 모두에서 농도-시간 경과를 성공적으로 재현
- **CYP2C19에서의 경쟁적 억제 모델**이 관찰 데이터에 적합
- 모델 예측값이 임상 관찰값과 일치 → 기전 모델의 유효성 검증

## 핵심 결론

1. **CYP2C19 경쟁적 억제 모델로 DDI 재현 가능**: 상호작용 기전을 수학적으로 확인
2. **모델 활용 가능성**: 다양한 용량·투여 일정에서 활성 대사체 예측 가능
3. **참고 데이터**: 기존 6개 임상 연구 데이터 검증 → 문헌 간 일관성 확인

## 임상약리학적 시사점 (본 DDI 시험 설계 관련)

1. **기전 분류 참고**: 이 모델은 CYP2C19 경쟁적 억제를 적용 (단, Ogilvie 2011, Boulenc 2012, Shirasaka 2013 등은 mechanism-based inhibition 주장) → 두 기전이 동시 존재할 수 있음 (복합 기전)
2. **모델 검증 데이터**: 기존 임상 시험들의 일관성을 확인하는 보강 근거
3. **시험 설계 시뮬레이션**: 본 DDI 시험 전에 다양한 투여 일정에 대한 사전 시뮬레이션 가능
