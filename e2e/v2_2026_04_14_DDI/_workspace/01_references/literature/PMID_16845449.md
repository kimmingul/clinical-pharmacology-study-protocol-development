# PMID 16845449 — VerifyNow P2Y12 Validation (Malinin 2006)

## 서지 정보

| 항목 | 내용 |
|------|------|
| 저자 | Malinin A, Pokov A, Swaim L, Kotob M, Serebruany V |
| 제목 | Validation of a VerifyNow-P2Y12 cartridge for monitoring platelet inhibition with clopidogrel |
| 저널 | Methods Find Exp Clin Pharmacol |
| 발행연도 | 2006 |
| 권호 | 28(5):315-322 |
| PMID | 16845449 |

## 연구 방법

- 대상: 관상동맥질환 환자 131명
- clopidogrel 치료 중 VerifyNow P2Y12 카트리지로 혈소판 억제 측정
- LTA (ADP + PGE1 응집법)와 비교 검증

## 주요 결과

- VerifyNow P2Y12 vs LTA (ADP+PGE1): 강한 일치도 (평균 억제율 93% vs 95%)
- **정밀도 (CV)**: < 8% (within-run)
- **간섭 물질**: 나이, 혈소판 수, 헤마토크릿 영향 없음
- **결론**: "VerifyNow-P2Y12 is a reliable, simple, and sensitive device for monitoring P2Y12 inhibitors"

## 시사점 (본 DDI 시험)

- VerifyNow PRU를 2차 PD 지표로 사용하는 근거 논문
- CV < 8%: 측정 정밀도 신뢰 가능
- 검증된 방법론으로 계획서 방법 섹션 인용 가능
