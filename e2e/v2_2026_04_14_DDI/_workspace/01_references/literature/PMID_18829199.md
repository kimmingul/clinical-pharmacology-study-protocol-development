# PMID 18829199 — Clopidogrel Active Metabolite LC-MS/MS (Takahashi 2008)

| 항목 | 내용 |
|------|------|
| 저자 | Takahashi M, Pang H, Kawabata K, Farid NA, Kurihara A |
| 제목 | Quantitative determination of clopidogrel active metabolite in human plasma by LC-MS/MS |
| 저널 | Journal of Pharmaceutical and Biomedical Analysis |
| 발행연도 | 2008년 12월 |
| 권호 | 48(4):1219-1224 |
| PMID | 18829199 |

## 연구 방법

- LC-MS/MS 방법 개발 및 검증 (BMAP 유도체화 사용)
- 인간 혈장에서 H4 정량 분석법

## 분석 조건

- **유도체화 시약**: 2-bromo-3'-methoxyacetophenone (BMAP)
- **이온화**: ESI 양이온
- **교정 범위**: 0.5–250 ng/mL
- **LLOQ**: 0.5 ng/mL

## 검증 결과

| 파라미터 | 결과 |
|---------|------|
| 정확도 | ≤ 12% (상대오차) |
| 정밀도 (CV) | ≤ 6% |
| 회수율 | 85–105% |
| 안정성 (-80°C) | 4개월 |
| 안정성 (실온) | 1주 |

## 시사점 (본 DDI 시험)

- **H4 분석법 핵심 참조 논문**: BMAP 유도체화 LC-MS/MS의 검증 기반
- LLOQ 0.5 ng/mL: 건강한 지원자 75 mg 복용 후 Tmax (~0.5–2 hr) 농도 측정 충분
- 채혈 후 즉시 BMAP 첨가 프로토콜의 근거 제공
- 계획서 생체분석 방법 섹션에서 인용
