# PMID 26071277 — Simon et al. 2015: Omeprazole/Pantoprazole와 CYP2C19가 Clopidogrel PK-PD에 미치는 영향

## 서지 정보

| 항목 | 내용 |
|------|------|
| PMID | 26071277 |
| 제목 | Omeprazole, pantoprazole, and CYP2C19 effects on clopidogrel pharmacokinetic-pharmacodynamic relationships in stable coronary artery disease patients |
| 저자 | Simon N, Finzi J, Cayla G, Montalescot G, Collet JP, Hulot JS |
| 학술지 | European Journal of Clinical Pharmacology |
| 출판연도 | 2015 |
| 권호 | 71(10):1185-91 |

## 연구 목적

- 관상동맥 질환 환자에서 Omeprazole/Esomeprazole 복용 여부 및 CYP2C19*2 유전형이 Clopidogrel PK-PD 관계에 미치는 영향 분석
- Sigmoid Emax PK-PD 모델 개발

## 시험 설계

| 항목 | 내용 |
|------|------|
| 설계 | Randomized controlled trial, PK-PD 모델링 포함 |
| 대상 | 안정적 관상동맥 질환 환자 (MI 이후) |
| CYP2C19 층화 | Wild-type 58명, CYP2C19*2 헤테로 41명, 호모 7명 |
| 투여 | Clopidogrel 300 mg 또는 900 mg 로딩 |

## 선정/제외 기준

**선정:**
- 심근경색 후 관상동맥 질환 환자

**층화:**
- CYP2C19*2 변이 여부
- Omeprazole/Esomeprazole 복용 여부

## 주요 결과

### PK-PD 모델 파라미터

| 파라미터 | 값 |
|---------|---|
| Emax (최대 효과) | **56 ± 5%** (혈소판 응집 억제율) |
| EAUC50 (효과의 50% 달성 AUC) | **15.9 ± 0.8 h·μg/L** |
| PK-PD 관계 | Sigmoid Emax, steep dose-response |

### DDI 영향 (PK-PD 관계)

- **CYP2C19*2 대립 유전자 보유 + Omeprazole/Esomeprazole 복용**이 활성 대사체 청소율의 개인 간 변동성과 유의하게 연관됨
- 가파른 Sigmoid 관계로 인해 소량의 활성 대사체 변화도 임상적으로 의미 있는 PD 변화 유발

## 핵심 결론

1. **Omeprazole/Esomeprazole + CYP2C19 변이의 상승 작용**: 두 인자가 함께 존재 시 활성 대사체 노출 최대 감소
2. **가파른 PK-PD 관계 (Sigmoid Emax)**: EAUC50 = 15.9 h·μg/L → 소량 AUC 감소도 PD에 큰 영향
3. **PK 평가의 중요성**: PK 변화가 PD에 직결 → 본 DDI 시험에서 PK(AUC, Cmax)를 1차 평가변수로 설정하는 근거

## 임상약리학적 시사점 (본 DDI 시험 설계 관련)

1. **1차 PK 평가변수 설정 근거**: PK 변화가 PD에 직결되므로 AUC₀₋₂₄ GMR이 임상적으로 의미 있는 평가변수
2. **환자군 데이터 참고**: 본 시험은 건강인 대상이지만, 이 연구의 PK-PD 관계를 참고하여 GMR 기준치(80–125%) 벗어남의 임상 의의 설명 가능
3. **CYP2C19 유전형 층화 필요성 재확인**: CYP2C19*2와 Omeprazole이 상승적으로 작용 → 유전형 선별 중요
