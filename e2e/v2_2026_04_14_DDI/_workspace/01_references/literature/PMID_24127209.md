## 주요 PK 결과: 활성 대사체 H4

### Clopidogrel 75 mg 유지 용량 (n=41)

| 파라미터 | H4 (활성 대사체) | H3 (불활성 이성체) |
|---------|----------------|----------------|
| Cmax (ng/mL) | **7.13 ± 6.32** | 5.29 ± 5.54 |
| AUCt (ng·h/mL) | **11.30 ± 9.58** | 7.37 ± 6.71 |
| CV% (Cmax) | **~89%** (계산값: 6.32/7.13 × 100) | |
| CV% (AUC) | **~85%** (계산값: 9.58/11.30 × 100) | |

*주: 이 값들은 심혈관 질환 환자에서 측정된 것으로 건강인보다 낮음. 건강인 기준 값은 별도 연구 참고.*

### 임상적 상관관계

- H4 Cmax와 혈소판 응집 사이 통계적 유의한 상관관계 확인 (p = 0.025)
- H4 Cmax가 항혈소판 반응의 개인차 예측 지표로 사용 가능

## 핵심 결론

1. **높은 개인 간 변동성**: H4 CV% ~85–89% (심혈관 환자 기준)
2. **건강인 대비 낮은 노출**: 환자군에서 낮게 측정됨 → 건강인 DDI 시험에서는 더 높은 H4 값 예상
3. **H4 Cmax와 PD 상관**: PK 지표로서 H4 Cmax의 임상적 의미 확인

## 임상약리학적 시사점 (본 DDI 시험 설계 관련)

1. **고변동성 약물 확인**: H4 AUC CV% ~85% → DDI 시험 sample size 계산 시 높은 변동성 반영 필요
2. **채혈 시점 설계**: H4의 짧은 t½ (~0.5–1시간) 고려 → 초기 밀집 채혈(0, 0.5, 1, 1.5, 2, 3, 4, 6, 8시간) 필요
3. **건강인 vs 환자**: 본 시험은 건강인 대상 → 이 연구 값보다 높은 H4 노출 예상
4. **AUC 산출**: H4의 짧은 t½로 인해 AUC₀₋₂₄가 AUC₀₋∞에 충분히 근접 → 절단 AUC 적용 불필요
